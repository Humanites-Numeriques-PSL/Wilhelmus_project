 LOf-sjongh de Heer, uwt herts-gruwns njoe, Hy iz goe, ja 't heagste goe: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Lof-sjong God; dy, wa t'Goads neamm' Preal-fiert, twang-stjoert, az ijnn' teamm': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Lof-sjong d'Heer dy 't heer oer-heer't Dat, aerne' Heer, heersck in rejear't: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy allinne, boppa wet, Tjin Natuwre, wond're det: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy, dy de' Hijmm'len mey forstân, Macke', ô wirck fen socken hân! Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy de' yerd', swier fen wichte', oer 't wiet Fest, oer 't onfest weage', uwt-spried't: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy de' haed-Ljeachten, glanz'-gleaun' great, Macke', al het eag bruwckt to steat': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy de Sinne blier in bly, Oere Dey joe de' heerschappy': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Moanne' in Stiert' joe Hy de macht Wad in fadschippe' oere Nacht: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy Egiptens aet in naet, Heeg, leeg, minsch, djiers eer-bern' daet: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. In kritse' Isr'el, az in buwt, Spijt Nijls tosschen, dear midz uwt: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Mey' hanz kreft, dy 't mord-lee wreck't, Mey' hiet fall'gjende' earme', uwt-streck't: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy 't Rea Mar oon twa'en trog smiet, Midz trogh kleauwd', sney, schaet in spliet: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. In fierde' Isr'el trog dy steeg', Trog Marz midz-hôl, wetter Ieeg: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dear ijn Phar'o, fen Him wirt, Mey sijn Krijs-ljue, gjealp, oer-stirt: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy sijn Folck laet', sijcker, fry, Trog dy wijlde wôsteny': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy kreft-greate Keen'gen sleyn, In uwt-dijllge het, schien t'eyn': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy trog-loft'ge Kening-staef, (Greatsch, ontsjoen', heag) hompe' yn 't graef: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Az Sihon, oerste' Amoriet, Dy hy, az dol-tong're, smiet: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. In de' oer-greate' Og, Basans haed Blix'me hy, oon gruys to naet': Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. In hy joe, uwt mijlder han, To erve'-eyndomm' herre Lan: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. To erve'-eyn', in plante 'er del, Sijn ljeav' Tjienner, Israël: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy mey-wirddigh uwz oon-seag, Yn uwz dol-druwx jamm're pleag: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Dy (uwz mijld) krefts-hânz schoerde uwt, Uwt dea-fijnn' kluwr', tosck in tuwt: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Ja dy Heer, dy alle flesch, Jout, mijld, fiedsel-brea, ney esch: Want sijn goeheyt giet so fier, Son on-eyn-brie, so goertier. Lof-sjong, al het iz, mey my d'Hijmm'len God: ô lof-sjongh dy: Want sijn goeheyt giet so fier, So on-eyn-brie, so goertier.
