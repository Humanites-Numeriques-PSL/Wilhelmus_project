 T'Valt mijn // een pijn Aldus verkleynt te wesen, By der Ieucht fijn Ben ick vergheten heel, Tfenijn // voor wijn Wert seer van haer ghepresen, Het drincken dunckt hen eel, Daer schade schijnt voordeel Soo dom is jeuchts ghespeel, Welks bloem valt van haer steel: Siet hoe dat hier onachtsaemheyt Den tijt verquist ende afleyt, Sulck seyt // ontseyt Goet oordeel diet misgreyt. Comt Ieucht // en veucht V onder mijnen leeren, Goe oorden en deucht Leert // die sijt goet van aert, V vreucht // gheneucht En wilt van u niet weeren, Ontfanght mijn liefde waert: Hierom gheen moeyte spaert, Schout vliet // wat traecheyt baert Sy niet, dat schaedt vergaert: Wilt onlust van u jaghen dan, V tijt dan wel ghebruyckt voortan, Den strijt // met vlijt Aenveert om dien ghy vrijt. Siet toe // ick doe Een yghelijck ten goede, Die op mijn hebben // goe, Acht met hert sin, ghedacht Denckt hoe // ick spoe Als die niet werde moede, My te ghebruycken tracht Den tijt na niemant wacht, Die sluymt, by daech oft nacht Versuym, baet gheen na clacht, 'Tverlies tot zijnder grooter schant Van een so costelijcken pant, Niemant // t'weer vant Dus oeffent u verstant. Die Wet // gheset Om weren wat mach schaden, Neemt aen sonder verlet, Sijt oorden onderdaen, Te bet // ghy met Den tijt so sonder smaden, Des wijsheyts pat sult gaen Haer inden gront verstaen, Dit is, die rechte baen Ghewis, gheen ydel waen, Diet oyt te recht verstonden siet 'Tverbont viel hem so zwaer doch niet, Ghy cont // goet ront Vermeerderen uwe pont. Nu groeyt // en bloeyt Mijn gheestkens uyt ghelesen, Met s'Gheests natuer besproeyt Weest neerstich int ghebou, Lust voeyt // cloeck roeyt En laet u niet verknesen Ick wijsheyt uwe Vrou, Sal gheven u mijn trou Dijn Naem // ick voor behou Den faem // diet oyt vergou, Met rijckdom, goet en eer belaen Ver van haer salse sijn ghedaen, Die teer // in leer Maer int versuymen meer. Mijn zee // oock mee De jonckheyt vordel dede, Wilt ghy niet sijn ghehoont My voor u tuchtster croont, Rust vree // in stee Van tweedracht staen alrede, Soo my met reden toont Dat oorden by u woont, Met const, wert ghy verschoont, Tijts jonst, u wel beloont, Hierom u tijt doch wel besteet Tot deucht en eer zijt snel bereet, D'ontucht // ontvlucht Zijt gierich tot leer sucht.
