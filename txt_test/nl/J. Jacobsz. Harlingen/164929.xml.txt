 Ick sal voorwaer // den Hemelschen vaer, Int openbaer // gaen prijsen, Sijn lof seer claer // sal voor en naer, By sijn dienaer // noch rijsen, Die siele mijn // sal met blijde schijn, Hem verheughe dijn // o wijse, Dattet hoore fijn // op dit termijn, Die ellendiche dijn // en verjolijse. O Goddelijcke graen // met malcander beraen, Met druck belaen // verschoven, Laet ons gheraen // met Davids vermaen Des Heeren naem // gaen loven, Doen ick den Heer // noch sochte van veer, Hy antwoorde my weer // van boven, En verloste my teer // door sijnen leer, Van alle mijn zeer // my beroofde. Die hem aensiet // en tot hem vliedt, Die en wort niet // tot schanden, Want God die biedt // die ellendige met vliet Die tot hem riep // sijn handen, En helptse voort // al wordense verstoort, Wt alle noot // en banden, Sijn enghel hoort // beleghert met accoort Der gheloovigher oort // spijt vyanden. Siet ende smaect //watten vriendelijcken saec Dat hem ghenaeckt // na desen, Die na God haeckt // en op hem waeckt, Niet en versaeckt // maer vreesen, Vreest God altijt // en hem belijt, Ghy die nu zijt // verresen, Want die met vlijt // hem vreest en belijdt, In desen tijt // sal zijn ghepresen. Werpt u begheer // alleenich op den Heer, Voort quaet ghebeer, // u wilt hoeden, Die hongherde seer // na sijn Goddelijcke eer, Vervulde hy weer // met goeden, Maer die hem versmaet // nu zijn versaedt Rijck hooch van staet // van moede, God weer verlaet // nootdruftich gaet, En hem ghenaeckt veel teghenspoede. Hierom verlaet // dit vuyle ghewaet En sulcken staet // mispresen, Gods vrede aenvaet // die s'menschen raet, Te boven gaet ghepresen, En gheeft gheen wijck // ist waerachtelijck, Oprecht cuysch lieffelijck // int wesen, Ist wel luydelijck, ist deuchdelijck, Ist lof eerlijck // gheresen, Comt herwaerts hoort // Gods kinderen sijn woort, Na waerheyt spoort // te deghen, Wie isser voort // die nae 't behoort Met goet accoort // te leven, En wie gherust // tot goede daghen heeft lust, Iou tonghe uytblust // voor sneven, Iou lippen sust // van bedrieghen onrust, En boose lust // te streven. Vant boos wilt vlien // doet goet nadien, Na vrede allien // wilt jaghen, Want God ten tien // wil haer ansien, Die hem ghemien // behaghen, Oock haer ghebeen // en droevich gheween, Wilt de Heere alleen // draghen, Die haer afscheen // en boosheyt deen Wil dit Heere van een // verjaghen. Die Heere voldoet // t'ghebedt seer soet, Eens verslaghen ghemoet // vol smerten, En hy is goet // die tot hem spoet, Ghebroken vroet // van herten, Die gherechticheydt // voor die menschen verbreydt, Van t'quaet hem scheyt // met moeden, Hoewel hy schreyt // hem zijn bereyt En toegheseyt // veel goeden. Maer die nu lacht // en wort gheacht, Van s'menschen gheslacht // ghepresen, En toont sijn macht // over t'gheslacht Die uyt al haer cracht // Godt vreesen, Die wil God weer // werpen ter neer, Sal worden seer // begresen, Door Godes leer // die nimmermeer Sal nemen keer // ist bewesen. Hierom ghy al // der Christenen ghetal, Treedt den wegh smal // na t'leven, Al rijster bal // druck ongheval, Hier in dit dal // der sneven, Voor t'cleyne verdriedt // d'welcken hier schiedt, En wilt doch niet // so vreesen, Maer hier op siet // wat God onthiet, En ons aenbiet // te gheven. Hy roept comt by // die beladen zy Ick sal u maken vry // van sonden, Scheyt uyt het ghecry // wilt wesen bly, Mijn volck sult ghy // zijn bevonden, Mijn jock is soet // dat op u doet Mijn last is goet // den ghesonden, Maer t'is al roet // en teghenspoet, Die sijn verwoet // mijn gronden. O Princelijcke Godt // zijt ghy ons deel en lodt, Bidden wy in slot // ons sanghen, Teghen s'werelts spot // door u ghebodt, Sterckt uwen rodt // haer ganghen, Vwen gheest sendt // dat wy present, T'gheloove jent // aenhanghen, En s'Hemels tent // na desen endt, Seer excelent // ontfanghen.
