 ICk heb bedacht // van het gheslacht Dat dach en nacht // neerstich verwacht Gy nieuw kracht // van Godt te cryghen: In haer ghevecht // recht ende slecht Ghelijck Iob secht // haer hope lecht Op hem wiens recht // niemant mach bedriegen Als met Arents vleugelen zy opvlieghen; Zy loopen en wandelen en worden niet moede: Maer haer vyant // die raeckt in schant Aen elcken kant // nae Schrifts verstant; Doch Godes hant // is haer behoede. Op Noachs daet // nu dencken gaet Hy vont ghenaet // door Godes raet Voor zynen saet // en wordt verheven: Die werelt ront // die wederstondt Des Heeren mont // bleef in haer sondt Ginghen te grondt // verlooren het leven; Die heeft Godt beyde door't water bedreven: Dat vyer deed oock die vrome gheen hinder, Sodoma siet // en die Lot riet En hoorden niet // bleef in't verdriet Maer hy ontvliet // met zyne kinder. Heel Israel // leefden rebel Godes bevel // hy verdeed snel Haer met ghequel // ses hondert duysen D'een afgodt eert // d'ander hoereert, Die derd' temteert // vierd' murmureert, Die vijd' begheert // menich abuysen, Soo verginghen oock al haer gheslachten en huysen: Maer Iosua en Caleb en d'ander haer jonghen Zijn wel ghespaert // van Godt bewaert, Van boosen aert // soo Schrift verklaert, En zy vergaert // van vreuchde songhen. Al die niet vreest // in dit foreest Midts dat men leest // dat altijt meest Quaet zijn gheweest // zijn buyten ghesloten, Die Godt bemint // en recht begint Gheestelijck ghesint // hem selfs verwint Blijft hy ontrint // haer medeghenoten Die vrome te bewaren heeft hem noyt verdroten: Die boose brengt hy tot straffe oock klaerlijck Nu 'tlicht doorbreckt // 'tvuyl wort ontdeckt, Nu ruymt die sect // Gods volck begeckt. En maeckt suspect, gantsch openbaerlijck. Neemt hier op acht // hoe dat elck wacht, Met list by nacht // by daech met cracht, Gods volc met smacht // te brengen in sneven Die Salmists reen // tneghentich een, Acht dit niet cleen // troostet ghemeen, Allen den gheen // die haer begheven, Onder die bescherminge des almachtigen te leven En onder sijn schaduwe hem aldus aenroepen Mijn Heer mijn God // mijn burch mijn slot, Mijn deel mijn lot // wy Iacobs rot, . Na u ghebodt // op u wy hoepen. Niet een vergaet // int strick ghevaet Die hem verlanct // op Godes raedt, Maer t'sijnder baet // salmen verheffen, Al viel int sant // aen u sijdts cant, Duysent ter schant // ter rechter hant, Tien duysent want // t'sal u niet treffen, Maer sult u lust sien hier beneffen, Oock hoet den Godloosen wert vergouden, V sal gheen schaet // door yemants daedt, T'eenigher maet // vroech nochte spaet, Aencomen quaet // wilt dit onthouden. Elck neem noch waer // gans openbaer, Dat by Gods schaer // druck angst en vaer, Bangicheyt naer // vaeck is t'werven, Maer sulck verdriet // wert niet bediedt. Schriftuer aensiet // dat het gheschiet, Om haer tot niet // gants te bederven, Wy hadden besloten seyt Paulus te sterven Voor den druck die ons in Asien bedeckte, T'welck ons gheschach // op dat elck mach Dencken die dach // doen hy aensach, Al ons gheclach // die de dooden verweckte. Sulcken doot banck // in haer aenvanck, Stijf ende stranck // al duerdse lanck, God noch bedwanck // tot onse leere, Ghedenckt oock meed' // een wijse reed', Die Iudith deed' // in haer ghebeed', Oock t'hooft af sneed // van den Velt-heere, Tot die ouders des volcx nam sy haer keere, Men mach den Heer tijt noch dagen aenreden Lijd' wy ghequel // vermaentse snel Exempel stel // Abraham wel, Isack Israhel // en Moyses veel leden. Ia allegaer // die God lief waer, Perijckel swaer // quam over haer, En bleven daer // trou sonder crakielen, Maer die in noot' t'zy cleyn of groot, Door elck aenstoot // oft valsche boodt, Oock inden doot // van God vervielen, Ginck hy door den verderver vernielen, En sal hem noch aen sulcke wreken, Daerom so sult // ghy met ghedult, Kennen u schuldt // op dat Gods hult, In u vervult // nimmermeer ghebreecken. Al die Godt vreest // ghedachtich weest Dit swaer tempeest // minst ende meest Hoe 't Davids gheest // dus gaet belyden: Ick prijs u Heer // dat ghy my weer Verhooghet seer // door ws-naems eer Mijn vyandt neer // niet laet verblyden; Doen ick u aenriep Heer ghy waert t'mijnder zijden En ghinckt mijn siel voor't verderven bewaren: En hebbet my // levendich ghy Behouden bly // daer alle zy Ter Hellen sy // zijn neder ghevaren. Comt Heyl'ghen teer // lof singt den Heer Danckt prijst zijn eer // want zijn toorn seer Die duert gheen meer // maer ons verheuchde Den avont lanck // gheduerde stranck. Het weenen banck // maer nam een ganck Smorgens vry vranck // so quam die vreuchde: Maer ick sprac doent my wel ginck in deuchde: Ick sal nimmermeer ter neder vallen; Want Heere goet, door u behoet Met rechter spoet // hebt ghy mijn moet Als een berch soet // sterck maeckt met allen. Och doen ghy dijn // heylich aenschijn Berchde voor mijn // och watte pijn Tot dien termijn // verschrickte ick seere; Och heere Godt // wy Iacobs rot Waren tot spot // sonder u lot Nae u ghebodt // wy smeecken den Heere Wat isser nut aen mijn bloedt nu meere? Want na mijn doodt sal u 'tstof oock dancken: Heere toch hoort // nae u accoort En heylich Woort // soo helpt ons voort Tot aen dien oort // daer wy nae jancken. Boven dit al // mijn plaech misval Veel in't ghetal // meer dan ick kal Hier in dit dal // hebt ghy verandert, In rey verhoocht, want ghy vermoocht En my uyt toocht // den sack verdroocht Begort niet vroocht // als een vast standert Op dat al die in deuchde wandert Vwen lof singhen met eerbare zeden; En uwen eer // toch nimmermeer Stilswyghen weer // soo sal ick seer Dancken mijn Heer // in eeuwicheden. Wie dit nu vaet // en meent hy staet In rechter maet // in deucht voortgaet Siet toe met raet // dat hy niet valle Niet stout van gheest // nu zijt maer vreest En selver leeft // hy is gheweest Streng over meest // die natuerlijcke alle Die poort is eng' die wech is smalle, Hy sal ons niet ontsien noch sparen Lucas derthien // gaet wel besien Van dese twien // Galcese lien, En noch achtien // die tot Syloa waren. So wie gheen boet // oprecht en doet, Hem neerstich spoet // altijt int goet, Die selfde moet // so gaen te gronde, Leeft noch voort daer // hy seyde haer, Doen allegaer // een byspil swaer, Op dat elck claer // die sake verstonde, Een had een vijgeboom geplant hy vermonde In sijn wijngaert daer hy vruchten of sochte, En vanter gheen // sprack hy ghemeen, Met cloecke reen / drie jaer aen een, Met quade zeen // gheen vruchten hy brochte. Dus hout hem of, dat wilde stof Is veel te grof / te hebben t'lof, In s'Heeren hof // als vruchtbare boomen, Merckt hoe ick las // hy antwoort ras, Laet hem dit pas // staen als hy was Ende in cas // gheen vruchten en comen, Na dat ick hem neerstich heb waer genomen Met graven en messen na t'recht der vromen Daer na met vlijt // maeckt hem u quijt, Sonder respijt // op dat gheen tijt Elck ypocrijt // met bladeren staet roemen. Ey neemt nu waer // elck ghetals jaer, Seven en vijftich daer // twee sessen claer, Twee achten swaer // wat gingher te gronden Comt nu yemant // roemrijck ter hant En seyde want // sy zijn te schant, Oock al van cant // door hare sonden, En wy zijn noch oprecht bevonden, T'blijckt wy staen in des Heeren gaerde, Daer stont hy meed // verhaelt alreed, Drie jaer men leed // noch een ter beed, Vervult een steed // maer niet en baerde. Dat wy noch staen // den dau ontfaen, Is niet ghedaen // om ons recht gaen, Maer God siet aen // hoe wy ons bekeeren, Die nu vertraecht // t'zy knecht oft maecht, Ionck oft bedaecht // God niet behaecht, Maer van hem jaecht // na Salomons leeren Daerom laet ons die lanckmoedicheydt des Heeren Voor onse meeste salicheyt betrachte Wie niet is cloeck // in elcken hoeck, Die crijcht een vloeck // dit selfs na soeck, Int propheets boec // Ieremias veertich achte. Iammer waert groot // sou geene noot, Noch swaer aenstoot // hoe naeckt en bloot, Ia veler doot // ons maken verleghen, Boven dit al ons eyghen mis val, Bitter als gal // clinckt met gheschal, En't ons niet sal // meughen beweghen, Och vraecht na die voorgaende weghen, En wandeltse trou so suldy ontfanghen, Der sielen rust // in dien ghy blust Stilt ende sust // des vleysches lust Die waerheydt kust // al met verlanghen. Tot een besluyt // laet dit gheluyt Als een goet kruyt // als Christus bruyt Niet een minuyt // uyt hert en sinnen; Al u ghedacht // bedwingt met kracht V neerstich wacht // dach ende nacht Voor't Helsche gracht // om te verwinnen Als oft ghy van nieuws aen wildet beginnen Dient Godt met vreesen verblijt u met beven Die Heer is soet // barmhertich goet Oock metter spoet // toornich verwoet Want hy elck doet // sulck loon sulck leven. Memorie hout // op nieu en out Christum betrout // hebt altijdt fout Dat u niet rout // 't eeuwighen tyden; Niet stout maer vreest // ghedachtich weest En selver leest // hoe noch het beest Noch listich gheest // niet viert van stryden; Dus wilt ghy eeuwich met Godt verblyden Omgort u lenden u licht laet branden En verlangt seer // nae uwen Heer Want hy komt neer // met grooter eer Beveelt u teer // in zynen handen.
