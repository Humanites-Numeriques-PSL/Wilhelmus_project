 IN Esaie // prophecye Int tweede staet gheschreven Hoe dattet zy // ten laetsten ty Als dan sal hooch zijn verheven, Den berch ghewis// op welcker is Mijn groot ghedruys // des Heeren huys, Als datter veel hoopen // hene loopen, Met vlijt // verblijt Elck ander dus toe coopen. Al afghelaen // comt laet ons gaen, Tot desen berch ghepresen, Die vast sal staen // nimmermeer vergaen, Maer eeuwich t'hoochste sal wesen In welck die Godt // van Iacobs rodt, Nu hout present // sijn schoone tent, Dat hy ons leer te deghen // sijne weghen, Sijn paen // te gaen, En in gheen vreemde steghen. Niemant aenhoort // noch gheenen oort Dan Syon uytghelesen, Heeft Godes woort // met goet accoort Ierusalem ghepresen, Voert Godes wet // die onbesmet, Oordeelen sal // die Heydens al, In oprechtigher claerheyt // na der waerheyt, Sijn recht // bevecht, Die volcken sonder swaerheyt. In sulcken ganck // sal elck te danck Sijn swaert tot ploechysers breken, Die spietsen lanck // tot sickelen blanck, En dencken niet meer te wreken Van dese gheen // ja oock met een Die daer begheert // eenighe sweert Teghen yemant te luycken // en ghebruycken, Gheen striedt // gheschiedt Van haer noch sulcke stuycken. Also dees reet // van den Propheet Luydt gatet in onse tijden, Want veel is leet // haer overtreet, Waer over haer nu verblijden Hier niet alleen // Christus ghemeen, Maer oock seer claer // d'enghelen schaer, Want so staet hier beneven // claer geschreven Van een // onreen Die hem tot boet wil gheven. Comt doch eens naer // aenschout die schaer Die nu na t'beste haecken, Die voormaels waer // s'werelts dienaer Betracht nu andere saecken, Eertijts was sy // eyghen nu vry, Gheneycht tot quaet // nu tot weldaedt, Gierich oncuys sy saten // boven maten, Met veel // sulck speel, T'welck sy nu al verlaten. Oock siet hier meer // hoewel noch teer Die mee beginnen te comen, Al haer gheweer // legghen ter neer, Aensien die menschen als boomen, Wijn sonder alst // melck onghevalst, Begheeren sy // als kinderen ny, Op dat al jonge rancken // door die drancken Aemacht // tot cracht, Sy comen en God dancken. Peys en vree // is in dees stee, Met God voor sijn ghenade, Dus wendt u tree // ghy alle nu mee, Die vreest der zielen schade, Nu isset vry // comt alle mee by, Het is nu claer // t'recht gulden jaer na der Basuynen clangen // wilt verlangen, Dees dach // vermach lElck een sijn erf te langhen. Sal nu dees teyt // der salicheyt Dees aenghename daghen, Met goet bescheyt // worden verbreyt, En dat na Gods behaghen, Sulcx niet allien // hoort te gheschien, Met groot begheer // der reyner leer, Maer wy moeten niet beven // ooc ons leven Daer naer // gantsch claer Met neersticheyt begheven. Op dat wy geen // der swacker leen, Tot erghernisse strecken, Want vele ween // staen over deen, Die zijn des lichaems vlecken, Haer straf met spoet // haers naesten bloet, Och t'is seer swaer // comt al op haer, Als vruchteloose boomen // en onvromen, Met vliet // toesiet Om sulckes te ontcomen. En wilt doch niet // den blinden yet, Tot eenen aenstoot setten, V leven biet // sonder verdriet, Dat elck daer op mach letten, Duldich in druyck // noot angst misbruyck, In slaghen daer // ghevanckenis swaer, In oproer arbeyt vaecken // oock in waecken, Het zy // of ghy Om u gherucht oock raecken. Nimmermeer laet // aen u yet quaet, Met waerheyt zijn te wreecken, Corinthen staet // ick houde maet, Op dat ick andere te preecken, Niet self en ben // verwerpelijck slem, Dus laet oock wy // waecken bety, Dat wy sonder beswaren // recht verclaren Ons baen // wilt gaen, Want wy zijn Gods dienaren. Gheen oorsaeck gheeft // der onbeleeft In dese tijt der ghenaden, Dat ghy niet sneeft // en hy niet heeft Waer mede hy u mach schaden, Op dat ons pont // niet gae te gront, Maer onse licht // com int ghesicht, Op dat ghelijck wy lesen // oock by desen, Gofs naem, eersaem, In ons hooch wert ghepresen. Alsulcken leer // eyscht God die Heer, Die beyt met woorden en wercken, Soeckt sijne eer // cloeck is te weer, Sijn teere spruytkens te stercken, Dese met deucht // verweckt die jeucht, En maect verblijt // alle die belijt, Christum en sijnen vader // drucx ontlader, Doet goet // met spoet, Ghy Christen alle gader. Voor sulcke vreucht // voor sulcke deucht, Voor sulcke groote ghenade, Die ons verheucht // al dat ghy meucht, Lovet God vroech en spade, Wie sal ghenoech // in eenigher voech, Lof prijs en eer // doen onsen Heer, Van dese schoone saken // vant vermaken Doort sien // van dien, Die totter kennisse raken. Tot lof en danck // der lippen gheclanck, Laet ons gheheelijck begheven, Vry ende vranck // met vrolijck ghesanck, Altijt so langhe wy leven, Oock hert en sin // van dit ghewin, Want hy is goet // die't wassen doet, Bidden dat hy ons soare // en beware, Na dit ghecrijt // met hem subijt, Ons hem ghelijck verclare.
