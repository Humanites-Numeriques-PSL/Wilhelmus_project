 Een dienaer, lichte claer // Godes schaer Als ghetrouw, helt // stelt, Menich jaer // heeft hy haer voorspelt, T'oordeel swaer // hoe't sal zijn ghevelt, En vertrocken // met wat wapen. Dat die bocken // van de schapen, Ras en snel // totter hel // wreet en fel In den eewighen gloet // moet, T'sal zijn wel // lustich ende soet, Die 't bevel // van den Heere doet. Hier na hoort // sijn geboort // sach men voort Anno vijfthien // schien, Nae't accoort // metten vromen lien, Menich oort // heeft hy moeten vlien, Men bedreven // d'uytvaert clachtich, Alsmen schreven // tweentachtich, Doen hy smertich // had goedthertich // een en dertich Iaer dat God verlient // dient, Duysent partich // ende die ghetient, Mede wartich // met Christo verient. Hy blies den hooren // om te spooren // Godes tooren, Dien hy over t'lant // sant, Wel te vooren // dreychde hy Godes hant, Snel verlooren // ginckt aen elcken cant, Want sijn reen // die met beven, Van't ghemeen // volck naet leven Sou ghevaet // zijn met raet // is versmaet, En niet op ghepast // vast Godes raet // is meenich een last, Die soet praet // is haer liever gast. Een goede strijt groote spijt // in sijn tijt, Heeft hy wel ghestreen // leen Claer verblijt // vrymoedech ghetreen, Den loop wijt // totten eynde reen, T'gheloof ghesont // int benouwen Tot alder stont // vast behouwen, Dat met recht // zy ghesecht // trouwen knecht Nu sal zijn die croon // t'loon, Niet en vecht // teghen dese throon, En voorlecht // t'volck veel woorden schoon. Laet doch af // dat ghy caf // ende draf Heerlijck voor goe weyt // spreyt, Een ander straf // u nu toebereyt, Denckt om t'graf // dat ghy niet en schreyt, Als die Heer // comen sal besoecken, Werck en leer // na luydt der boecken, Dat eersaem // uwen naem // na 't betaem, Wert ghevonden daer // claer Te zijn bequaem // als ghetrou dienaer, Met hoogher faem // in Godes bewaer. Ernstich weest // ende leeft // na den Gheest Dese spreucke claer // naer, Aldermeest // dien hy was te swaer, Ende preest // hem noch openbaer, Quaet quaet // seytmen int leven, Twelck staet // Proverbi twintich schreven, Alst vergaen // van die baen // en ghedaen Is, so prijstment weer // seer, Wien vermaen // eyscht dan daedt by leer Moet vergaen // ja oock self die Heer. Noemter een // int ghemeen // die met vreen In sijn tijt gheleeft // heeft, Isser gheen // my ghelijck hier gheeft, Soeckt gheen leen // niet langher en sneeft, Moyses leet // dreyghen om te steenen Elck Propheet // moste dickwils weenen, Waerom want // alle schant // door het lant Men al den Propheet // weet Wech van cant // een yder creet, Wie haer hant // boodt creech mee een smeet. Eere groot // strack men boot // na die doot Die men dee ontgaen // traen, Menich stoot // heeftmens aenghedaen Nu isset noot haer wegh moetmen gaen, Och hoe groen // wie sal dees ghesellen, Schricken doen // voor die pijn der Hellen, Want waer met // niemant bet // weet Gods Wet, Elck vermaent met spoet // soet Haer ghebedt // ende wercken goet, Als trompet // men oock weten moet. Roemt niet meer // van een leer // die ter neer Door u selfs ghestort // wort, Soeckt gheen eer // om dit leven cort, V begheer // tot het eewich port, Datmen u niet // gaet verwijten, Dat ghy siet // als d'ypocrijten, Ghemaniert // dat ghy viert // en versiert, Der gherechter lijck // dijck, En u tiert // als van deuchden rijck Bedroeft gheviert // wort bevonden slijck. Toont doch niet // datmen u siet // met verdriet, In des Werelts doen // spoen, Dat ghy u biedt // sterck // met woorden coen, Ende vliedt // t'werck // om smenschen soen, Die werelt tiert // en heeft behaghen, Ia sy viert // die Heylighe daghen, Prijst haer leer // met begheer // ende eer, En die nu noch die daedt // vaet, Ende weer // oprecht leven gaet, Drucktse neer // ende wort ghehaet. Bovwen ghy // oock als sy // even bly, Den want in valscher schijn // blijn Spreket vry // het is die medecijn, T'bleeck hoort my // aen haer quaet fenijn, Israel // die so ginck leven Is rebel // gantsch verdreven, Wt haer lant // t'alder schant // haer vyant Veel haer bloot // vergoot // root Berooft verbrant // met een slach of stoot, Dat niemant // oyt sach meer der noot. Sulcken noot // hoewel groot // ende snoot, Daer God Israhel // snel Mee besloot // grouwelijck en fel, Was maer bloot // eenen schadu spel, T'ware wesen // is gheleghen So wy lesen // Marcus neghen, T'worm niet sterft // men beerft // en bederft, Eenen eeuwich vuer // suer, T'welck doorkerft // gheest ende natuer, Niet meer verft // vreest t'eeuwich ghetruer. Och hoe veer // vliedt dan eer // alsmen teer Al des werelts staet // laet, Aen den Heer // t'ontijt soeckt ghenaet, Och hoe seer // weynich die 't recht vaet, Onder die 't // woort Gods dienden, Menich niet // die't recht minden, Immers gheen // hoort mijn reen // dan alleen Timotheus daer // waer Paulus gheween // ende nu oock swaer Achtme cleen // nauwelijcx een haer. Eenen staet // ende raet // ghelijcke daet, Als vant begin ghesweeft // heeft, T'selfde maet // men als nu noch gheeft, Even quaet // als voor die mensche leeft, So treffen necht // alle die Schriftueren Effen recht // tot deser ueren, Ia all' ellent // en torment // God haer sent Die nu so onbedacht // lacht, Die hem oock went // uyt alle sijn cracht Blijft ongheschent // want God hem wacht. Neemt in danck // desen sanck // u verlanck Zy so veel ghy meucht // deucht, Poocht u ganck // d'outheyt als die jeucht, Om eer lanck // t'erven d'eeuwighe vreucht, Crijcht ghy yet // hier bedencken Aen dit liedt // wie't mach schencken Iou bericht // gheeft u licht // dit ghedicht Noemt hem in dit vars // dwars, Al diet sticht // vast staet // klemt als hars Int quaet swicht // ontgaet // Satans pars.
