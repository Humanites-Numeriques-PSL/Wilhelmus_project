 ALle vleesch is hoy, na Schrifts belijdt, Kort is den tijdt, van smenschen leven, Als bloem in't veldt, ontploken wijdt, Moet hier subijt, sijn glory sneven, Wat is met den Mensche bedreven? Een uyr en is hem niet ghegheven, Hoe langhe hy // sal leven vry, Sijn perck hebt ghy // o Godt beschreven. Dat oudt verbondt // is stervens noodt, End' nae den doodt // wy al verwachten Noch eenen dagh des oordeels groot, Daer werden bloot // alle ghedachten, Als Christus komt met sijn heyrkrachten, Sullen huylen alle gheslachten Der Aerden quaet // veel drucx toestaet, Al die den raet // Christi verachten. Voor den Rechtstoel Christi ydoon, Op den persoon // werdt gheen aenmercken, Die 't herte kent, sonder verschoon, Sal gheven loon // elck nae sijn wercken, Kracht en sal niet baten den stercken, Noch geen kloeckheyt den wijsen klercken, Recht oordeel klaer // sal openbaer Strijcken aldaer // dat hooft der Kercken. Seer vreeslijck Godt hem wreken sal Over 't getal // syner vyanden, Die de Wereldt beminnen al, Want haren val // wert groot ter schanden, Haerlieder vyer sal eeuwigh branden, Haerlieder worm die blijft voorhanden, Rouwe, gheween // werdt daer ghemeen, Angst ende een // knerssen der tanden. Dan sullen die rechtveerdich zijn, Vry van ghepijn // ofte weeklaghen, Blincken ghelijck der Sonnen schijn, Met vreuchden fijn // niet om ghewaghen, Int Rijcke Gods, nae haer behaghen, Onsterffelijcke Croonen draghen T'Lammeken soet // sal haer tot spoedt, Der tranen vloedt // van't aenschijn vaghen. Een Princelijcke Majesteyt Heeft Godt bereydt // sijn uytverkoren, Dus is des loons groot onderscheyt, Vast toegeseyt // ende ghesworen, Tusschen die Godloos Godt teghen sporen, En die hem dienen wedergheboren, Wat neersticheyt // men hier toe leyt, Gheenen arbeydt // en werdt verloren.
