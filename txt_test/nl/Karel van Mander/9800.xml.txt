 V wel bequame // Bruydt, Door uwen Name // uyt Helpt rechten wonder // dinghen, In u macht, stercken // Heldt, Men 'svyandts wercken // velt, En al kan t'onder // bringhen. In uwe groote // macht, Eer ick de bloote // slacht, Helpt my toerusten // stadigh. Want tis van noode // wel, Op dat ick doode // snel Altijdts de lusten // schadigh. T'gheloove t'eenen // schilt My altijdts leenen // wilt, Want meer dan stalich // desen My dient in elcken // tijdt, En oock in welcken // strijdt, De hop' om saligh // wesen, Dien Hellem vryde // my Oock aen mijn zijde // zy 'sGheests zweert, de waerheyt // mede, Tis eenen goeden // Kreeft, Die reyn ghemoeden // gheeft, In suyver klaerheyt // vrede. In liefde vyerigh // wis Die goedertierigh // is, Wilt my rechtsinnigh // gorden, Mach sy my blyven // by, Ick sal beklyven // vry, En overwinningh // worden.
