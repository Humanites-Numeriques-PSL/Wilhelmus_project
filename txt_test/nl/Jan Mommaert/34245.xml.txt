 WAer neemt ghy soo haestigh henen uwen gangh, Seght my Galathé door uwen soeten sangh? Bestiert u voetjens // en gaet wat soetjens, ’k Groet u met gewenschte dagen langh. Wel wie spreeckt my hier soo minnelijcken aen, Zijt ghy dat Tyter? Waer henen wildy gaen? Ick gae om melcken // mijne Koeytjens, welcke Ick wedd’ sy my al loeyende wachten staen. Galathé,, leydt my mee, moght het my gebeuren. Tyter maet // ’t waer geen raet // yemandt moght ons speuren. Wie sal ons hier sien by een? ’t Veldt heeft oogen in ’t gemeen. ’t Zijn al quae gedachten // maer. Maeghden moeten wachten // haer. Laet my u geleyden Door de groene weyden, Op dat ick u mijn trouw en min verklaer. O Galathea, mijn schoone Herderin, V eerst aen-sien brocht my ’t minnen in den sin, V schoon lichame // reyn en bequame, Heeft willigh my gebrocht tot uwe min. Jae Tyter, en let u anders niet? Ghy en soeckt my maer te brenghen in ’t verdriet, Sulcken kluchten // die sal ick wel vluchten, ‘k En sal u Herder licht gelooven niet. Schoon Jongh-vrouw // kreegh ick nou // troost tot mijnder wonden. Tyter siet // geen verdriet // is oyt aen u bevonden. Ghy weet beter dan ghy praet. ‘k Raed’ u Tyter dat ghy gaet. Galathé, zijt niet soo wreedt, Ick en doe u doch geen leedt. Is dat geen leedt, ’t Is ’t meeste dat ick weet, Dat yemandt oyt een sterffelijck mensch aen-deedt. Hoe menigh-mael heb ick u verhaelt Vw’ deughden, en uw’ schoonheyt af-gemaelt? Hoe menigh-mael gepresen // u eerbaer wesen, Waer door ghy mijn hert ende vryheydt staelt? Hoe menigh-mael liet ick mijn schaepkens al Gaen dolen, oft gestolen in den stal, Om u te spreken? soudt ghy dat wreken Met een on-waerdigh weer-geschal? Tyter vliedt // ’t is om niet // ghy moet my verlaten. Ey! waerom // liefken kom // laet ons noch wat praten, Ick moet by mijn Koeyen // gaen. Laet die noch wat loeyen // staen. Neen Tyter, men wacht naer my. Dies te beter acht men dy. Och! ick moet scheyden. Neen, ghy moet wat beyden, Den dagh die is noch vroegh genoegh van ty. En sult ghy dan noch nimmermeer V wreedt gemoedt tot my-waerts keeren weer? En sal ick u sinnen // niet eens verwinnen, Daer ick u bemin op deught en eer? Jae Tyter, soud’ ick soo jonge teere Maeght Soo licht gelooven de klachten die ghy klaeght: Wegh ware lusten // rouw dat waer mijn rusten; Want sulcken bedrogh de herders meest behaeght. Schoon jongvrouw // eer ick sou // de geloften breken, Van mijn trouw // liever sou // ick mijn hert af-steken: Den Oliphant sou vlieghen eer, Eer ick mijn sinnen van u keer; Och mijn Lief, mijn troost, mijn al. Tyter, tiert u niet soo mal. Ey! en wilt niet klagen, Tot u behagen Ick noch een vrolijck Liedtjen singen sal. Alhier in dit genoeghelijcke dal, Vol bloemetjens schoon, by desen groenen wal, Daer dit klaer stroomken // langhs menig boomken In ’t midden wispelt met een soete val; Hier sagh ick eerst mijn schoone Galathé, Wiens glants scheen helder als de nieuw-gevallen snee, Wiens eerbaerheden // ende brave zeden Mijn hert met heete min ontsteken dee; Haer blont hayr // gaf sy daer // den wint mee te speelen, Haer gelaet // metter daet // mijn vry hert quam steelen, Haer ooghskens bloncken bruyn van verr’, Veel schoonder als de klaerste sterr’: Och! vrouw Venus, hoogh-geacht, Seyndt door Cupido uw’ kracht, Dat mijn Galathea Nu eens segge mee Ja, Voor al mijn droevigh lijden voor en na. Ja Tyter, hoe zydy soo gewondt? O Schalck en Vos, hoe speeldy met den mondt! Gut, hoe kondy queelen // en de herten steelen Van de Maeghden met een lieffelijcken vondt! Ja, mijn Galathea, ’t is al ’t oude liedt, Mijn singen, en mijn springen, dat daer door geschiet, Is om te vermeeren // uw’ lof, prijs, en eeren, Noch en wint de liefde uw’ jonste niet, Boefjen snoodt // ick word’ roodt // Gut hoe kan jy praten! Ey! waerom // Liefjen kom // ‘k sal u niet verlaten. ‘k Blijf te langh van huys, eylaes! Galathé maeckt geen geraes. Wel het schijnt of ’t dond’ren wil. Neen, het weer is wonder stil; Och! mijn hoop’, mijn leven, Wilt my nu troost gheven, En maeckt doch eens een eynd’ van ons geschil. Bemindt doch, o mijn Galathea braef, Die u trouw dient als d’aller-meeste slaef, Ghy krijght in handen // alle mijn landen, Mijn Ossen, Geyten, Schapen, en mijn haef. O! Tyter, swijght, het is genoegh geklaeght, Ghy wint victori’ op my jonge Maeght, Ick ken de krachten // van mijn gedachten, Mijn Moeder moet eerst zijn om raedt gevraeght. Neen, jongvrou // onse trou // knoopen wy hier t’samen. Tyter, wat // seght ghy dat // ghy hoort u te schamen. Het wordt op eer en deught gebouwt. Tyter, stil uw’ handen houdt. O mijn Galathea soet. Ick sal roepen soo ghy ’t doet. Roept soo ‘t u seer // doet, Swijght soo ick ’t meer // moet, Een kusjens en ontneemt u eer noch goedt. O! soete Galathea, mogen wy Eens gedurigh blijven zijd’ aen zy, En dat niemant scheyden // en mocht van ons beyden, ’t Waer een hemelsch Paradijs voor my. Jae Tyter, hier mede soo gae ick deur, Ghy mooght my volgen: Adieu, dat gaet u veur; En al ons praten // wilt verborgen laten, Op dat wy niet en komen in ’t getreur. Galathé,, op wat stee // komen wy te samen? Tyter vriendt // hier toe dient // dese wey bequame: Adieu Tyter, goede rust. Galathé, noch eens gekust. Mijn Moeder die sou kijven // och! Wat sou sy meer bedrijven // doch? Och mijn Galathé, Liefjen ick gae mee, Ons liefde laet doch leven in peys en vree.
