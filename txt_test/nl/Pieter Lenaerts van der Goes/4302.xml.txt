 MInsaem soete natuere // Die my contrariert, Hoe seer dijn willecuere // Ootmoedich wert begeert, Toogt heden // maer // Een reden // claer // Waerom ghy rebelleert? Een wtgelesen // vrouwe Scat ick naer weerde // dy, Om wien ick desen // rouwe, Lijdsaem aenveerde // vry: Geen lieuer ieught, Die my verheught, Op geheel d'aerde // sy: Sal my maken mismoedich, Der wonden een, of twee, O neen! ick strijde bloedich Tot inde laetste wee: Al briesschen som Vyandich om My, als een wilde zee. Verhaert vileyne // tongen Verberght u wreet aenschijn, Vipers onreyne // iongen, Serpentich boos fenijn: V paden // gaet, Geen quaden // raet, Sal hier van weerde zijn. Edel, ia magnifijcke Beelde, pointich gewracht: Theseus ick my gelijcke, Doch Ariadne slacht: Soo werd ick uyt Den Dool-hoff ruyt, By uwer hant gebracht. Roup ick aldus om bate, Princesse, altijt meer: Sijt my een honighrate, In al mijn cranckheyt teer, Daer vooren sal, Ick over al, V schoonheyt loven seer.
