 CRachtich // noyt Geen gesont // hert gewont, Als het mijne onverschoont: Clachtich // oyt Mijnen mont // gans den gront Mijnder sielen heeft verthoont, Aen een ionghe maeght, Die my iaeght // onversaeght, En gans desperaet // laet, Tis om niet geclaeght, Haer behaeght Desen mijnen droeven staet. Al dat in Mijnen geest // onbevreest Plach te leven sonder noot: Heeft den sin Aldermeest // met tempeest Vast benout, tot inde doot: Eylaes! wie sal my Maken vry // nu dat sy My houdt inden last // vast? die my gaen voor by Roupen sy, Over my ellendich gast. T Herte mijn Sucht en treurt // angstich scheurt, In benoutheyt schier versmacht: Moet het zijn Dus besuert // my gebuert (Arme weese) meer dan cracht: Heb ick metter daet Haren raet // oyt versmaet, Of geacht te cleen: // neen? Maer als delicaet Toeverlaet Haer ter noot bekent alleen. Anders geen Troost soo soet // die t'gemoet Geeft de ruste principael: Dan dat reen Edel bloet // die my doet s'Daegs suchten thien dusentmael, Valt sy my wat swaer Corts hier naer // wert troostbaer Al wat sy beleeft // geeft: Sulcx betrouw' ick haer Die voorwaer Een goet hert te my-waert heeft. Rustich sal Ick met vlijt // zijn altijt Dienstbaer vast in haer verbont, T'sy of al Mijn profijt // storm, en strijt, Wesen sal, ter laester stont: Daer my anders niet Dan verdriet // en geschiet, Wacht ick goet accoort // voort: Soo sy my aensiet, Ende biet Een gesontmakende woort. In mijn hert Schoonder kint // noyt gesint, Niet aensiende druck, of noot, Thisbe wert Vierich int // hert bemint Van Piramus totter doot: Maer dit excelent, Partinent // bloemken ient, Climt in hooger graet // iaet! Ben ick een moment Haer absent Tis een leven desolaet. NAer dien sy Even vast // groijt en wast In mijn hert sonder af gaen, Vind' ick my Aengetast // overlast (Boven reden) aengedaen: Ick moet even wel Mijn gequel // achten spel, Wat baet veel geclach // ach! Al valt sy rebel, Niemant el Mijn versuchten eynden mach.
