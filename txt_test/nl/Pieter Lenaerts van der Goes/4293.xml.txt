 NAer dien dat ick moet // Subiect, en dienstbaer staen, En mijn vryheyt soet // Geheel te buyten gaen, Meer onderdaen // Dan my rebel // verthoont // Heb ic, naer recht // Alsulck een knecht // Wert namaels wel // geloont. Edel, dier // en // weerdt, Is zy, die t'hert aensiet: Al dreyght vier, en // sweert, Ick stae tot haer gebiet, Deed'- ick sulcx niet, Soo waer ick fel // gehoont: Wie twijffelt dies, Sulck tijt verlies, Wert namaels wel // geloont. Een alsulcken ieught Blijf ick eygen voortan, Wert het eynde vreught, Soo sit ick edel man Als Coninck van Gans Israhel // getroont, T'sy hoe dat gaet, De minste daet Wert naermaels wel // geloont. Liever smert // en leedt, T'aenveerden, angstich swaer, En haer t'zijn gereet Een knecht, drie seuen iaer, Soo werd' ick naer, T'gewonnen spel // gecroont: Dat dunct my sienst, Getrouwen dienst, Wert namaels wel // geloont. Keert sy mijn labeur Te zijden, als yet cleens: My arm serviteur Toelaet luttel gemeens: T'sal werden eens Dat mijn gequel // verschoont, Want die altijt Verdraeght, en lijdt, Wert namaels wel // geloont. Es oock meerder last, Vervaerlick elcx gepeys: Dat een dorper gast, Geniet der minnen cheys, Venus paleys Sonder bevel // bewoont: Och! t'is te cleyn, Dat sulck vileyn, Wert namaels wel // geloont. Niet versagen // sal Mijn Princelick gemoet: Liever dragen // al, Des weerelts tegenspoet Want diet my doet, Is so ydoonen maeght, Voor cleyn geween, Ghy namaels geen Soo wel beloonen // saeght.
