 AErdich reyn // gelaet, Int wesen amoreus, In geen pleyn // en staet Sulck bloemken pretieus, Schoon maget smal, Die wesen sal Geheel, en al // mijn vreught: Laet my (by uwer deught) Met u eens zijn verheught. Een eerbare // bruyt Zijt ghy, die wonder doet: Als int clare // luydt, V voysken honigh-soet: Mijn droufheyt groot (Soo swaer als loot) Als vander doot // verrijst, T'is recht datmen u prijst, T'bevinden sulcx bewijst. Laet dan henen // gaen Vileyne quade becx: Houdt verdwenen // staen T'gepeyns vol van gebrecx: Met u alleen, (En anders geen) Salt algemeen // doch zijn, Cause hier van, is dijn Goddinnelick aenschijn. Kiest (O vrouwe // reen) Geen ander lief oprecht, Want in trouwe // geen Soo vast, als uwen knecht: Mijn hert geheel (Als een iuweel) Sal u tot deel // geschien: Dus lief, wilt my by dien Genadelick aensien. Est dat heden // mijn Versouck, u eens behaeght: Geen voorleden // pijn En wert van my beclaeght: Noyt heb ick schat, Soo lief gehadt, Als u reyn vat // ghy zijt Van God (tot aller tijt) Meer dan gebenedijt. Neemt (Princesse // vry) Dit liedt, in u gedult: Mijn Meestresse // ghy Voort altijt wesen sult: Des herten gront, Getuyght den mont, Waer by ghy cont // verstaen, Dat dit simpel vermaen Wt liefde is gedaen.
