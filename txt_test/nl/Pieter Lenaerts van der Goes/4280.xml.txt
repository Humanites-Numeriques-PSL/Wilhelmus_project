 CEderboom jeughdich groen // plantsoen // Deurluchtich paragoen // u doen // Is soo betrout // Waer ick ter doot benout // Ghy sout // zijn mijn rantsoen. Abel, en magnifijck // gelijck Pallas, die gans aertrijck // publijck Sticht int gemeen, Zijt ghy beelde alleen, Want geen // soo autentijck. Troostbaer in tegenspoet // onsoet Sydy, als t'quaet gebroet // my doet Suchten vervaert: By dien vreught openbaert T'beswaert // treurich gemoet. Hierom ben ick perfect // gewect, Een dienaer t'sijn subiect // gestrect, Voor u: soo mach Eynden (naer langh verdrach) T'geclach // secreet bedect. Als ghy (Liefste) dit wrocht // my docht Ghy waert soo wtgesocht // ten mocht Schoonder Robijn Onder der sonnen schijn, Niet zijn // soo dier gecocht. Ront om d'aerdtsche tonneel // geheel En is liever Juweel // tot deel Gewenscht voorwaer, Dan u, die zijt een swaer Dierbaer // gulden parceel. In mijn hert noyt gegront // en stont Sulck lief, wiens rooden mont // my iont Troostich geniet, In alle mijn verdriet, Aensiet // ghy my gewont. Nu ghy my diligent // bekent, In dienst, geen tijt absent // gewent, Schinct my (voor loon) Wt uwen hoogen troon Een schoon // coronnement. Aengenaem edel kint // gesint, Naer dien ghy u dus vint // bemint, Jont uwen knecht Bystant, dat hy met recht, T'gevecht // eens overwint.
