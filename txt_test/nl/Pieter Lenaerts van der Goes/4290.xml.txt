 O Soetvloeyende be-ke // Reyn speciale // dracht: Iont (troostbaer apote-ke) // Medecinale // cracht: En maect mijn quale // sacht // Want hier ter werelt geene // En heeft dan ghy alleene // De principale // macht. Schoon lief, in wien ick stille Heusche maniere // vant: Ontfermt (ist uwen wille) T'hert, dwelck in viere // brant: O schoon Lauriere // plant, Al doet ghy my dees smerte, Noch schinck ick v mijn herte, Ist niet een diere // pant? Dit pant, als cleen van weerde, Ghy luttel gade // slaet: Ick moet hemel // en eerde, Clagen, dees quade // daet: Lief, eens te rade // gaet, Of ghy geen troost zijt schuldich, V dienaer, die gehuldich Naer v genade // staet? Al ist, dat ick ontsinne, Ick heb om clagen // noot: Want onvergolden minne Moet ick nu dragen // bloot, Liever geslagen // doot, Dan te minnen met pijne, En niet gemint te zijne, T'is een mishagen // groot. Wat baet dat aertsche weelde Is om verwerven // goet: Als ghy (o schoonste beelde) Mijn herten sterven // doet, Cruyt, noch conserven // soet, Connen geen droufheyt scheuren, Al s'weerelts vreught is treuren, Als ick u deruen // moet. Princelick sal ick blijuen V dienaer euen // fijn: Al wildy my ontlijuen, En voor troost, geuen // brijn, Coemt met verheuen // wijn, En laeft my, o Meestresse: Want ghy sult mijn Princesse Nu, en naer t'leuen // zijn.
