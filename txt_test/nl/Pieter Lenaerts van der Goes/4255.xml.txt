 MInnelick schoon // gelaet, Rancke mijns leuens weerdich: In Venus throon // bestaet Ghy, als een blomken aerdigh: Mach my (tot lof) Dijn schoonheyt eens gelucken, Ick sal v af plucken Voor t'schoonste inden hof. Aerdiger geene // maeght En was, die smerten bluste: Dies ghy alleene // draeght T'confoort, der herten // ruste, Deurschoten // lief Heeft my u zeed-baer wesen, Sent my doch t'genesen, In een gesloten // brief. Ingelick weder // paer, Voor swaer tempeest, en donder, Zijt ghy den Ceder // daer Mijn ziel sal rusten onder: Der sonnen // schijn, Passeert ghy int geheele, Valt ghy t'mijnen deele, T'sal veel gewonnen // zijn. Keert dan der pijnen // vloet In perfecte gesonde: Want ghy van mijnen // moet T'secreet weet, tot den gronde, En cleeft altijt Op thert, als crachtich seghel: O volmaeckten spegel, Die sonder schaduw' zijt. Een sulck behagen // vast Draech ick u Jeught // ter // eeren, Geheel der plagen // last Cont ghy in vreught // verkeeren, Ter schanden // laet V dienaer niet bederuen: Mijn leuen, en steruen, In uwen handen // staet. Nobel deesse // soet, In danck, wilt dit aenvaten, Mijn schoon Princesse // moet Ghy zijn, sonder aflaten: Hier mede // schrijft Mijn hant, dat u ontmoeten Hondert duysent groeten, En voorts in vrede // blijft.
