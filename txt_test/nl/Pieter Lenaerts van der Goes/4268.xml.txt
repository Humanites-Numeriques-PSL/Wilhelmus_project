 PRompt, en volmaect // Is mijn wtvercoren // Soo wel geraect // En was noyt Helt geboren // Alvoren // can Hy wech, en regel // schrijven // Op t'herte sal dan // Dees jongen man // Voortan // als segel // blijven. In een loyal Hert, heb ick besloten: Nu dat den strael Der minnen is geschoten, Genoten // zijn Moet met hem vreugd' // en // weenen, Ken achte gepijn, Ick sal t'sanguijn Aenschijn // in deugden // nemen. Eenpaerlick staet T'concept mijner sinnen, Dat ick (naer raet) Hem niet en sal beginnen Te minnen // small, Maer als mijn heere // croonen: In spijte t'getal Der tongen al Ick sal // hem eere // thoonen. Troostbaerder // geen Inder Goden troone, Al waerder // een Adonis in persoone, Zijn schoone // glans Moet moet noch menich // buygen: Want, hy draecht den crans Der ionger mans, Dits gans // mijn eenich // tuygen. Een Paragoen Is hy, vroom gesegent: Van dien Baroen, My alle troost beiegent, Genegent- // heyt Theimwaerts, can geuen // smerte, Maer laet met bescheyt, V zijn geseyt, Hy leyt, my neuen // t'herte. Rustich, en fray, Moet mijn Prince varen: Den Rijnschen bay Wensch ick hem sonder sparen: Geen swaren // moet Wil ick ten fijne // maken: Dit gracelick bloet, My altijt doet, Meer soet // dan pijne // smaken.
