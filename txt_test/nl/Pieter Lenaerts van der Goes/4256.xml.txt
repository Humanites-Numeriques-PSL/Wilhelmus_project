 EErbare, soete // ieught, Bloemken weert alder eeren: Dat u ontmoete // vreught, Wensch' ick, int salueeren: Ghy zijt voorwaer Een Peerle onder vele: T'elcken avont spele Is uwen naem lofbaer. Laet my dijn name // gaen Vol prijsen naer haer weerde, Siende bequame // aen Hoe ghy betreet de eerde, Seer minnelick Moet mijnen geest ontspringen: Ghy zijt in u dingen Volmaect, godinnelick. Ionge volmaecte // bruyt, Beleeft in u faconde< Noyt man en smaecte // fruyt Soo soet, tot zijn gesonde, Veel smerte // dect Ghy, deur troostbaer genesen, V amoureus wesen Liefs hert aen t'herte // trect. Siende dijn wangen // root, Int herte ick verschiete: Ghy sult den strangen // noot Dijns liefs, haest doen te niete: Als hy sulcx merct, Ghy bluscht hem al zijn pijne, Soo de medecijne Een crancke siele sterct. Acht dit de waerheyt // al, Noch meer mocht hier ontbreken: Niemant de claerheyt // sal (Dijns lofs) geheel wtspreken: Doch euen wel Al zijt ghy weert de croone, Maeghden fris, en schoone, Zijn meest haer lief rebel. Bloemkens van geure // soet, Connen veel schoonheyts dragen: Maer u coleure // moet My bouen al behagen, Vaste pilaer, Manierich, ende stille, Coemt u liefkens wille In alle heusheyt naer. Es t'uwen houe // rijck, Aengenaem dit gedichte: Soo ist wel loue // lijck Dat ick een schoon gesichte (Deur dit cleyn werck) Int minste mach believen: Ick sondt anders brieven, Waer ick ter cunsten sterck. T'is tijt om scheyden // nu, Dits dan mijn supplieeren, God wil geleyden // u: Tot dat wy wederkeeren, Princesse net Wilt ons vrientschap niet mincken, Alst u sal gedincken, Peynst om my altemet.
