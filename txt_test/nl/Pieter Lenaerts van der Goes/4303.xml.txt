 CVpido god der minnen // stout // Ghy // geeft my // Int hert menich wonde, En in bedwanck mijn sinnen // hout // Dat // ick mat // Ben versmacht te gronde, Noyt en conde // my ontmoeten // Troost voor een verslegen // moet // Ander ionde // ghy t'versoeten // My, veel drucx en tegen // spoet. Als ghy te mywaerts wrimpich // siet, Lijdt // en strijdt Mijn ieught overstreden: Ghy zijt die my so schimpich // liet Van // elck man, Als een worm vertreden: By wat reden // hebdy voren My te dooden sonder schult: Laer verbeden // zijn dijn toren, Naer u goddelick gedult? Tuwaerts moest t'allen dagen // oyt Mijn // aenschijn Hem subiect verschoonen: Nochtans ghy my veel lagen // strooyt Ach // ick mach Suchten voor t'beloonen: Om soo schoonen // kindt der eeren Lijd ick veel, die my can doen Deur haer Croonen // triumpheeren, Als deurstreden Campioen. Heb ick Apollo dan // geslacht, Met // opset, V geschut verschoven: O neen ick niet! wie can // de // cracht Gaen // weerstaen, Diet al gaet te boven? Princen hoven // u geboden Nemen met groot wonder // aen: Dies u loven // alle goden, En u macht zijn onder- // daen. Al lijd ick dan een smerte // wreet, Lof // hier of Moet u zijn gesongen: Venus (t'moederlick herte // heet) Heeft // beleeft, Vwen schicht deurdrongen: Hoe bedwongen // ghy elck slave, Int prisoen van suchten leydt: Als ontsprongen // wt den grave, Geeft my voetstaens liberteyt. Reden? beveelt met billich // recht V // alls nu Vroom te adoreeren: Ick dan, als een goetwillich // knecht, Naer // d'Altaer Wil ick t'aenschijn keeren: Supplieeren // daer soetmondich, Om gehoor, tot mijn propoost, En begeeren // seer hertgrondich, Eens in liefden t'sijn getroost. Ist dat u gramschap neder // daelt, Is // gewis T'eynden mijn vernoyen: Mijn aenschijn bleeck // dan weder // haelt Een // seer cleen Schoon rosierich bloyen: Weeldich groyen // is voorhanden, Al was ick seer straf // gevaen: Kerkers, boyen // ende banden, Zijn subyt my af // gedaen. NAer dat ghy (als machtich // princier) Slaet // en laet Wederom genesen: Regiert deur u crachtich // bestier, Haer // eerbaer Princesselick wesen, Die gepresen // ieught bepeerelt Heb ick wtvercoren // ras, Wtgelesen // noyt ter weerelt Soeter kindt geboren // was.
