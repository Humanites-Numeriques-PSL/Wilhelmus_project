 PLach mijn jonck hert Dus te versuchten // swaer, Gelijck het nu begint: Soo dat my smert Volght, met veel duchten // naer, Om een maeght wel bemint: Geen soo gesint Te voren // oyt, Geen schoonder kint // en is geboren // noyt. Een sulcken last Doet my der minnen // godt, Dat ick vertwijffelt stae: Of sy my vast Sal doen gewinnen // t'lodt Daer ick om dolen gae, Ick overslae Mijn smerten vry, T'blijft even nae // geen troost en is voor my. Troostbaerder was Sy ten beginne // dan Sy my nu dunct te zijn: Hoe can soo ras Sulcken vriendinne // van My keeren haer aenschijn? Meerder gepijn Doolyck bereyt, En heeft noyt mijn // jonck hert neder geleyt. Roup ick om troost Wie sal verhooren // my, T'is al om niet geweent: Voor mijn propoost Verstopt haer ooren // sy, Rebel, en hert versteent: Siet my vercleent Arm serviteur, Ander verleent // sy troost, en my rigeur. Van Antheros, Is sy deurschoten // sterck, Wiens schickt vercoelen biedt, Dies Atropos. Met een gesloten // zerck Fineert al mijn verdriet: Misdeed' ick iet, Dat sy my haet, Och neen ick niet // dan een reyn hert toestaet. Scheyd' ick dan met Een deerlick schromen // ach? Wt desen weder-spoet: Van sulck op set Sy haer beromen // mach, Als een maeght straf verwoet, Mijn reyn gemoet Niet falen // sal, Dwelck met zijn bloet // t'rantsoen betalen // sal.
