 MAeghden blomme, suyver parceel, Die v verheuen // stelt: De hooft-somme // gans, en geheel, Sydy, daer leuen // gelt: My quelt // t'gewelt // der sinnen, Van binnen // s'herten gront, Want geene Soo reene, Alleene, ghy cont Doleuren Doen scheuren, Voor treuren // terstont. Al dijn vroetheyt // goddinnelick, Soo hemelsch af gemaelt, Is een soetheyt // gans minnelick, Te vollen noyt verhaelt, My daelt // en straelt // ter herten Veel smert // en // ongeval, Achermen! Mijn kermen Ontfermen // wilt al, Of sterven, Bederuen, Duyst-werven // ick sal. Rancxken aerdich // moet noot // en // recht, Doch soet van aerde // zijt, My (onweerdich verstooten // knecht) Een sauegaerde // zijt: Naer strijt // sendt tijt // van vreden, Zijt heden // in mijn cruys Soet- // mondich, Goet- // grondich, T'bloet- // hondich // gespuys, Laet drijuen, Zijn blijuen, Catijuen // confuys. Int versuchten // naer troost // en // raet, Ligh ick gans kercker // vast, Nu mijn duchten // ten boosten // gaet, Den brant noch stercker // wast, Dien last // deurtast // my schandich, Vyandich // met veel grief: O rare, Troostbare, Int sware // miskies, Van desen, Wilt wesen, T'genesen // schoon lief. Als idoone // Princesse // my Int laetste secoureert: O mijn schoone // Deesse // vry, Mijn hulpe mainteneert: Cesseert // geleert // verstandich Mijn brandich // lijden straf: Of! sonder, Groot wonder, Ick onder // int graf Moet dalen: En halen, T'betalen // hier af.
