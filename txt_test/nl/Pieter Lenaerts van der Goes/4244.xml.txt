 CLouck ingenieus // volmaectste aller flueren // Vroom, en gratieus // in gauen der natueren // Eenich boeten // en versoeten // daert al by verheught // u t'ontmoeten // ende groeten // is een soete vreught. Al wat sonder // u Geschiet, en can niet treffen: Ligh ick onder // nu, Ghy cont my haest verheffen, By gedoogen, Wilt my thoogen In mijn wederspoet, V bruyn oogen, Of verdroogen Moet mijn vleesch, en bloet. T' waer wel reden // dat Ick u van minnen / sprake: Als gestreden // zat Ick brandich binnen // blake // Wtgelesen, Hoogh gepresen, Beeldeken geeert, V by wesen Can genesen, T'swaerste dat my deert. Hebt in handen // ghy De macht, dat (naer t'volherden) Al dees banden // my Een vreught connen gewerden, Wijt vermaerde, Jonck geiaerde, Sijn my (alst u lust) Een vermaerde Sauegaerde, Daer mijn herte rust. Als ick schade // loos Aen u vinde ontsetten, Alle quade // boos, En connen geensins letten: Wat verscheyden Tongen seyden, Acht voor schimp, en spijt, Wilt verbeyden V bereyden Dienaer, taller tijt. Ramp, en schade // dan Acht ick voor windigh blasen, V genade // can Mijn droufheyt versolacen: T'soet beluchten Can doen vluchten, Dees bruinen swaer Dat mijn duchten, Treuren, suchten, Moet verdwijnen // claer. Int vertrouwen // vast, Van u deught-saem beloonen, Geen berouwen // wast, Van mijn gedienstich thoonen, Hadde connen Zijn gewonnen, Van u, yet gesienst, Onder sonnen Waer begonnen Noyt geen meerder dienst. Neemt (O weerde // maeght) Dit danckbaer int geheele, Al wat d'eerde // draeght, Schinck ick tot uwen deele Daer beneven Lijf, en leven, Is u, metter daet, Toegeschreven, Overbleuen Niet, dat my aengaet. Adieu vreden // borght, T'scheyden heeft nu oorsake, Voor u heden // sorght Den geest, altijt in wake, Dit bedrijven! Laet nu blijven V contentement, Voor te schrijven Veel meessijven, Wensch ick tzijn present.
