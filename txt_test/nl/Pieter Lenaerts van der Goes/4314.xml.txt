 COmt Aura lief // V soetheyt nu ontmoete // my, In desen brief // Val ick absent te voete // dy, Een groete // sy // V van herten daer neven // Geschreven // Laet geschieden // Mijn ontbieden // Die my smerten // cont // Wilt af vlieden // Sulcke lieden // Die u rieden // My te wieden // Wt der herten // gront. O f ghy my als Ballingh, van u gesichte // drijft, Op mijnen hals Dit jock (swaer van gewichte) // blijft, In dichte // schrijft Mijn hant noch even seere, V eere, Wat can schaden Der genaden Tijt, verwachten // langh Ben beraden, Met versmaden Dus geladen, T'gaen den quaden Ongeachten // gangh. Rayende licht Ontsprongen inden morgen // vrough, Mijn siel verplicht Aen u, altijt met sorgen // iough, Verborgen // drough V overgroote minne: Goddinne, Laet der schanden, Niet verbranden, My, met groot // verwijt: Spijt vyanden: Kerckers banden, Deur u handen. Werd' ick vanden Swaersten noot // bevrijt. Naer redens recht V lief, Laurier, en croone // vought: Wee zy den knecht, Die yet op dijn persoone // wrought: Hoe schoone // lought Ghy, in ons eerst genaken, Wy spraken Soo besneden, Soeten reden, Sonder schamen // yet: Waer is heden Dien voorleden Tijt getreden, Die met vreden, Ons versamen // liet? Een uerken cort Dunckt my gelijck veel iaren // zijn) Wanneer my wort Ontvreemt soo hemelsch claren // schijn: Hoe swaren // pijn, Ick draeght in uwen name: Eersame, Wat ghy dochte, Ick volbroghte, Oyt al even // vroet: Wtgesochte, Welgewrochte, Diergecochte, Peerle, mochte Ghy my gheuen // spoet. Leef ick int vier, Deur u, als den Salmander // doet, Cupido schier T'hert af, met zijnen brander // wroet, Leander // moet Ick volgen, niet gedincken T'verdrincken: Deur mijn poogen Sal ick boogen S'waters stercken // vloet: Wilt medoogen Vanden hoogen Toren, toogen, Dat mijn oogen, V licht merckten // soet. Iason verslough Den Draeck, die hem fel tegen // blies, Heeft inden plough De Stieren wreet geslegen // dies Vercregen // t'vlies: Cause was d'uytvercoren Alvoren Dat ghy ionde Sulck ghesonde, Sonder scheyden // van, Quets', noch wonde, Deeren conde, D'aertsche ronde, Dienstbaer stonde Onser beyden // dan. Als magnifijck Princesse, ghy ten thoone // sit, Ja puer gelijck een duyfken zijt: idoone // wit, Mignoone // dit Ontfangt (van my arm crancke) In dancke: Soo sal desen Brief, int lesen, V behagen // meest: Dijn gepresen, Hoogh geresen, Soet by wesen, Sal genesen, Mijn verslagen // geest.
