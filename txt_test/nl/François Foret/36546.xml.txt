 In Jesus naeme // Broeders eerzaeme, Stelt u eenpaer // Nu te beginnen Hem te beminnen // Al in dit zoete Nieuwjaer. Een herte reyne // Ook anders geyne, Dit 's al voorwaer // Minnen alleene 't Kindeken kleene // Al in dit zoete Nieuwejaer. Hij heeft geleden // Hy is besneden Dit 's openbaer // Wilt u besnyden Zonder vermyden // Al in dit zoete Nieuwejaer. Zyn heylig sterven // Genoegte derven Dits' openbaer // Die doet verwerven Dit hemelsche erven // Al in dit zoete Nieuwejaer. Och Maria schoone // Bid uwen Zoone, O Maget klaer // Den God hier boven, Wy eeuwig loven // Al in dit zoete Nieuwejaer. Vader almagtig // Zone eendragtig, Heylig Geest klaer // Een God Dryvuldig Verheft ons schuldig // Al in dit zoete Nieuwejaer.
