 EEn Liet // wil ick gaen beginnen, 't Bediet // sult ghy oock wel beminnen, Want al mijn hart en sinnen Die winnen // nu niet dan vreught, Dus schenckt en brenght 'tHert denckt // en wenckt // op u reyn jeught. Ick moet // dit nu schoontjens uyt vegen, Het voet // de mensch in alle wegen, Dus moet // ick dit te degen Gaen legen // en klaren uyt, Volght na // seght jae, Het gae seer dra al in mijn huyt. Dit glas is nu ledigh ghedroncken, Seer ras sat 't worden gheschoncken, En niet langh laten proncken, Maer loncken // hier aen mijn zy, Op dat // het nat, Int vat // ons mach maken bly.
