 LOf prijs en eer, sy v O Heer // en milde Vader Die voor den morgen, niet te sorgen, hebt bevolen, Dewijl ghy alleen // En anders gheen // ons allegader Lijf en Siel voet // ende doet // hongher en dorst vlieden Voor de weldaden // en ghenaeden, die ons gheschieden, Eyscht ghy maer // Een hart danckbaer // willich en bereyt Dus t'allen ty // Motten wy // Oock alle lieden, God hier boven // danckelijck loven // inder eeuwicheyt.
