 V Minne gunst // is t’overkunst O gey,e Herderinnetjes, En t’ooghenblick // is als een strick Voor ons verliefde sinnetjes. Als ick int groen // eens dertel soen V nat bedaude lipjes, Soo vind ick mijn // gheseylt te syn Aen twee Corale Clipjes. Daer ick dan koom // tegen de stroom Van de ghebruysde baertjes, Soo dat ick voel // doort soet ghewoel Een bevingh in mijn aertjes. Ick steeck het sweert // en span de gheert Nu styfjes dan weer slapjes, Dan clim ick op // tot byde top Vander Hooft-touwen trapjes. Daer ick de schoot // vast aende boot Belegh met strick en knoopjes, En deur het nat // maeck ick weer rat De stramme overloopjes. En sien ick weer // als ick laveer Nu hoochjes dan weer laechjes, Dat ick niet win // soo vind ick in Een reefjen voor de vlaechjes. Tis tovery // siel rovery En soete Boereknaepjes, Als wy met u // int Bosjen luw Gaen houden middach slaepjes. Daer ghy met ons // den brant des Sons Ontschuylt in diepe holletjes, Daer ghy ons solt // daer ghy ons rolt Als kittelsiecke snolletjes. Het schijnt dat ghy // gheen vryery Ghebruyckt, maer toverkunsjes, Dat ghy kort // deelachtich wort Ons lieve Minne gunsjes. En Coridon // myn Goude Son Ick haet al myn Vriendinnetjes, Wanneer u beelt // soo haestich speelt Int diepste van myn sinnetjes. Wanneer ick voel, het soet ghewoel En twemfelen uwes tonghetjes, Myn hertjen teer // doet op en neer Veel Minnelijcke sprongetjes. Het nockt, het beeft // het lockt, het leeft Nu laechjes dan weer hoochjes, Het tuycht en spreeckt // wat myn ghebreeckt Deurt dryven van myn ooghjes. Maer als ick wen // en beter ken Tusschen door beyde rotsjes, Soo peur ick an // ghelijck een Man Tot spyt van Thetys trotsjes. Daer myn de vloet // strackx dryven doet Tusschen twee hooghe baeckjes, Waer ick met vlyt // strackx buyten smyt Het Ancker met twee haeckjes. En als myn hert // heel quackjes wert Doort wellen vande stroomjes, Soo legh ick aen // een sloech te gaen Naer de belade Boomjes. Maer Ameril // wat coude gril Cruypt door myn moedeleetjes, Als ick opt strant // gae deur het sant Met wanckelbare treetjes. Myn dunckt de Zee // waer in ick tree Staech welt onder myn voetjes, Dan vloetse seer // dan ebtse weer En hippelt oock seer soetjes. Het klaecht tot dat // ick weet niet wat Voor wonderlijcke cluchjes Door ons geslacht // soo weerdich acht Verdryven al myn suchjes. O Coridon // mijn Goude Son Hoe syt ghy nu dus stilletjes, Is dan u lust // soo haest gheblust Van u verhitte willetjes. Wiens inghewant // int eerste brant Soo vierichjes, soo graechjes Als of de vreucht // van uwe Jeucht Sou dueren duysent daechjes. Maer als terstont // u vast verbont En Ionckend soete lachjes Verkeerden in // gheveynsde min En trouweloose clachjes. Eerst vluchten wy // doe suchten ghy Met meer minne sanghetjes, Nu vlucht ghy schuw, nu soeck ick u Met nat betraende wanghetjes.
