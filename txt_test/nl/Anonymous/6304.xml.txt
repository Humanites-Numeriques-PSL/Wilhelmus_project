Ghy vlucht te zeer // ik begeer // doch geen meer, Als dat ghy // loonen wilt mijn lang gevry. Heelt de smert // die my 't hart // heeft verwart In u min // en verkeert eens uw harde zin. Ach! ghy hoord niet na mijn leet! Waarde Maagd', hoe is uw hart zo wreedt? Keer, ay! keer // Stut uw loop // vlucht geen meer, Maar vertoeft // dat mijn ziele sich niet meer bedroeft. Is zy ontvloon? // groote Goon // 't zal my doon; Neen zy keert // en is van de min verheert. Ga en blust // nu uw lust // en haar kust Voor haar mond // smeekt dat zy eens geneest uw wond. Koom, mijn schoone harderin, En beloont mijn trouw' en zuyvre min.
