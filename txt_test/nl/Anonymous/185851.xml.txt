 Ave Maria // Gods liefste lief ydone Der maechden kronen // Maria weest gegroet Gratia plena // claer als de sonne schoone Int Hemels troone // vat vol genaden soet Der deuchden volle vloet // een suyver hert ootmoet Hebt ghy gebaert // Gods soon ons hoochste goet. Dominus tecum // die heer sy met v eene Suyver fonteene // vrou alder werelt wijt Benedicta tu // in mulieribus alleene Al waert ghy cleene // ghy zijt gebenedijt Geen vrou te geender tijt // so salich als ghy zijt Noch so verheven // van Godt tot vyants spijt. Et benedictus tuus // gedanckt, gelooft, ghepresen, Moet ghy altijt wesen // dat hoochweerdich pant Fructus ventris tui // dijn vrucht reyn wt gelesen Wt Ieffa geresen // Iesus Christus playsant Die als een Heer valjant // ons brac der hellen bant Ende die gevangen brocht // in haer vaders lant. Sancta Maria Princesse // hooch verheven krone Edel geboren // van koninck Davidts saet Ora pro nobis // bidt voor ons die te voren Waren verloren // verdoemt door sonden quaet Tot dat ons quam te baet // Iesus onse advocaet Van u geboren // o schone dageraet.
