 Ghy Christelijcke lieve jeucht, Cleyn spruytjes, jonck van jaren, Laet varen // alle ydel vreucht Verheucht // in deucht So veel ghy meucht, En wilt u Ziel bewaren. Bewaert u Ziel en sorge draeght Dat ghy niet wort bevanghen, Van slangen list die menich plaeght, Maer jaeght // en vraeght Wat Godt behaeght, Beveelt hem al u ganghen. Let op u ganck, en zijt geen kint Al in ‘t verstant onvroedigh, Maer spoedigh // soeckt en ondervint Hoe blint // sy mint // en heeft ghesint De wel-lust wanckel-moedigh. Wel-lust bedrieght als sy verblijt, ‘t Iolijt // is niet ghedurich Doleurigh // haren dienaer krijt Als tijt // is quijt // dan heeft hy strijt In sijn gemoet seer treurich. En daer ‘t gemoet so knaegt en wroet Onsoet // is sulcken leven, Bedreven // en vol anghstigh bloet, Maer doet // ghy goet // in u gemoet Wert hoogen troost gegeven. Wie desen troost is toe ghevloeyt Besproeyt // van ‘s Heeren zeghen, Ter degen // hy in deughden blpoeyt Hy roeyt // en snoeyt // al watter groeyt Door ‘t vleysschelijck gheneghen. Gheneghentheyt is den aenvanck Des sonds, dus wie het quade heeft stadelijck in sijn bedwanck Lofsanck // en danck // van die ontfanck Bewijs hy vroegh en spade. Neemt aen in danck dit cleyn gedicht, Zijt den Leeraer der scholen Bevolen // die u onder-ricjt Hy sticht // verlicht // en geeft ghesicht Om nimmermeer te dolen.
