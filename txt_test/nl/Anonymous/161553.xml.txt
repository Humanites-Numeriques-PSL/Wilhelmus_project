 PArst uyt 't ghesicht een droef ghetraen, Laet big'lend' langhs dees kaecken gaen Bracke vloedt // die ick moe Op d'aerde gieten Met naer ghesucht, helaes om een, Wiens harde hert blijft als een steen, En niet acht // nae de klacht Van mijn verdrieten, 't Gheen ick staegh // alle daegh Doe voorwaer // of ick haer // noch hier naer Hier door eens mocht ghenieten. Doch 't schijnt mijn smeeckingh sy begeckt, Of dat haer borst een rots verstreckt: Want wanneer // ick begeer Met soete woorden Haer aen te spreecken, stuers en straf Draeyt sy haer wesen van my af, En gaet heen // of mijn reen Sy niet en hoorden, Jae het schijnt // dat sy vijnt Hier in rust // en haer lust // werdt gheblust Met dus mijn ziel te Moorden. Wel Amarille seght my doch Wat hebt ghy voorghenomen noch Voor mijn trouw // waerde vrouw Mijn te doen erven? Wat is het gheen, seght Harderin, Dat ghy met mijn hebt in u sin? Sleghts ghebiedt // daer is niet, Lust u mijn sterven? Of begheer // ghy noch meer, 'k Ben te vreen // wilt my leen // vry van een, Van lidt tot lidt doen kerven. Is 't dat u wraeck dan meerder dorst: Soo komt en opent vry dees borst, Wt sijn stee // rockt vry mee Mijn trouwe herte, Klooft het aen tween, en siet Goddin V eyghen beeltenis daer in, 't Gheen ghewis // d'oorsaeck is Mijns vreughds en smerte, Ghy sult sien // datter gien Minnaer trou // is Me-vrou // of ick sou In liefd' hem durven terten. Want wat ghy wilt, ick ben te vreen, Ick vrees gheen straf, hoe groot, o neen: 'k Ben gherust // als gheblust Werdt u begeeren, Want Lief, mijn leven en mijn doodt Bestaet in u vermoghen groot, Wat ghy doet // 't is al goet, 't Kan mijn niet deeren, 't Gheen u haeghd // waerde Maeghd Doet dat vry // laes aen my // 'k sal my bly Nae u behaghen keeren.
