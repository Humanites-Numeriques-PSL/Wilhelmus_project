 GHebenedijt // Weest altijt // O Heere almachtich Die al dat leeft // voetsel gheeft // door v ghenaden In Christi naem // ons lichaem // niet alleen sijt ghedachtich Maer met v woort // rechte voort // ons siel wilt versaden Van eyghen rust // en wellust // ons hebt ontladen, O Heere wijs // gheeft dranck en spijs // die met v vereent, Sulcx dat wy niet // misbruycken yet // van de weldaden Die ghy goedich // overvloedich // ons mildlijck verleent.
