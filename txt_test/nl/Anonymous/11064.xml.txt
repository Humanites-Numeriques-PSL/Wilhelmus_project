 EEn danckbaar liet // der Heren Heer aenbiet Die op ons neder siet // om weder te ontladen Van t'groot verdriet, dat velen bracht te niet Door swaren siecte hiet // zeer qualick omversaden En merckt hoe zeer // Dat God den Heer // De Menschen teer // verblijden, Als hy sandt neer // Ghenade weer // Wie zal zijn eer // in deesen nieuwe Iaar belijden. Wy die in noot // van hulpe scheenen bloot En telcker uur de doot //zeer broesselijck wacht Door vreese groot // veel volckr ons ontbloot Om sulcken swaren stoot // t'ontgaen met haer geslachten Maer onverstant // Doet buyten t'lant Haer oock met schant // versuchten, Want Godes hant // En can niemant// Hoe seer vayliant // tot geener tijt van t Iaar ontvluchten. De mensch, gemien // wel spoedich tracht t'ontvlien De plaetsen daermede sie // dat God bereyt zijn plagen Maar t'punt van dien // waar door de straffen schien Wy uyt ons niet en wien // dat meest is te beclagen, Wou elck met boet, Het quaet verwoet, Wt zijn ghemoet // verdrijven, De Heere goet, Liet metter spoet, Die straf ontsoet // int nieuwe Iaar wel achter blijven O Mensch verheeft // zijn lof die eeuwich leeft Door dien ghi met en bleeft, in dotelick verstrangen Zijn hulp aencleeft // ootmoedich u begheeft Voor hem daert al voorbeest, dat leven heeft ontfangen De wijl hy staet, En u omvaet, Om troost en baet // te geven Die sich verlaet, Op zijn ghenaet, En metter daet // int nieuwe Iaar zijn woort beleven. Hierom besnijt // u hart en ziel met vlijt En wilt tot alder tijt // des Hemels Prins verbeyen U schult belijdt // en maeckt u alles quijt Dat teghen t'ghoede strijdt // soo suldy u bereyen Om met bescheyt // De Salicheyt, Daar elck na beyt // te hooren, Als Christus zeyt // Wert in gheleyt // T'rijc is bereyt // voor u mijns vaders wt vercoren.
