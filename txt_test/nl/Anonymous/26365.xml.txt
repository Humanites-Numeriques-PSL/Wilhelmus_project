 O Christus lief minnare // soet, Die my schinckt u dierbare // bloet, Nae u verlanght my t' allen uren, Soo den Hert sijnd' in vare // doet, Nae de Fonteyne clare // vloet, In dorstich pijnlick besueren, Ick leve in sware dolueren Hoe langhe, Lief, sal dit ghedueren Wanneer sal ick uyt rouwen, sijn, Wanneer sal ick aenschouwen, dijn Aenschijn, schoon boven naturen? O lief, in s werelts wijcke // niet Van my in gheen versijcke // vliet, Maer helpt my, dat mijn teere leden, Niet en buuyghen ghelijcke // riet, Maer van u soete Rijcke, siet My hier in dit dal beneden. Want ick vinde my bestreden, Van den Prince der duysterheden, De werelt in t' verstranghen // quaet, Alle dat nydich Slanghen // zaet, En laet my niet met vrede. Jonstich voor my gheneghen // hecht, Leydet my in uwe weghen // recht, door uwen Gheest, o Lief verheven, Want siet ick ben bedeghen // slecht, Een teecken daer elck teghen // seght, Voor raserye acht men mijn leven, En ick nochtans becleven Wt uwe zyde sonder sneven, U lief, u Bruydt, u Kercke // reen Ick doe door u, O stercke // steen, Die Helsche poorten beven. O Prince der Princieren //jent, Ghy die herten en nieren // kent, Aenhoort mijn bitter claghen, Uwen Gheest goedertieren // sendt, Dat hy door zijn regieren // went Mijn ledekens nae u behaghen, Want t' sijn nu quade daghen, Hierom, o Lief moet ick u vraghen, Sal ick van al dit lyden // vry, Met u niet haest verblyden // my, En s Levens Croone draghen?
