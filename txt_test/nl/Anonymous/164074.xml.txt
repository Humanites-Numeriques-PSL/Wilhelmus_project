 DEs Werelts pronck is als een water-bel, Die ick seer schoon den mensch voor oogen stel: Olyme bedroefde Vrouw / u zielens rouw, Klaeght vry aen de Goon / gist'ren stont ghy schoon, Gist'ren stont ghy schoon / verheven, En als een Bel ghy nu moet sneven: Helaes wat baet my nu d' Egypsche kroon? Wat glory ken Alexandrias troon Nu Cleopatria zijn // meer als fenijn, Nu ick inde hant // Cesars mijn vyant, Nu ick in zijn hant moet glyen, Door 't moedigh en manhaftigh stryen. O ghy Jupijn verheft u Donder niet, Ooghd' op mijn staet, en pynelijck verdriet. Voorkomt de mynen val // ghy grooten al // Ey straft mijn niet meer // siet u Maget neer // Siet u Maget neer gebogen // U treurigh smeken om medoogen. Medoogen hebt bid, ick met een slavin, Die wel eertijts was 's Werelts Monarchin: Medooghd' o Hemel my // dat bidden wy // Spaet dit machtigh rijck // mijn volck al gelijck // Spaert dit machtigh rijck // o Goden // Roeyt uyt die ons pooght uyt te rooden.
