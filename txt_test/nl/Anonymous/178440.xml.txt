 Comt ter feesten // suyver gheesten Godts dienaren groot, Die daer boven // in Godts hoven Sijnen lof verbreyt // tot alder tijt. Helpt ons singhen // groote dinghen Van een suyver Maeght, Die ons’ leven // heeft ghegheven Brenghende hier voort // Godts eeuwigh woort. Hoogh ghepresen // uytghelesen Maeght en moeder reyn, Naer u wenscyhen // alle menschen, V soeckt groot en cleyn // in d’aerdsche pleyn. Laet doch dalen // uwe stralen Der bermherticheyt, Wilt u ooghen // naer ons booghen Keert van ons het quaet // o toeverlaet. Heel deurwonden // deur de sonden, Maeckt ons al ghesont En u daeden // vol ghenaeden Toont ons al ghemeyn // als medicijn. Al ons crachten // en ghedachten Zijn tot uwen lof, Elck een prijst // en bewijst V alderhooghste eer // reyn maghet teer. In de sure // laetste ure Als men sterven moet By ons staet // metter daet En neemt in u behoet // Maria soet.
