 ICk moet een liedt, in's Heeren naem beginnen, Op dat ick jeet // mocht an u Ziel gewinnen: Al mocht ick dy // de gantsche Werelt geven, 't Heeft geen weerdy // by een Godtsaligh leven: Der Zielen Goet // is't hoogste goet gepresen, Een reyn gemoedt // ontfanght Godts Heylich wesen. Wilt gy sijn een // verkoren Bruyt des Heeren, Tot hem alleen // moet gy u liefde keeren, Vergetet al // u Vaders huys en magen, Den Koningh sal // in u hebben behagen, Maer ist dat gy // Egipten wilt gedencken, Soo en sal hy // sijn liefde u niet schencken: Ieloers is Godt // en met liefde ontsteken, Selfs wil hy tot // ons Ziel inwendigh spreken, Sijn Bede-huys // ter eeren sijnen name, Moet reyn en kuys // sijn, an Ziel en lichame: Geen sond' onreyn // staet hier vry om te dragen, Daer Godt alleyn // regeert na sijn behagen. Scheydet u van // de Wereltlijcke lusten, Roertse niet an // die u Ziele ontrusten, Laet sijne Geest // een woningh in u maken, d' Hemelsche feest // sult gy dan lieflijck smaken. Sijn Avontmael // sult gy met hem genieten, Soo gy sijn tael // u niet en laet verdrieten. Sulck een beloft, heeft ons den Heer gegeven, Wie dit versloft // die mist het eeuwigh leven, In duysternis // wandelt hy als de blinden, Seer ongewis // waeyt hy met alle winden: Maer die verlaet // des werelts woeste wegen, En doet Godts raet // die heeft de rust verkregen.
