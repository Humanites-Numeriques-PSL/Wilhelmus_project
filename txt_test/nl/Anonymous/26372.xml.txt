 DEn Heere triumphant // zy prijs en danck ghestade, Die bysonder wonderlijck zijn volck bevrijt, Want met zijn stercke hant // om Israel voor schade Te verlossen // rossen // wagenen subijt, Wierp hy omme sonder sparen // so dat d'Egyptenaren Dreven op de baren // aen den oever doot, Die Israeliten waren // dickmaels verlost van haren Vyanden, end' beswaren // angst ende noot. Christus Hoochpriester groot // heeft om eeuwich te blijven, Teghen zonden // vonden // een verlossen goet, Door zijn eyghen bloet, root // tot een salich beclijven, Hier voor elcken // welcken // hem ontfangen soet, Haer wort de macht ghegeven // kinderen Gods te leven, En Godt heeft verheven // zijn nieu verbondt, Met die so zijn gedreven // om eenicheyt t'aencleven, Nae Gods Wet geschreven // in't herten grondt. Al wijcken ongesont // veel argerlijcke leden, Die verdwenen // henen // gaen en d'eenicheyt Des Geests verlaten ront // ja en den bandt des vreden, Dat haer laetste // t'quaetste // wort so die Schrift seyt, Godt blijft noch sonder mijden // by zijn volck t'allen tijden, Die hem oprecht belijden // voor dese sal Hy schermen en strijden // sy en sullen niet glijden, Al viel oock bezijden // een groot ghetal. Al waert dat heel end' al // die weerelt t'onder ginghe, Al verdroncken // soncken // berghen ind' Zee, Soo sal, vry van den val // de stadt, s'Heeren wooninghe, Noch gherustich // lustich // blijven sonder wee, Het Wijse-mans ghestichte // onbeweghelijck dichte, Op des steens ghewichte // niet en vergaet, Dat vredelijc gesichte // zijnde vol claren lichte, Met oordeel, gherichte // eeuwich bestaet. Dat huys Gods delicaet // lieflijck met soet behaghen, Davids hert en sinnen // binnen // waren daer Jae hadt zijn toeverlaet // om daer zijns levens daghen, Met verheughen // meughen // blijven eenpaer, Om daer sonder verflouwen // den schoonen Godsdienst t'aenschouwen, Met een vast betrouwen // in Godt vervroyt, Maer die bezijden bouwen // haer aen't hooft niet en houwen, Sy comen in rouwen // seer wijdt verstroyt. Princelijck Herder noyt // en verliet ghy u Schapen, Die u woorden // hoorden // ghehoorsaem fijn, Met u te zijn gekoyt // al haer genoechte rapen, Door't volherden // werden // eeuwich termijn, Gheluckigh en ghepresen // van alle druck ghenesen, Een weyd' uytghelesen // in s'Hemels pleyn, Vinden sy boven desen // vol rusten onverknesen, Haren dranck sal wesen // s'Levens fonteyn.
