 Als Palmen groenen // seer suyverlijc, En Cederen doenen // seer heerlijck, So moet ghy bloeyen // in deugden rijck, Wassen en groeyen // seer lieffelijck, Op dat ghy o jeught // in ware deught, Bly en verheught // altijt recht lichten meught. * Laet in u schynen // dat ware licht, En heel verdwynen // dat u ontsticht, So moet ghy bloeyen // in deugden rijck, Wassen en groeyen // seer lieffelijck, Op dat ghy o jeught // in ware deught, Bly en verheught // altijt recht lichten meught. Stelt u ghelaten // in stilligheyt, En wilt aenvaten // goetwilligheyt, Soo sal dan op dijn // vloeyende zijn, Godes Geest fijn // alst water uyt den Rijn. Op dat ghy o jeught // in ware deught, Bly en verheught // altijt recht lichten meught. * So sal dan vloeyen // van u eerbaer, Stromen die vloeyen // seyt die schrift claer, Hoe sijnen gave // heeft die ontfaen, Geen rijcker grave // en vintmen saen, Dan die mens reyn // ootmoedig cleyn, Die soo alleyn // drinckt uyt ‘s levens fonteyn. * O reyne ziele // die soo gheplant Staet in priele // schoon en playsant, Wel rijck gesegent // zijt ghy voorwaer, Want op u reghent // voor ende naer, Van God seer goet // uyt genae soet, Des gheestes vloet // waer voor ghy wassen moet. * O jeught bequame, in sulcken hof Wens ick u t’same // om daer Gods lof Recht te verbreyden // als spruyten jent, Vol vruchtbaerheyden // seer sijn ge-ent, Aen die wijnranck // vol soeten dranck, Daer niet uyt spranck // dan God tot lof en danck.
