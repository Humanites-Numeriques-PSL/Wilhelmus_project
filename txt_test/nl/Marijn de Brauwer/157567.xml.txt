 O Heere Godt van machte // groot Voor u sijn mijn gedachten // bloot Daer en is niet soo ergen // man, Die hem voor u verbergen // kan Want ghy binnen en buyten // siet Men kan voor u opsluyten // niet. Tis voor u als den lichten // dach Gheeft in my dat stichten // mach Eenen nieuwen bequamen // geest, Met al die uwen name // vreest Dat ick mach hier beneden // pleyn Wandelen in der vreden // reyn Sonder de vrede seer dier // baer En sal men mogen sien hier // naer God in zijn eeuwich wesen // fijn De knecht sal niet gepresen // sijn Die met onvrede quellen // gaet En zijn medegesellen // slaet Met stock ofte met roeden // stijf En meen ick op dat goede // lijf maer met der tongen haesten // praet Spreken van zynen naesten // quaet Soo met der tonge griefde // rat Maer niet en leert de Liefde // dat Al trock ghy ooc een blodich // mes Tegen u vriendt die nodich // es En u goet principalick // riet Soo doet ghy noch soo qualick // niet Als dat ghy met versmaden // maeckt Hoe men in kromme paden // raect Die zynen mondt tot laster // wendt Den vre hy veel tonvaster // kent Want na Syrachs gewagen // goet Hy doet van al zijn dagen // boet, Die hem tot qualijck spreken // gheeft En selven vol gebreken // leeft Heere bewaert my in de // pat Naer die reyne beminde // stadt Mijn tong' en my beneven // dien Dat ick hier na mach tleven // sien En goede dagen mede // voort, Och! Heere mijne bede // hoort Heer mijn gebedt verhooren // wilt Dat daer door uwen tooren // stilt Over alle mijn sonde // boos Door u genade gronde // loos Vergeeft my dit altsamen // vroet Want ick my voor u schamen // moet Laet my Heere voor u vry // gaen Van sonden als den Publy // kaen Die voor u inden Tempel // bat Geeft my na dat exempel // wat Dies ghy na u verklaren // vast Quijt maeckte van den zwaren // last. Of ick u soo behaeghde // reen, Heere ghelijck als Maegde // leen, Doen zy verkoos dat beste // deel Scholt ghy haer quijt de reste // heel Der sonden tot versoeten // dies Sy met tranen u voeten // wies. Veel biddens zy met tranen // de Oock dat vrouken van Cana // ne Sy badt met groot betrouwen // snel, Voor haer dochters benouwen // fel Ootmoedich en van herten // leech Van u zy haer begheerte // kreegh, Haer dochter deet ghy Heer bystant Sy was gequelt van den vyandt Deed haer veel persequeren // an Maer ghy saecht dat verneren // van sMoeders geloove machtich // sterck Soo toonde ghy u krachtich // werck, Seer veel krachtighe daden // soet Heere ghy deur genaden // doet Wie tot u met geloove // quam Melaetsche, kreupel, doode // lam, Den blinden sach, den stommen // sprac Ghy hebt van haer genomen // tpack Och! Heere God bermhertich // vaer Den man die achtendertich // jaer, Opt Bedd hadde geleghen // kranck Hebt ghy kloeckheyt en degen // ganck Wt liefden hem goet ront // ghejont, En maeckte hem terstont // Gesont. Princelicke God devijn // sijt mijn Een eeuwich medecijn // met wijn En Oly my inwendich // bint Waer dat ghyt Heer elledich // vindt In my een gheest verslaghen // schept Daer in ghy een goet behagen // hept.
