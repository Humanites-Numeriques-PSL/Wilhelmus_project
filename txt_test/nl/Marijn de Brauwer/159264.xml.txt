 KNechten des Heeren looft den Heer die hemel en Aerde heeft gemaect Wilt zynen naem verbreyden seer Die met hem uyt de doot sijt geraect, En oock smaeckt vriendlijck dat Die Heere is elck // goet ghelijck het water nat, Vloeyt van hem Wijn en melck // soet Comt doch al die daer dorst en drinct, Alle beyde Wijn en melck om niet Die Heere seer mildelick schinckt Want zijn Fonteyne altijt over vliet, Want hy anbiet zijn gena Dus komt met vlijt // smaeckt, Doet naer zynen ra Eer dat ghyt al quijt // raeckt Elck moet hem so besteden // fris Te lange toeven ist lopen quaet So lange alst met ons heden // is Zyne genade altijt open staet Want haest vergaet hier den mensch Als roock verdwijnt // snel Niet naer haren wensch Alzoo men dat bevint // wel. Stellen niet uyt tot morgen // maer Soect den Heer hier int misprezen // dal Soo komt ghy niet in zorgen // zwaer gy en wit niet wat morgen wezen // zal Haest tot afval // komt die blom Met haer coleure // jent, So die menschen som Qualijck zynen speure // hent Princelijck God van machte // groot Ghy waert, en sijt, en ooc blyven // sult Veel sijn van ons geslachte // doot Want ghy haer met vel katijven // vult Hebt doch gedult met ons Heer Die doch van slijcke // sijn Wilt ons tuwer eer, Bringen in u Rijcke // fijn
