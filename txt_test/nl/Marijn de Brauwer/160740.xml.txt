 QUijtschel van sonden geeft de Heer Door zijn louter ghenaden Die hier af laten en niet meer Lust hebben tot den quaden Als een groete // laet hy soete Inde Schrift orconden // dan Komt met moete // neemt u boete Goetwillich voor die zonden // an. Ick en begheere niet den doot (Seyt de Heer) der sondaren Maer dat zy haer bekeeren soot Behoort, en laet varen Tsondigh sneven // dat zy dreven, t Sy heyden of Jode // boos 'K wil vergeven // dat zy leven Wie tot my komt hoe gode // loos Op dien daer van het quaet afstaet Wil ick niet als den scherpen Seer nauwe straffen zijn misdaet Noch hem oock niet verwerpen Wilt doch komen // met den vromen, Soeckt u ziele bate // ghy Hebt verschromen // voort verdomen, Eer dattet hier te late // zy Heb ic u niet gebeden // vrient, Dat ghy u zout bekeeren Tot my, en my met vreden // dient So wil ick u recht leeren Mijne wegen // zijt genegen U vant boos voorstellen // dwinght Dat mijn zegen // u verlegen Hier namaels uyt der Hellen // bringt Waeromme wilt ghy sterven // doch Vraegt God door die Propheten Keert weder om verwerven // noch Mijn goetheyt ongemeten U onthouwe // en mijn trouwe Weder aen u warghde // reen Boet en rouwe // heeft die vrouwe Bewesen, g'naemt Maeghde // leen Ick op der aerden niet en quam Voor die rechtveerdich waren Maer voor die daer ic meest vernam Lagen in 's doodts bezwaren Met veel wonden // onverbonden Heb ick onverholen // daer Tzelven stonden // wel ghebonden Met Mattheus den Tolle // naer Ist dat met u noch heden // heet Wilt u doch naer my schicken Zijn u zonden voorleden // leet So sal ic u verquicken Tot versmullen // wilt vervullen, Mijn Huys seer ootmoedich // bloot Sonden sullen // wit als wullen Worden al waer zy bloedich // root tVrouken van Canane seer droef Dat bat met seer veel smerten Die Heere gaf nae haer behout, Hy sacht geloof int Herte, Christum wrochte // en volbrochte Dat niet en misluckte // maer Voor haer Dochter // wiert al sochter Want den Duyvel verdructe // haer Den Pharaseen en Publicaen, Hadden vooren genomen In den Tempel om bidden gaen Die heeft hem gaen beromen D'ander wende // hem niet ende,/ En dede gestadich // vlijt, Hy bekende // zijn ellende En sprak Heer mijns genadich // sijt Die Publicaen quyt schelt ontfinck Van sijn Goddeloos wesen Rechtveerdich in sijn huys hy ginck Voor den ander mispresen Prince cierich // goedertierich // U deucht doet versachten // milt Die manierich // tot u vierich Comt, ghy niet verachten // wilt.
