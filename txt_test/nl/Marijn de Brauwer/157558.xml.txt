 O Heer God Hemels Vader // goet Die rechte Fonteyn-ader // zoet, Sijt ghy so Schrift oorcont Hier in de Werelt ront Zijt mijn sonden ontlader // vroet Want ick als een misdader // moet Bekennen mijne zondt Voor u uyt sHerten gront Die daer in onbetamen Van my wel sijn geschiet Dies moet ick my nu schamen Voor u Heer diet al siet Van in mijn jeucht // so had ic vreucht Tot alle quaet, gedreven // milt o Heer vol deucht // maect my verheucht Mijn schult vergeven // wilt Ghy hebt bereyt een schone // stadt Den wegh is ongewone // wat Te gaen, vol van dangier, Van water, en van vier O Heere wilt my jone // dat So den Verloren sone // badt Om vergevinge hier O Vader goedertier Want groot sijn mijn misdaden Die u Heer sijn bekent Vader vol van genaden Hoe was ick so verblent Als ic wegh ginc // zonder bedinck, en my tot boose wercken // gaf Maer u geschinc // was kleet en Rinc Beter dant Vercken // draf Den Herder naert behoren // riep doen ick lange verloren // sliep Gelijck een Schaep verdwaelt Van Gods Rijcke gefaelt Ick int zondich versmoren // liep En lagh so in Gods toren // diep Maer u liefde doorstraelt Heeft my weder gehaelt Wat sijn wy arme menschen Dat ghy ons zo gedinckt Al watmen goets kan wenschen Wt genaed ghy ons schinckt tot ons welvaert // u niet en spaert, Hebt ons met uwen bloede // root Gekocht vergaert // sijnde bezwaert Al met armoede // groot. Prince alder Princieren // dan Koning diet al regieren // kan In Hemel en op Eerdt Ghy sijt wel prijsens weert En loff, die ons soo diere // wan Al komt ons qua maniere // an Keert van ons dat ons deert dit wort van ons begeert Om niet meer so t' ondegen Te loopen voort en voort Maer hier op uwe wegen sterckt ons doch met u woort, Dwelc is u spijs // goet en propijs Tot onser Ziel bequame // fijn O Hertoogh wijs // vol loff en prijs Moet uwen name // sijn.
