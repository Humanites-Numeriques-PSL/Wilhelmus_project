 HEere zijt ons hender // by En die u geven // danck dat wy van die schender // vry Doch sijn ons leven // lanck Dat hy niet // wat hy ziet en bespiet // int verdriet Of houdt in zynen dwanck. Dat wy Vader // vol genaden Altydt nader // in u paden Sijn bevryt // van den nyt En den stryt // worden quyt Dat naer mach sijn jolyt. Den mensch van God geschapen // is Heylich goet groot // geacht door sDuyvels betrapen // wis Die heeft de doodt // bewracht Doen hy plat // overtrat En oock wat // daer van at En hielt zy geen goe wacht Want een bevel // Gods der goden Zy wistet wel // twas verboden, Dat zy wan // alleen van Den Satan // was den ban De doot die quam haer an Alzoo in doots gewelde // quam De Menschen alle // gaer Den Helt alder Helden // nam Ons uy den valle // zwaer Dien Heylant // heeft vermant Den Vyandt // met der hant Deylt uyt zijn rooff aldaer Den Leeu Juda // heeft verwonnen Dat goed' noch qua // doen en connen In ootmoet // quam hy soet Wat hy doet // tis al goet Verlost ons met zijn bloet Dits den rechten Herder // fijn Die sijne Schapen // voedt: Daer en kan geen werder // sijn Die ons vant slapen // hoedt Hy kan leen // dat niet een Groot noch kleen // en moet scheen Tot sWolfs gapen // verwoet Dan alleen dat // kint verloren Daer David wat // seyt te voren Twas Judaes // dat boos aes Dat eylaes // als een dwaes: Verriet sijn eygen baes. Prins alder Princieren // stam Van hoogher weerden // fier Als een goedertieren // Lam Quaemt ghy op Eerden // hier Hebt gehaelt // dat gefaelt Was verdwaelt // en betaelt Met uwen bloede dier Den ingang tot // d'eeuwigh ruste Is van Godt // gemaeckt tot luste Hy wil schoon // elck persoon Seer ydoon // tot haer loon Schincken des levens Croon.
