 OM dat daer gheen mensche zalich // wordt Deur eygen werck tallen stonden, O Heer op ons liberalich // stort, U genade niet om deur gronden Wie seyt hy sonder sonden // zy Leugen die gaet vermonden // hy Daer en is niemand gevonden. Ia noch hoeft men groot en kleyne // die, Prijs die God hoorde tontfangen Soo Paulus seyt in Romeyne // drie Soo lagen wy gevangen, In sonden na Paulus bediet // reen Niemant gerechtich ja oock niet // een Al verslonden in doots verstrangen. Neemt ons aen Heer den genade // groot stelt al onse schulden besyden die ons hebt vande quade // doot Verlost deur u doot en lyden Deur uwe liefde grondeloos, Genaest ghy al mijn sonden boos, Dat wy met u souden verblyden. Kennelijck ist dat ons allegaer door Adams overtreden Sijn wy gekomen ten valle // zwaer Maer deur u Heere in vreden In de vrede ons bewaren // wilt Op dat ons geen bezwaren // schilt Daerom Heere heb ick gebeden. Lam en kreupel, O Heere // mijn Laet my niet gaen hier beneven Dat ick mach sijn // deur u leere // fijn Ghesont int geloove verheven, Op dat ick hier in mijn Aerden // Vat Altijt beware dien waerden // schat Die ghy my o Heer hebt gegeven Berecht My Heere na u goetheydt bewaert mijn herte en sinnen dat strijt // met vlijt // onder den voetscheyt Ghy cond o Heere druck verdinnen Sterck my met u ghenadich woort En help my altijt ghestadich // voort Dat ic komme ter stede binnen Van Dese stadt die vol goeden // leyt, Vinden wy klaerlijck geschreven Van Esdras die zijn bemoeden // seyt En van Gods gest gedreven Al die hier overwinnen // saen Die sullen daer moghen binnen // gaen Om by u Heer eeuwich te leven Den Inganc tot dese reyne // stadt Is vol prijckel en dangiere een menschen trede is de kleyne // padt Beset met water en met vierre Hoe sal ick o Heer noch komen // daer Door desen wech, vol verschromen // swaer Helpt mij o Heer goedertijere Dalen en lechten ghedincken /// wilt Dat sijt oock moghen beseffen U goetheydt wilt haer schincken // milt Dat zy u waerheyt treffen, Ootmoedich met een verslagen // gheest Al die u na u behagen // vreest Die zult ghy hier naer verheffen Doch-ter wylen o Vader // rijck wy donder u niet en vermogen Wy sijn doch allegader // slijck En stoff die nergens toe en dogen O Princelijc God van machten // groot Voor u sijn mijn gedachten // bloot Siet ons aen met u genadige oogen.
