 Waeckt op, waeckt op ‘t is meer dan tijt En laet de werelt varen // jae varen, Maeckt u des boosen levens quijt, En wordt uyt Godt ghebooren // ghebooren. De lust des werelts moet vergaen, Heeft Ioannes gheschreven // gheschreven. Maer die den wille des Vaders doet, Die sal eeuwichlijck leven // jae leven. Gelt ende goet // en daer toe moet, Dat moet men weynich achten // jae achten, En volghen den Heere van herten goet, Dat eeuwich goet te betrachten // betrachten, Wie eenen Thoren bouwen fel, Die mach hem wel versinnen // versinnen, Overdencken eerst de kosten wel, Eer hy dat gaet beginnen // beginnen. Vleesch ende bloet mach niet bestaen, Dat moet men daerom wagen // jae waghen Want het moet al seer korts ondergaen, En sijn kruys moet men dragen // jae dragen. Het is den vleesch een sware pijn, Het en kan ‘t niet wel lijden // jae lijden, Als ‘t van de werelt veracht sal zijn, Dan sal de Gheest verblijden // verblijden. Nu gheeft my u ghenade Heer, En wilt my wijsheyt gheven // jae gheven, Dat ick mach wandelen nae uwe leer, Haten mijn eyghen leven // ja leven. De Stadt is seer lustigh en fijn, Daer in ick hope te komen // te komen, Maer daer moet veel om ghestreden zijn, Dat heb ick wel vernomen // vernomen. Veel Vyanden aen elcken kant, Zijn op dees enghe straten // jae straten, O Heer doet my nu onderstant, En wilt my niet verlaten // verlaten. Vleeschelijck ghesint zijn // is de doot, Soo betuycht Paulus leere // jae leere, Gheestelijck ghesint zijn is ‘t leven bloot, En vrede in den Heere // jae Heere. Daerom heb ick my korts bedacht, De waerheyt nae te jaghen // te jaghen, Niet meer te wandelen in de nacht, Godt alleen te behaghen // behaghen. Maer het staet niet in mijnder magt, Doen kan ick niet alleene // alleene, Daerom, O Heer neemt mijnder acht, Maeckte my dat herte reene // jae reene. Ick sal ‘t waghen door u ghenae, Op u beloften schoone // jae schoone, Op u beloften schoone // jae schoone, ‘t Lichaem moghen sy dooden // jae dooden. Dit Liedeken is uyt-ghericht, Tot den prijse des Heeren // des Heeren, Is ‘t dat daer yemant door werdt ghesticht, Dat is wel mijn begheeren // begheeren.
