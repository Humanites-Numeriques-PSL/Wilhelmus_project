 In mijnen geest // moet ick verjubileeren Noyt soete feest // dan den Sabboth des Heeren Wie hem met lust // viert van sijn quade werken Van zonden rust // en God die hem kan stercken Die heeft seer pleyn // den Sabboth recht gehouden, Godt heeft hem reyn // sijn zonden quijt gheschouden Oock zalig dan // is hy tot allen stonden Al sulcken man // en rekent Godt geen zonden In wiens gheest fris // gheen bedroch noch quae perten Gevonden is // oock niet dobbel van herten Maer als Gods knegt // de vrome Iob gepresen Slecht ende rechtvolherdich in Gods vresen Oock die gheen ghelt // op woecker heeft ghegheven Noch geen gewelt // aen yemant heeft bedreven Maer leent met vlijt // den armen t’elcker uyre Sonder respijt // alleen uyt liefden puyre, En geeft haer al // die hem niet konnen loonen Dien sulcken sal // in Godes Tempel woonen So wie met vliet // uyt lust hier beklijven, Sy sullen niet // ledigh onvruchtbaer blijven In kentenis // van Gods volmaeckte leere, Die heylsaem is // in Christo onsen Heere God die vermeert // de gaef sijns geest beseven Wie hen verneert // die wort van God v’heven Is een in noot // die hem dan troost van herten, En breeckt sijn broot // een ziel vol honghers smerten, Die niet en kan // rusten eten noch drincken Als hy een man // nootdruftig kan bedincken Maer hem eerst spijst // uyt liefde nae en verre Sijn licht dat rijst // op als de morgen-sterre. Als hy besorght // de Weeskens reyn van zede Geeft, leent, en borcht // elk na geleghenthede, Die haer so kan // voorstaen in alle saecken, Regt als haer man // raet tot bidden en waken Geestigh ghesint // sacht, stil tot allen tijden Dat sy verwint // haer druk en ‘s vyants strijden Niemandt en kan // tot desen Sabboth komen, Oft ten zy dan // dat hy tot sijnder vromen, Gantschlijck versaeckt // die werelt met haer lusten, En niet en smaekt // dat aersche vol onrusten, Met sijn gemoet // gantsch draegt in God verheven, Iae suyr en soet // danckbaerlijcken te leven. Seer onbesmet // is voor God eerweerdig, Die niet en sit // by de spotters lichtveerdig, In haren raet // nog in haer kromme wegen Geensins en gaet // maer kontrari daer tegen, Al sijnen ganck // heeft in het Hemelsch wesen, Met lof en danck // als een met Godt verresen. Soet ende fijn // moet ‘t eynd’ aller geboden De Sabboth zijn // daer maer een is van noden. Dits ‘t eenig een // stil houden Godt gelaten Als Magdaleen // die koos tot haerder baten, ‘t Best eenig goet // ‘t welck haer by is ghebleven. D’oprechte vroet // en is geen wet ghegheven O Gods Gheest net // ghy kondt ons wel regheren Ghy zijt de wet // niemandt heeft u te leeren God heeft u voort // den heylighen gegheven, En na sijn woort? // in haer harten geschreven, Ghy zijt een licht // lanteerne t’haerder voeten Nae Davids dicht // ‘t welck sy nae volghen moeten Och liefden graet // gy bant aller secreten In u bestaet // de Wet en de Propheten Ghy zijt niet smal // het eynd’ end’ de hooft somme, Der wetten al // haer oorsaeck en waeromme Na Schrifts uytlegh // groot boven alle gaven, De hooghste wech // de rust doch aller slaven Nu Prince God // diet alles werkt in allen Ghy zijt mijn lot // laet my van u niet vallen, Werckt nach en dach // wilt al mijn kracht regieren, Op dat ick mach // in u den Sabboth vieren, U jock is soet // u last is licht om draghen, Wie leert ootmoet // sijn ziel rust t’allen daghen.
