 ‘,,Ghelijck stadich ‘,,Inde zee//t'roer loos schip/ ‘,,In perijckel sweeft dach ende nacht; ‘,,D'welck smijt schadich ‘,,Licht in twee// op een clip/ ‘,,Oft het singht door t'groot onweders cracht/ ‘,,De baren heffent op/en neer/ ‘,,Nu hooghe // uyt d'ooghe/ ‘,,Dan schier ten gronde weer. Soo ist even Met den mensch // soo't hier blijckt Aen Eudotia weert ghelauw'riert/ Wyens leven/ Ende wensch // recht ghelijckt/ T'schip/ dat door de baren wort ghestiert! Dats door Fortun', diet hoochst / meest treft/ Die weder // worpt neder Al 't ghene sy verheft. Dese Maghet Arm van staet/en gheslacht/ Die Atheneas eerst was ghenoemt/ Heeft behaghet Om haer daet // hoogh gheacht/ Wijsheyt/deucht/en schoonheyt wijt beroemt Den Keyser Theosiom, Die houde // en troude Met dese weerde blom. Als nu prachtich Was in weldt // roem/ en eer Dees die boven wensch verheven sadt/ Die groot machtich/ Scheen ghesteldt // tot een leer/ Schier op 't hoochste van Fortunas radt/ Als Keyserinne boven al Tot haren // seer swaren/ En al te droeven val. Met beliefte Heeft den Prins // op een tijt Haer/een schoone raere vrucht vereert/ Welcke giefte Onversins // heeft vol nijt/ s'Keysers min in jelousy verkeert/ Midts sy die aen een ander gaf: ,,Hoe slechte // t'onrechte ,,Lijt menich smaet en straf. Desen goeden Heer/ liep ras // tot den stoel Vanden Keyser/en schonck hem den Vorst: Die vermoeden Strackx dat 't was // haren boel/ Die sy desen Appel gheven dorst/ En naer wat ondersoechkingh/knap/ Verlietfe: // en stietse In droeve ballinckschap. ,,Leert hier menschen! ,,Dat de snoo // werelt is ,,Recht een schaduwe/want als ghy meynt/ ,,Dat gh'u wenschen ,,Hebbet/soo // sijt ghy wis ,,Niet/ gansch niet/ja min dan niet vercleynt. ,,Schout dees vermomde streelster dan ,,Dient meere // den Heere ,,Diet eeuwich loonen can.
