 EEn roede groen // uyt Jesse Stam gesproten, 's Winters saysoen // wy 't vruchtbaer hout genoten, Wiens vrucht is spijs // om de ziel te verquicken, Heylig en wijs // met noch meer ander sticken, Die door sijn kracht // levendig overvloeyen, In ons gebracht // als rancxkens op te groeyen. Dit vruchtbaer hout // staet aen een schoon Reviere, 't Brengt twaelf fout // vruchten van waerden diere: En op een Jaer // alle maenden sijn ooghsten Voor al die daer // van te rapen begosten Zijn geresolveert // voor alle vremde qualen Ongesommeert // wie kan sijn deught betalen. Wie hier de smaeck // van sijn vrucht heeft gekregen, 't Geeft hem oorsaeck // om te meer zijn genegen: 't Is in den mondt // boven spijse der stercken, Het maeckt gesont // in den Lust-hof te wercken: Men wort daer door // voorsichtig en kloeckmoedigh, En om wel voor // 't quade te zijn verhoedigh. Prins uwen lust // zy in Gods Wet gestadigh Vierigh gerust // in Gode seer genadigh, Gebruyckt dit hout // tot een staf in u handen, Het maekt u stout // te gaen door de waranden Van 't Jammer-dal // daer men veel verdriet merckte, Het leyt u al // van d'een tot d'ander stercte.
