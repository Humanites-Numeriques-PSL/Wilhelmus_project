 IN mijnen noot, roep ick tot u o Heere, Laet my niet bloot // want ic ben swac en teere, Want ick doch niet // sonder u en vermagh, In dit verdriet // troost my Heer nacht end' dagh. Troost my o Heer // en wilt my niet verlaten, In dit verseer // komt my Heere te haten. Op u alleen // betrouw' ick Heere mijn: Wie magh ons scheen // van u liefd' op dit termijn? Vw' liefde soet // bewijst Heere gheduerigh Watmen ons doet // dat wy niet wesen truerigh. Door uwen geest // o Heer Daniel bystaet, In Babel meest // is hy by die Leeuwen quaedt. Die Leeuwen fel // zijn oock in de woestijne, En soecken snel // ons te brenghen in pijne. En slanghen fier // die spouwen vyerigh fenijn, O Godt zijt hier // onsen rechten medecijn. Geen kruyden soo goet // diemen sou mogen lesen, Noch plaester soet // en kan ons niet ghenesen, Maer Heere ghy die Israel bystont Door Christum vry // maeckt ons zielen ghesont. Ons zielen kranck // zijn altijdt in dangiere Door peste strangh // en sieckten menigertiere, Soo dat hier vergaet // menigh met jammer groot Die haer gheen raet // soecken teghen den doodt. Sy beminnen meer // dat tijtelijcke leven En 's werelts eer // van vleesch en bloet gedreven, Sy zijn eylaes // seer jammerlijck verblent, Als die Maeghden dwaes // die door liefden zijn gheschent. Sy zijn inder nacht // oock slaperigh bedeghen, Als die gheen acht // op 't licht hebben ghedreghen, Als den Coningh haer // sal opwecken subijt, Dan sullen sy daer // kloppen maer ter ontijdt. Dan sullen sy // beschaemt daer buyten blijven Dat soecken daer by // en sal haer niet beklijven. 'Twert dan te spae // om na oly te sien Haer sal geen genae // vanden Bruyd'gom geschien. Och vrienden gemeen // wilt bidden ende waken, Omgort u leen // laet u licht vyerigh blaken, V sedigheyt, laet wesen openbaer, Weest soo bereydt // als wijse maeghden eerbaer. Als een reyne bruyt wilt u alsoo vercieren, Toont overluyt // u licht in all' manieren, Want 'tis nu d'uer // van den slaep op te staen, Door Christum puer, tot den Vader te gaen. Prinslijcke Godt // in eeuwigheydt verheven, Dit voor een slot // mijn vriendekens gegheven, Die van 't ghetal // sijn weergheboren nieu, Dit schenck ick u al // voor een vriend'lijck adieu.
