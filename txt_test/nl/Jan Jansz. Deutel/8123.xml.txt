 O Ionckheydt groene, Bloeyende Ieught, In dit saysoene, Schickt u tot deught, Wilt u begheven, tot sedigheyt, Alle u leven // daer toe bereydt, Met sout doch menght Wat ghy voortbrenght, Spreeckt ofte singht // alsoo ons Paulus seydt. Laet u doch raden, o Iongheydt fier Teder van bladen soet van manier, Wilt u doch keeren, van 's werelts vreught, Vreest doch den Heere // terwijl ghy meught, Want siet die doodt, spaert kleyn noch groot, Hoe jongh, hoe oudt, 't wordt al tot hem geveught. En wilt niet treden // om wat nieus te sien, Als Dyna deden // ten ouden tien Als sy aenschoude, des Landes Heer, Quam sy in rouwe, en swaer verseer, Haer schoonheydt jent // heeft hy gheschent, V hier van went // ghy jonghe dochters teer. Maer wilt aenschouwen, Het vroom ghemoet Vande ghetrouwe // dochter Iephta soet, Die gingh versmaden // al 'swereldts eer, En liet verbraden, haer lichaem teer Op den Outaer // van haren Vaer Spieghelt u klaer // en houdt dit voor een leer. Dit differenti, wilt doch aensien, En maeckt doch menti // al van des twien Hoe d'een bedorven // ende beschaemt, D'ander ghestorven // heel ghehoorsaemt, Dus wilt doch fijn // de ouders dijn Ghehoorsaem zijn // sedich en bequaem. Op dat ghy princelijcke Dochters fijn, Mocht mede van de wijse Maeghden zijn, Die metten Heere, werdt ingheleyt, Want tot d' oneere // sal worden gheseyt, Dit droevigh Liedt // ick ken u niet, Gaet in 't verdriet // day voor u is bereydt. Daer sal gheschieden // soo groot dangier, Nae Schrifts bedieden // in 't Helsche vyer, Wringhen der handen // druck ende pijn, Knerssen der tanden // t'eeuwighen tijn, Hierom o jeught // schickt u tot deught, Op dat ghy meught // hier van bevrijdet zijn, En dan meught hooren, het soet gheluyt, Mijn uytvercooren // mijn lieve Bruydt, Wilt tot my treden, ontfanght de kroon, En gaet in vreden // in 's Hemels troon, Blincken hoort mijn // als der Sonnen schijn, Dat sal dan zijn // der uytvercoren loon. Dus wilt vercieren, u Lampen subijt, Met goe manieren // waer dat ghy zijt, Laet u licht rijsen, wilt liefde schoon Altijdt bewijsen, soo sal Godts Soon, V blijde ontfaen // en voor u staen, En dienen saen // al in zijn Vaders Throon. Oorlof hier mede, o Ieught eerbaer, Godts eeuwighe vrede wensch ick u klaer, Dat by ons blijve // tot 't ende goet En ons verstijve // in een vroom ghemoet, En nae desen tijt, te zijn verblijdt, Wensch ick met vlijt // de Ieught tot eenen groet.
