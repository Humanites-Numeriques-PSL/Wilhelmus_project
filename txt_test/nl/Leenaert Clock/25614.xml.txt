 Dancklijck eenmoedich // prijst Godt den Vaer, Want hy seer goedich // is voorwaer, Sijn gonst om prijsen // gaet hy eenpaer Ons nu bewijsen // int openbaer, Och vrienden ghy // singht doch met my Van herten bly // danckt doch Godt allegaer Int huys des Heeren // sullen bereyt, Wy gaen met eeren // soo hy toeseyt, Om vreuchd’ verwinnen // hier boven gaen, Ons voeren binnen // de poorten staen, Blijft op den ganck // u leven lanck, Al zijn wy kranck // die hop’ v’steect ons saen Een stadt ghebouwen // heerlijck en breyt Haer Borghers houwen // die eenicheyt, Men mach by trouwen // seer hoogh gestelt Haer licht aenschouwen // het lucht int veldt Och vrienden jent // maeckt oock bekent, Dat ghy nu bent // borgers elck als een helt. Recht doch u wesen // als Christen goet, Aen Ziel ghepresen // hert en ghemoet, Als d’eerste deden // met Ziel en lijf, Eenich in vreden // te samen stijf, Gods glory schoon // spant daer de Croon In s’Hemels Troon // sal blincken haer bedrijf. In dese mueren, eert Godt den Heer, Lieft sonder trueren // elckandren meer, Gaet opwaerts beden // voor Godt aldaer, Drijmael t’gheeft reden // int nieuwe Iaer, Hem offer bringht // danckt, loeft en singht, Dit vyer ontsinckt // de liefde in ons claer Climt op de graden // volheyt der deucht, Dat ghy gheraden // t’gheloove meucht Betuygen waerlijck // als volck Gods vroet Wt Godt schijnbaerlijck // ghesproken soet, Met cracht en daet // soo Christus raet, Elck huys bestaet // dat men so bouwen doet Kent, wilt ghevoelen // daer zijn gheset De Rechter-stoelen // nae rechter Wet Oordeel te gheven // in Davids huys. Nae een elcx leven // sonder abuys Crijcht elcken slecht // daer loon nae recht, D’onnutte knecht // sal daer worden confuys Perfeckten vrede // soo’t Godt behaeght, Crijcht niemandt mede // dan die nu draeght Liefde te vooren // int herte hier by, Reyn uyt vercooren // sonder party, Godtloosen siet // en hebbens niet, Want vrede diet // Christus den sijnen vry Ick wensch u t’samen // vrede van Godt, Liefde met namen // t’nieuwe ghebodt, Naer Schrifts verclaren // Ierusalem Wilt openbaren // volght Christi stem, In deucht gaet voort // al nae sijn woort, Niet anders hoort // en sal behaghen hem. En in u toorens // breete ende smal Moet zijn te voorens, vred’ over al, Daer men behoedich // de wapens hout, Bereydt en spoedich // seer menichfout, Om de trompet // te blasen net, Zijt ghy gheset // t’volck Gods is betrout. Treedt op u voeten // t’gheluydt nu staet, Ontwaken moeten // sy niet te laet, Den vred’ wilt preken // straft nu by tijt, Soo onbesweken // u Ziel bevryt, Gods volck certeyn // de kudde cleyn Bewaert nu reyn // want ghy haer wachter zijt. En wassen, groeyen // moet ghy Gods stadt En voerich bloeyen // te wensch u dat, Om t’lof te bringhen // voor Gods aenschijn, Is nu het singhen // en t’wenschen mijn Vreedlijck vergaert // Godtlijck gheaert, So Schrift verclaert // met Christen sinnen fijn. Recht op’t vervallen, bindt met der hant, Heer, onder allen, des vredens bant, Bouwen, vergaren // souden wy sterck, Als trou dienaren // soo aen u Kerck, Iae als een man // timmeren dan, Gheen vyant can // ons dan beletten t’wercl. Staet om verhogen // Heer by met cracht, Want onvermoghen // is ons macht, Gheensins verblijden // en moghen wy, Helpt ghy hier strijden // aen onse zy, In u macht staet // ons cracht en daet, Verwinnen laet // ons doch niet moede bly. Salicheyt nacy // ghy Christen schoon, Zy u, soeckt stacy // in s’Hemels throon, Dat ghy met namen // sonder beswijck, Soo meucht versamen // int Hemelrijck, Liefdens vermaen // heeft dit ghedaen, Neemt het doch aen // van my soo al gelijck.
