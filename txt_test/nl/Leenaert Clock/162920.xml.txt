 DIe vred' Gods die te boven // gaet Diens name hoogh te loven // staet, Wensch ick u al te samen Waer dat ghy woont, met namen, Int groene lant // aen elcken cant, Wandelt na u betamen, De Heer biedt u sijn hant. Al schijnt ghy nu verlaten // schier, De Heer com u te baten // hier, Och wilt dat wel bewaren, Dat hy u ginck verclaren, Sijn heylich woort // dat ghy ghehoort Hebt, van synen dienaren, Brenght doch goed' vruchten voort. Mijn liefste uytvercoren // al, Wel hem die dees stem hooren // sal, Comt hier mijn uytvercoren, Mijn Rijck staet u te vooren, Ontfanght het loon // in 'sHemels throon, Ghy sult vreuchden orbooren, Ontfanghen 'slevens Croon. Siet, niemant sal verblyden // saen, Dan die nu deur het lyden // gaen, En die als rechte leden, Christi voetstappen treden, Den smallen ganck // haer leven lanck Met Christelijcken zeden, Ghelijck een vruchtbaer ranck. Treurt nu niet om den tegen // spoet Den welcken ghy verdraghen // moet, Ist so Gods wel-behaghen, Soo wilt hem willich draghen, Lijdtsamich zijt // in uwen tijt, Het duyrt hier weynich daghen Ghy sult noch zijn verblijt. En wilt doch des vergheten // niet Dat u de Heer noch weten // liet, Sijn heylich woordt gepresen Wilt spreken ende lesen Te samen hout // elckander bout, Elck zy ghetrou in desen, Dat hem hier is betrout. Recht houdt doch u becoepen // vry, Ghy Ouden, dit ontknoopen // wy, Wilt een goet voorbeelt gheven, Den Iongen met u leven, 'Tmoet zijn gewaeght, weest niet vertsaeght Ghy sult worden verheven, Soo ghy u dapper draeght. Siet toe ghy jonghe Ieuchde // bloot, En hebt doch gheen gheneuchde // groot In s'weerelts lust voor ooghen Dat ghy niet wordt bedroogen, Van 'svyandts strijt // die u benijt, Sijn doen is al ghelooghen Wacht u daer voor met vlijt. Den Weduwen tot reender // eer, Gheeft Paulus schoon tot eender // leer Dat sy haer hope maken, Op Godt voor alle saken, Want hy beleeft // d'uytcomen gheeft Wel hem die dit can smaken, Gods Gheest niet teghen streeft. Iaeght doch nu na den vreden // soet, En sout in al u reden // doet, Elck wil sijn tongh' bewaren, Om niemant te beswaren, Oock leughens quaet // de Heere haet, Hier in als Gods dienaren, Bruyckt doch de rechte maet. T'samen wilt doch nu draghen // 'tlast, Elckanders na't ghewaghen // vast, 'Tis Christus Wet waerachtich, Weest des altijdt ghedachtich, Al dat ghy saen // gheern hadt ghedaen, Doe nae dat ghy zijt machtich, So sult ghy wel bestaen. Vrienden ghy staet int herte // mijn, Het soude my groot smerte // zijn, Dat ick van u moest hooren, Anders als hier te vooren, Doe ghy ghesont // bekendet ront Dat ghy als uytvercooren, Woudt houden Gods verbont. Ghesonden is dit cleen // ghedicht Tot u, dat ghy wordt reen // ghesticht, En dat ghy oock meucht vaten, Dat ghy niet zijt verlaten, Maer dach en nacht // in ons ghedacht, De Heer com' u te baten, Bewaer u deur sijn cracht.
