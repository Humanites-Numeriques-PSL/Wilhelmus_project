 O Droeve tijdt // die ick verslyt Met vruchteloos beclaghen O aertsche Hel // doot mijn ghequel, Door t'snoeyen van mijn daghen. Ick ben alleen // verd van de geen En tegen danck gebannen, Helaes! van die // ick 't liefste sie Van alle jonghe Mannen. Ach ongheluck // ach leyde druck Sult nimmer ghy ophouwe, Dit hart ghewelt // dat nu dus knelt, Dees teer benaude Vrouwe? O schraele doot // comt en door stoot Dees vrouwelijcke Borsten, Ach! strengh gemoet // dat so verwoet, Na u Kints bloet gaet dorsten. Mijn Moeder wreet // doet my dit leet En ghy versufte Grijser: Een Keysers kint // om dat het mint, Besluytmen straf int Yser. U fel ghegrim // u wreetheyt slim U dreyghen, noch u sweeren, Geen schric noch noot // noch duyvel doot En can myn Liefde deeren. Maer als ick sal // int traenen dal Door ongheneught verdwynen, Mijn schim bevreest // of bleecke gheest, Sal nae myn doot verschynen. En quellen swaer // myn Ouders maer Mijn Echte Man ghepreesen, Die sal ick dan // waer in ick can, Noch even dienstich weesen. Mijn Prinsch vertoeft // weest niet bedroeft Bedaert, u wilt verlanghen, Mijn hart blyft u // al ist lyf nu, In een Kercker ghevanghen. De trou en eed // dien ick u deed Die zullen geen sints lieghen Hoopt, en vaert wel // de dinghen snel, Op aerden licht vervliegen.
