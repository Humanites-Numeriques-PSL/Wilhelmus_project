 LAnghe dagen // wordt ghedraghen Godes Arck, door 't woeste velt: Menighwerven // most zy swerven, Nimmer wordt zy vast ghestelt, Tot seer schoone // Davids Zone, Haer in d' Heyl'ghe Tempel treckt, Daer twee Eng'len // t'samen streng'len, 't Welck haer tot een decksel streckt. Siet hier vinden // Godts beminden Eenen heerlijck teghen-beeldt: Want sy sweven // oock in 't leven, Rust wordt haer hier mis-ghedeelt. Maer de Heere // die wy eere, Davids Heer, en oock sijn Zoon, Die sit mede // nu in vrede Op des Hemels gulde Throon: Daer hy stadigh // seer ghenadigh, Van des Wereldts woeste kust Vol van Doornen // sijn verkoor'nen, By hem stelt in sal'ghe rust.
