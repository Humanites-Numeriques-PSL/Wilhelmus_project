 VOor 't al te schittrich licht // dat Venus uyt haer lampen Gheblasen, heeft ghesticht // in u hel aenghesicht, Hemel en Aerde swicht // en afgrondt met haer dampen. De StroomGodinnen gladt // die door den Aemstel swieren, O pracht van onse Stadt // hebben een nijdt ghevat, Op u aenschijn, en dat // met nygend voorhooft vieren. Zy quellen haer ghemoedt // met hondert duysent smarten, Om dat u schoonheyt doet // onsteecken, in haer vloet, Met onleschbare gloet // der WaterGoden harten. Met moedeloos gheween // dees Meereminnen quelen, Dat uw volmaeckte leen // van boven tot beneen, Met nieuwe konst ghesneen // hen al haer Vryers stelen. Een troosjen hebben zy // dat's hoop van te beleven, Dat door uw strafheyt ghy // den tijt sult gaen voorby, Van soete vryery // en hebben niet bedreven.
