 Met wandelen in ‘t groen Ginck ic mijn bedroefde geest wat vreugt aendoen, In bos en wey // Door sant en hey, Ginck ick betreen in de soete Mey, Geneughden // En vreughden, Dit was // In ‘t gras, Elck beesje las, Sijn behoef kruypt // Van ‘t groen gespruyt, Daer pluckte het sijn voetsel uyt. Het soet parigh gediert, Dat in de bossen en door de bomen swiert, Voelden de min // In hare sin, Want de telens lust was daer in, Sy songen // En sprongen, Met vreucht // Verheught, In soet geneucht, In soete lust // Geheel gerust, Hebben sy haer soete min geblust. De Herders in het velt, Hebben met geley haer Herderins verfelt, Het soet gevley // Was veelderley, Dat menige soete tongh daer s Tot minne // Te winne, De sin // Daer in, Te doen gewin, Had ider voor // Die hy verkoor, Elck een volghde des liefdens spoor. Menige Man en Vrou, Sach ick daer gaen o soet vermaeck van de trou, Die altemael // Met soet onthael, Malkander vermaeckte, voor quael, Veel vreughde // Verheughde, Haer hert // Dat wert, Geheel van smart, Door dien sy al // Met soet geval, Sijn lief kuste sonder getal. Ick sach menigh soet paer, Gaen wandelen door de velden hier en daer, Ick sach daer treen // Veel met haer tween, Niemant als ick most gaen alleen, De rouwe // quam houwe, Mijn hert // Dat wert, Vol grote smart, Dit was voor mijn // Een grote pijn, Ick dacht waer mach de mijne sijn.
