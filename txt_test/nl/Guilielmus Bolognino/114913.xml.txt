 GHelooft aendachtich // ghevoelt waerachtich Dat Godt rechtveerdich is en goedt In alle tijden // dat hy het lijden Uyt liefde op ons comen doet. 't Schijnt hier beneden // al sonder reden En sonder schick te zijn gheschiedt; Maer wilt Godt loven // 't heeft al daer boven Schick en sijn reden, watmen siet. Keert daer u ooghen // wilt ghy wel drooghen U tranen, en u rechten 't hert. Gaet dit bevroeden // wilt ghy verhoeden 't Ghekreun, en ongheschickte smert. Wilt oock aenmercken // dat in sijn wercken Soo vremt en duyster is den Heer, Om dus te croonen // en hooch te loonen 't Gheloof, dat hy hier oeffent seer. Zijt niet kleynmoedich // al is voorspoedich Den vyandt van het recht gheloof, Godt in 't verborghen // sal voor ons sorghen, Zijt Ketters schimp en laster doof. Op Godt u ooghen // die 't wilt ghedooghen, Met vast gheloof en hope slaet, Allen sijn raeden // zijn vol ghenaden, Hy sal 't al stieren tot ons baet. Sijn roeden leeren // sy doen ons keeren Tot hem alleen gans ons ghemoedt Ons selven sterven // de schepsels derven Om 't soet gheniet van 't eenich goedt. Siet ghy dan branden // in onse landen Het bitter lijdens salich vier, Laet 't hert niet sincken // het goudt sal blincken, Dat Godt bermhertich suyvert hier. Wilt hem dan loven // dat hy den oven Uyt ware liefde stoken gaet, Die gans verdwijnen // al is 't met pijnen, Sal doen den roest van u misdaedt. Siet ghy vernielen // de Christen zielen, Die gaen ter doodt door kettery, Door quade seden // Godts eer vertreden, Zijt daerom droef, schreyt daerom vry. Laet u dit seuren // gaet hierom treuren; Maer denckt wat ons de Waerheydt leert, Dat all' u suchten // sal in ghenuchten, In grooten troost eens zijn verkeert. Gaet traenen saeien // om vreught te maeien, Sucht voor de zielen tot den Heer, Klopt aen sijn doren // koelt sijnen thoren, Dat hy sijn aenschijn tot ons keer.
