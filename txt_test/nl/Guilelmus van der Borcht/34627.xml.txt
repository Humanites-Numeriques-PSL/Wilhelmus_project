 ACh ! wat een smerte // braeckt het minnen uyt, Wanner een herte // sijnen brandt besluyt, Sonder te klaghen // daer 't niet kan verdraghen Het vinnigh pranghen // van het groot verlanghen ? Het stoute minnen // seght my stadigh aen, Dat ick mijn sinnen // soud' verklaren gaen, Maer bleeuwe schaemte // en vrees' van de blaemte Doet my vertraghen // mijn Lief naer te jaghen. En noch doet hy my // d'alder-meeste pijn, Om dat hy by my // mijn Suster komt vry'n: Ach ! hy is stadigh // fel en seer onghenadigh Ontrent mijn deure // 't welck my meer doet treuren. Dus ben ick even // Tantalus die haeckt Naer 't vocht dat neven // aen sijn lippen raeckt, En noyt kan proeven // dan een staegh bedroeven, Om dat sijn lippen // noyt aen 't water stippen. Wat heeft hy aerdigh // doch in haer besint, Dat sy is waerdigh // boven my bemindt ? Noyt wirdt haer wesen // boven 't mijn ghepresen, Noch haer soet spreken // teghen 't mijn gheleken. Ick ben niet spijtigh // soo hy van haer klaeght, Maer siet ick vlijtigh // kom' hem naer-ghejaeght; Wilt hy mijn banghe // ziel' maer eens ontfanghen In sijn ghenade // eer het wordt te spade. Ick sal versaken // mijn Suster ter noodt, Midts ick niet haken // magh haer blijde doodt: Maer wil eer sterven // en oock metter spoet Mijn borst roodt verven // met mijn heete bloedt.
