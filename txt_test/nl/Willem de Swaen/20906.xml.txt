 WEest verblijdt // t' allen tijd 's Hemels Coninginne Aleluya // alleluya, Ick U opdraegh all' mijn sinnen. Want Ghy hebt, die 't all' schept, Maeght! in U gedragen, Alleluya // alleluya, God in U had groot behagen. Desen Heer // heeft sijn keer Uyt het graf genomen, Alleluya // alleluya, En is we'er op d' aerd' gekomen. Onverwacht // uyt het gracht Is uw' Soon verresen; Alleluya // alleluya, G'lijck voorseyt Hy had voor desen. Weest verheught // weest verneught, Waerdste Godes Moeder! Alleluya // alleluya, Opgestaen is ons' Behoeder. Maget bidt // dat wy 't Wit Van den Hemel raecken, Alleluya // alleluya, En dat wy ons God genaecken.
