 GEkomen // vernomen Is heden in 't stal Den Hoeder // den Voeder, Den Schepper van all': De Knapen // der schapen, Verlaten hun Vee; En soecken // in doecken Blymoedigh Hem dree. Haer keelen // en Veelen Die worden gestelt; Sy singen // en springen Van vreughden langhs veld, Sy rasen // en blasen, En spelen op 't Riet: 't Is wonder // dat onder Hunlieden geschiet. Sy loopen, sy hopen Het Kindtje te sien; Sy willen // het stillen, Sy willen 't verblien, Met kussen // met sussen, Met vrindelijckheyd, Met kallen // met rallen, Met Herders-beleyd. Heus Cloris // en Doris. Waer loopt ghy dus ras? Laet soetjes // uw' voetjes Volbrengen haer pas. Wy moeten // gaen groeten Te samen den Heer; 't Is reden // 'k wil mede Bewijsen Hem eer. O Tyter ,, wat Cyther, Wat snaare-gespel, Wat Orgel // wat gorgel Behouwen wy wel Om d' eere // des Heere (Die 't herte verheughdt) Te spelen // te quelen Met 's herten geneughd. Trompetten // Kornetten Behouven wy niet, Noch Bullen: // wy sullen Van buyten een Lied Beginne // uyt minne Van 't Opperste Goed; Waer heenen // ons' beenen Nu loopen met spoed. O Maetjens // ons praetjens Die dienen gestaeckt; Houdt Herder! // nier varder: Het Stal'ken genaeckt. Laet vallen // ons allen Ter aerde ter neêr. En groete // dit soete Kleyn Kindeken teer. Zijt welkoom, zijt welkoom, Zijt welkoom weerom Bestierder // Vercierder Van Sterren // en blom; Wien baerde // op d' aerde Dit Maeghdeken fier, Wiens glory // memory Wy vieren nu hier. Komt Machten ,, komt Krachten Van 't hemelsche hof; Komt Eng'len ,, vermeng'len Uw' stemme tot lof Ons' Broeders ,, ons' Hoeders, Met hemelsch' jolijt: Seght Amen: ,, weest t' samen Met d' Herders verblijdt. Messia ,, Maria, Wy groeten U t' saem; Verspreyden // verbreyden Wy sullen uw' Naem: Hier leggen // ons' Weggen, Hier laten w' ons' Melck; Wy jachten // wy trachten Nae huys toe weêr elck.
