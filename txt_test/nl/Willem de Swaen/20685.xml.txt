 DOrathee ,, ach! uw' twee Susters, God begeven. Sy besoecken // ('t wil niet loecken) Of Ghy, g'lijck als sy, wilt leven. Sy besoecken. Maer gehecht // en gevlecht Zijn met liefdes-banden All' uw' sinnen // all' uw' minne, Aen den Heer van alle landen. G' hebt geleert // G' hebt bekeert Uwe Susters deftich; En Sy leden // en Sy streden Voor de leeringh Christi treftich. Groote nood // ja de dood Hebben Sy verdragen: Hooge gaven // 's Hemels haven Kregen Sy na dese vlagen. Doen zijt Ghy // in de ly We'er daerom gekomen: In die dagen // hebt Ghy slagen Lustich op uw' le'en vernomen. Ey! Ghy hebt // en Ghy schept Vrolijckheyd in 't lijden. In het blaecken // in het kraecken Uwer le'en, Ghy U verblijden. Ghy ginght heen // Ghy ginght tre'en Na 't schavot blymoedich. U verbeyden // 's Hemels weyden En Ghy quaemt daer in gansch bloedich. Uyt dat land // heeft een mand' God met schoone Roosen Ne'er gesonden // op dier stonden, Die Theophilus sagh bloosen. Dese man // nam die an, En Hy sich bekeerden; Ja sijn leven // heeft gegeven Voor den Heer dees Rechtsgeleerden. 's Hemels lot // schonck hem God Voor dees minne-voncken; Om sijn deughden // is van vreughden Hy, en Ghy, gestadich droncken. Och of Ghy // oock op my, Dorothee, woud dencken! Ey! bidt krachtich // God Almachtich, Dat Hy my sijn Hof wilt schencken.
