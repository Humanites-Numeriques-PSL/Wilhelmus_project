 O Hemels groot Cieraed, Xaveri! Met deugd verciert in d' hoogste graed: Ick wil uw' deughd // en uwe vreughd Verhaelen hier met 's harts geneught: 'k Wil schrijven // 'k wil stijven // uw' eer // uw' leer, Staeghs meer en meer; Want ick mijn min // en al mijn sin Op U gestelt heb van 't begin. Van uw' gehoorsaemheyd, Xaveri! kan worden noyt genoegh geseyt: Met eene I // gingh vlijtigh Ghy De werelt rondom even bly: Ghy eerden // Ghy leerden // te saem // de faem, En Jesus Naem, Met groote vlijt // tot 's vyands spijt, Gheen tegenspreken Ghy en lijdt. Ghy had soo suyv'ren hart, Xavieri! dat vuyl gedacht U was een smart: Want als by nacht // een vuyl gedacht Den Duyvel U had voor-gebracht, Van minne // van sinne // Ghy bloedt // hoe soet Was dese vloed! Het speet U dat // uw 's lichaems vat Oock in de slaep sijn wet vergat. Uw' yver was soo groot, Xaveri! dat Ghy niet hebt ontsien de dood: U drouf gesteen // u drouf geween Vermurwden harten hart als steen. Ghy smeeckten // Gy preeckten // staegs aen: // Goods baen Deed' Ghy 't volck gaen: Het speet U seer, dat God uw' Heer Vergeten worden, en sijn eer. Ghy waert Mariaes vrind Xaveri! Sy achten U te zijn haer kind; In all' uw' rouw // was dese Vrouw Een troost aen U, ja u Behouw: In 't lijden, in 't strijden stondt Sy // U by, Ghy raeckten vry. Des helschs geweld // heeft S' ne'er geveldt; Ghy wiert door Haer soo vroomen Held. Ghy sijt geweest gestelt, Xaveri! een Prins in 't Indiaensche veld, Dat land geleert // dat land bekeert Hebt Ghy, en met Goods Wet vereert. Wat loont // wat kroone // wat lot // geeft God Van Sabaoch, Aen U sijn vrind // die Hem soo mindt, Dat Ghy aen Hem die Landen windt. God heeft U soo verheught, Xaveri! en ingestort soo groote vreughd, Dat Ghy riept, Heer! // gheeft my niet meer, Van waer komt my toch dese eer? Est Satis, est Satis ,, ay my! // ick ly Goods slaverny! Ick ben niet waerd // dat my ervaert, Ick bidd' U, Heer! uw' vreughde spaert. Is 't saecke dat Ghy zijt, Xaveri! geweest soo op dees aerd' verblijdt: Wat sal dat zijn // een soete pijn T' aenschouwen eeuwigh Goods aenschijn! Vernomen // bekomen // die kans // Goods glans, Hebt Ghy nu gantsch: Ghy slaet om-hoogh // op God U oogh, Wien ick altijd te dienen poogh. Maer aengesien ick ben, Xaveri! een sondigh mensch, g'lijck ick beken, In ziel gewondt // met sware sond', Soo roept uw' hulp mijn droeve mond. Gestadich // genadigh // U toont // my loont, Die boven woont: Dat is Xaveri! ,, bidt dat ick hier Wel-levende ontga het vyer.
