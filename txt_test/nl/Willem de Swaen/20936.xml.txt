 Weest verblijdt // t'allen tijdt 's Hemels Coninginne Alleluya, alleluya, Ick u opdraegh all' mijn sinnen. Want ghy hebt // die 't all' scheyt, Maeght! in u ged ragen, Alleluya // alleluya, Godt in u had groot behagen. Desen Heer // heeft sijn keer Uyt het graf genomen, Alleluya // alleluya, En is we'er op d'aerd' gekomen. Onverwacht // uyt het gracht Is uw' Soon verresen; Alleluya // alleluya, G'lijck voorseyt hy had voor desen. Weest verheught // weest verneught, Waerdtste Godes Moeder! Alleluya // alleluya, Opgestaen is ons Behoeder. Maget bidt // dat wy 't Wit Van den Hemel raecken, Alleluya // alleluya, En dat wy ons Godt genaecken.
