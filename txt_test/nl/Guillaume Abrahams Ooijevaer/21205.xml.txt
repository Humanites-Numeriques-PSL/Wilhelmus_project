 Ach Bellinde! lieve Susje, Ach mijn woeste lusje, De soetigheydt // ach lieve Meyt, Kan ick vergheten niet: Want doen ick by // u groene zy Lestmael verkreegh mijn rusje, Ie trooste my // waer deur dat jy Verkeerde mijn verdriet. Ick was heel moed en verslaghen, Als een Hart van 't jaghen, Daer toe bezwaert // en onghepaert Als een verloren Schaep, Maer doen ick quam // by u, mijn Lam, Ghedaen was al mijn klaghen, 'k Was soo gherust // deur soete lust Viel ick van vreught in slaep. Met ick was in slaep ghekomen, Kreegh ick sulcke Droomen: My docht ick sagh // een soet ghelach Van Nymphjes in het groen: Noch hoord' ick een geluyt (soo 't scheen) In 't ruysschen van de Boomen, Iae wel my docht // gheen dinck en mocht My meerder vreught aen doen. Met dit Droomen ick ontwaeckte Vyt mijn slaep gheraeckte, Waer deur ick schier // ach soete Dier Op sprongh van enckel vreught: En oock te meer // verblijd' ick seer, Om dat ick nochghenaeckte V, aerdigh Beeldt // soo op ghestreelt, 't Welck een jonck Hart verheught. Want gheen vreught op deser Aerden Is soo hoogh in waerden, Wanneer een Maeght // ghetrouw'lijck draeght Haer Liefd' op een Ionghman; Ach het is een saeck // vol soet vermaeck, Die noyt de Ieught beswaerden: Want Liefd' altijdt // dooft haet en nijdt, Ghelijckmen dencken kan.
