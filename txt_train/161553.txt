PArst uyt 't ghesicht een droef ghetraen,Laet big'lend' langhs dees kaecken gaenBracke vloedt // die ick moeOp d'aerde gietenMet naer ghesucht, helaes om een,Wiens harde hert blijft als een steen,En niet acht // nae de klachtVan mijn verdrieten,'t Gheen ick staegh // alle daeghDoe voorwaer // of ick haer // noch hier naerHier door eens mocht ghenieten.Doch 't schijnt mijn smeeckingh sy begeckt,Of dat haer borst een rots verstreckt:Want wanneer // ick begeerMet soete woordenHaer aen te spreecken, stuers en strafDraeyt sy haer wesen van my af,En gaet heen // of mijn reenSy niet en hoorden,Jae het schijnt // dat sy vijntHier in rust // en haer lust // werdt gheblustMet dus mijn ziel te Moorden.Wel Amarille seght my dochWat hebt ghy voorghenomen nochVoor mijn trouw // waerde vrouwMijn te doen erven?Wat is het gheen, seght Harderin,Dat ghy met mijn hebt in u sin?Sleghts ghebiedt // daer is niet,Lust u mijn sterven?Of begheer // ghy noch meer,'k Ben te vreen // wilt my leen // vry van een,Van lidt tot lidt doen kerven.Is 't dat u wraeck dan meerder dorst:Soo komt en opent vry dees borst,Wt sijn stee // rockt vry meeMijn trouwe herte,Klooft het aen tween, en siet GoddinV eyghen beeltenis daer in,'t Gheen ghewis // d'oorsaeck isMijns vreughds en smerte,Ghy sult sien // datter gienMinnaer trou // is Me-vrou // of ick souIn liefd' hem durven terten.Want wat ghy wilt, ick ben te vreen,Ick vrees gheen straf, hoe groot, o neen:'k Ben gherust // als gheblustWerdt u begeeren,Want Lief, mijn leven en mijn doodtBestaet in u vermoghen groot,Wat ghy doet // 't is al goet,'t Kan mijn niet deeren,'t Gheen u haeghd // waerde MaeghdDoet dat vry // laes aen my // 'k sal my blyNae u behaghen keeren.
