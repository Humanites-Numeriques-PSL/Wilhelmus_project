MYn lief is mijn, en ick ben Die my so dicmael trueren // Als hy absent // hem van my Droefheydt mijn hert besueren // Dickmael tot hem // heff' ick mijn Die my noyt te versmaden // Met ongheluck // of swaren Als hy my maer beladen // Dus met gheklagh // en swaer Soeck ick, die my versaden // Hy vrijdt mijn hert // van alle En al mijn kranckheydt hy // Hy komt daer in // vernieuwt mijn Verdrijft wat hem in my // Dus is 't met spoet // gheworden Dat ick de werelt derven // De sonden quaet // ick nu seer Die ick soo noo te sterven // Dat eerst mijn ziel // soo bitter Om 't selv' ick nu te erven // Nu ben ick bly // van sorghe Die my eertijts benoude // O wereldts eer // quelt my niet 'k Wil my van u nu houden // Wellust, voorspoet, al 't aertsche Magh nu mijn ziel vermeyden // Door haren raedt // en valschen Ick my wel eer verleyden // Nu metter daedt // ick vroegh en Wil Godt alleen verbeyden // Jesu mijn lief // al mijn Ghy blijven sult tot aller Met u alleen // ben ick te Wilt my toch helpen in den Boven gheneught // en alle De welcke spruyt van vleesch en Is het vermaeck // daer ik na Van uwer liefden soeten Om desen smaeck // ick heel Mijn eyghen ziel, en 's wereldts
