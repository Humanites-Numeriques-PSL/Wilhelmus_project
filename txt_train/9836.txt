T'Ghesichte, 't eerste der vijf Doet ons onderscheyden // Gods wercken al ter Werelt Die zijn Macht verbreyden // Noyt niemant moghen // heeft God sien met ooghen, soo hy Maer die tot // zijn Keert met En volherdt in deuchden // Die sal Godt // t'saligh Sien in T'wert een vreucht der vreuchden // T'ghehoor is den wegh daer Door komt, uyt des Heeren // Daer 'thert' opmerckigh, niet als Recht Gods bode leeren // T'ghehoor oft lesen // salight Self doender wesen // moetmen, Met verdragh // dat op Van Gods Van ons als verblijde // 'tSoet ghewach // ghehoort Zijn tot Komt ghebenedijde // Den roke der salvinghe Van des Heeren goeden // Die neder vloeyt uyt s'deuchts Hier in den ootmoeden // Elck Christen gheven // moet oock Soet rokigh leven // als de Van Sioen // zijnde Een kruydigh bequame // Altijdts groen // in wel Soet van Gheeft sy haer vreindts name // De smake van Gods goetheyt En 'sHeeren woort Twaer goet dat yeder daer aen Als kindt dat de Moeder // Suyght om op groeyen // met sulck Daer niet int vloeyen // is Alsoo haeck // na de Van de Heylighe Schriftuere // Sulck gheen saeck // tot Hier op Als dees voedtsaem puere // Ghevoelen laet ons t'lyden Dat hier onsen Naesten // Als Samaritaen die Hulp toe doen met haesten, 'tQuaet laet ons mijden // met En t'allen tyden // goet doen, s'Heeren raedt // ons wel Te Sonder te verkoelen // Want naer daedt // goet oft Namaels Int lichaem bevoelen //
