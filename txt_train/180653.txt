Schoon duyvinne // die reyn van My meer bemint dan uys selfs Want ghy inne // t’vier mijner Verslonden zijt van hier Door mijnen gheest // uyt t’aerdtsch Tot by my int Hemelsche In mijn slaepcamer Rust u hier dan // tot dat voor De hitte van // den middach Sonder scheyden // wilt u Met my en vast vereenicht Gheen verleyden // hoe sy V deeren mach, ghy sult En vreest niet, want // elcken Sal ick met crachte neder En legghen tot ons voet Dan wert altoost // ghewisselt Droefheyt in troost // haer vreucht in Openbaren // met mijn Sal ick eer lanck, tot u Den sondaren // tot een Die mijn aenschijn sullen Vluchten beducht // sonder Werden benout van herten Och siet hoe lief en weerdt Is sy die wy // hielden voor Sy is nu bly // wy droef O Princiere // blijft Lijdtsamich tot behoudt der Met den viere // uwe Vervolghers rijck sal ick Den nieuwen wijn // int rijcke Sal u herte vreuchdich En uwen mont vol lachens By my u steed’ // eeuwich zijn Troost u hier meed’ // int aerdtsche
