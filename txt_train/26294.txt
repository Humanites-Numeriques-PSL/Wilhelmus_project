FOnteyne soet // alleyne Om ons te wasschen Is Christus bloet, waerdeur men Worden volmaeckt Al die sondich Haer in dit water Sullen reyn worden Dus comen wy Tot u, wilt ons Want ghy sijt seer Een claer Fonteyn // sijt ghy Levendich water fijn // als Die t' herte reyn, verheught En oprecht Dit water claer // is Ghepresenteert te Al die daer naer // dorsten Om niet wil hy dat Al die comen tot In 't nieu Hoorende nae zijn En haer selven Sullen als Schaepkens Van den goeden dranck In eeuwicheyt // wert haer Te sijn in groote vreucht // Dit al voorseyt // met groet Doet ons dit water Maer Israel // verliet Die goede En graefden snel // vuyl putten Waterloos Veel vremde vonden Ghebruyckt op dit Buyten Christum Om salicheyt Maer eylaes Godts Sullen zy moeten Want alle die // En sulcke sonden doen // seer Godts heerschappie // sal straffen Zij sijn, naer dit Godts straffe sal // die Sondaers Treffen, en niemant Op den wech smal // sonder Wilt ons, o Heer, Laeft ons met des Gheests Want wy inwendich Moeten ons leven U hulpe sijn Om gaen den rechten Anders sijn wy niet Wat ons gheschiet // verlaet ons Hier nu noch nemmermeer // o Als ghy ons siet // in swaer Draeght onse swackheyt Prince ghy sijt // tot aller Die ons helpt t' velt Ghy voor ons strijdt // als wy met Vierich sonder Vrymoedich blyven Sonder u af te Met t' Harnasch Bereyt om cloeck te Doende nae u Als goe ghetrouwe O Prince Godt // ick bid in 't Mijn ghebedt doch verhoort // en Nae u ghebodt // leydet ons recht s Levens Fonteyn
