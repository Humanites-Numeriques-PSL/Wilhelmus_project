Ave Maria // Gods liefste lief Der maechden kronen // Maria weest Gratia plena // claer als de sonne Int Hemels troone // vat vol genaden Der deuchden volle vloet // een suyver hert Hebt ghy gebaert // Gods soon ons hoochste Dominus tecum // die heer sy met v Suyver fonteene // vrou alder werelt Benedicta tu // in mulieribus Al waert ghy cleene // ghy zijt Geen vrou te geender tijt // so salich als ghy Noch so verheven // van Godt tot vyants Et benedictus tuus // gedanckt, gelooft, Moet ghy altijt wesen // dat hoochweerdich Fructus ventris tui // dijn vrucht reyn wt Wt Ieffa geresen // Iesus Christus Die als een Heer valjant // ons brac der hellen Ende die gevangen brocht // in haer vaders Sancta Maria Princesse // hooch verheven Edel geboren // van koninck Davidts Ora pro nobis // bidt voor ons die te Waren verloren // verdoemt door sonden Tot dat ons quam te baet // Iesus onse Van u geboren // o schone
