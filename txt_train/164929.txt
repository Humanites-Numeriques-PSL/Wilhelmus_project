Die ellendiche dijn // en O Goddelijcke graen // met malcander Met druck belaen // Laet ons gheraen // met Davids Des Heeren naem // gaen Doen ick den Heer // noch sochte van Hy antwoorde my weer // van En verloste my teer // door sijnen Van alle mijn zeer // my Die hem aensiet // en tot hem Die en wort niet // tot Want God die biedt // die ellendige met Die tot hem riep // sijn En helptse voort // al wordense Wt alle noot // en Sijn enghel hoort // beleghert met Der gheloovigher oort // spijt Siet ende smaect //watten vriendelijcken Dat hem ghenaeckt // na Die na God haeckt // en op hem Niet en versaeckt // maer Vreest God altijt // en hem Ghy die nu zijt // Want die met vlijt // hem vreest en In desen tijt // sal zijn Werpt u begheer // alleenich op den Voort quaet ghebeer, // u wilt Die hongherde seer // na sijn Goddelijcke Vervulde hy weer // met Maer die hem versmaet // nu zijn Rijck hooch van staet // van God weer verlaet // nootdruftich En hem ghenaeckt veel Hierom verlaet // dit vuyle En sulcken staet // Gods vrede aenvaet // die s'menschen Te boven gaet En gheeft gheen wijck // ist Oprecht cuysch lieffelijck // int Ist wel luydelijck, ist Ist lof eerlijck // Comt herwaerts hoort // Gods kinderen sijn Na waerheyt spoort // te Wie isser voort // die nae 't Met goet accoort // te En wie gherust // tot goede daghen heeft Iou tonghe uytblust // voor Iou lippen sust // van bedrieghen En boose lust // te Vant boos wilt vlien // doet goet Na vrede allien // wilt Want God ten tien // wil haer Die hem ghemien // Oock haer ghebeen // en droevich Wilt de Heere alleen // Die haer afscheen // en boosheyt Wil dit Heere van een // Die Heere voldoet // t'ghebedt seer Eens verslaghen ghemoet // vol En hy is goet // die tot hem Ghebroken vroet // van Die gherechticheydt // voor die menschen Van t'quaet hem scheyt // met Hoewel hy schreyt // hem zijn En toegheseyt // veel Maer die nu lacht // en wort Van s'menschen gheslacht // En toont sijn macht // over Die uyt al haer cracht // Godt Die wil God weer // werpen ter Sal worden seer // Door Godes leer // die Sal nemen keer // ist Hierom ghy al // der Christenen Treedt den wegh smal // na Al rijster bal // druck Hier in dit dal // der Voor t'cleyne verdriedt // d'welcken hier En wilt doch niet // so Maer hier op siet // wat God En ons aenbiet // te Hy roept comt by // die beladen Ick sal u maken vry // van Scheyt uyt het ghecry // wilt wesen Mijn volck sult ghy // zijn Mijn jock is soet // dat op u Mijn last is goet // den Maer t'is al roet // en Die sijn verwoet // mijn O Princelijcke Godt // zijt ghy ons deel en Bidden wy in slot // ons Teghen s'werelts spot // door u Sterckt uwen rodt // haer Vwen gheest sendt // dat wy T'gheloove jent // En s'Hemels tent // na desen Seer excelent //
