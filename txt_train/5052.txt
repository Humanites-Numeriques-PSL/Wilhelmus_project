Ghy schoone Roos // die onder ‘t Hier zijt altoos // wilt u Dat u is opgheleydt // tot Dat ghy niet en sout // door lust Dat ghy niet en sout // V kleur, door ‘s Werelts sot V kleur is schoon // door het Van het uyt roon // ende Van al u lusten quaet // en Van ‘s werelts hoocheydt // en haer Van ‘s werelts hoocheydt, ende Die in de afgront moet Ghy Bruyt Christi // laet vry V naem // sijt bly // in u Lijdt stil het steecken wreet // die u doen Toont u sachte moet // en diepe Toont u sachte moet, Vergelt het quaet met goedt, Wijckt hier niet uyt // wilt liever Dan lydens kruyt // te doen Want ghy moet dragen ‘t kruys // sonder Wilt ghy dat niet doen // maer tot wreecken Wilt ghy dat niet doen, wilt Soo is u arbeydt V Bruyd’gom // heeft veel En heeft daerom // voor haer Die hem deden verdriet // hy wreeckte O exempel schoon // van Gods eenich O exempel schoon, Volcht hem nae met een vast Met hem sult ghy // eeuwich Soo ghy hoort my // met hem wilt Dan sult ghy in sijn throon // ontfaen de En aenschouwen soet // ’t lieff’lijck En aenschouwen soet’ Gods aenschijn in dat eeuwich In ‘s Hemels hoff // sult ghy Daermen Gods loff // lieff’lijck Met nieuwe tonghen bly // een Soet, sonder verdriet // want daer ghebreeckt Soet sonder verdriet // sal ‘t Lam Gods met syne Bruydt O Bruydt Christi // siet u In u Heer vry // wilt doch Wat hy u schencken sal // nae ‘t lyden Hebt dan gheen verdriet // in u lyden Hebt dan gheen verdriet // lijdt Draeght met ghedult uws Heeren
