ONlanghs ick savonts Ick weet serteyn wel Luystert hier naer // ick bid u Ick sach te gaer // twee Meyskens Sy praten // wanlaten, van Vryers En bedreven soo Ick dachte // en Jae wel hoe soet is oock, is oock de Ach! Trijntjen sey de Ick ben nu soo Want my strackx quelt // een jongher ’t Sy u vertelt // want met Wil trouwen // en houwen, myn voor zyn Tot dat de bleecke Sal scheyen // ons Dus ly ick staech een swaer, een swaer Wel Marritjen myn Ben ghy daerom Soo laet hem staen, en heenen Doet dit vermaen // of ist maer Wilt proeven // niet toeven // of hy Sijn Ambacht heel wel Ghy weet wel // By myn verholen blyven, blyven Maer Trijntje lieve Speelnoot t’sy u Ick heb ghewaecht // ken ben gheen Want siet ick draecht // myn Moeder Wat schort-jou, quyl-beck, dat ghy dus En ick die ben dan Mijn herte // vol Want siet mijn tijdt die is, die is haest Mijn rekeningh is Eylaes! ick sterf van Want Jan syn trouw // niet houden Foey hem ick spou // hoe garen Ick wreecken // en spreecken // hem hier op Dat hy myn soo Met looghen // Heeft hy myn strackx in slaep, in slaep Maer Marritjen myn Ghy bint voorwaer niet Al gady groot // dat is gheen ‘K lach oock eens bloot // en Keesjen Quam lustich // en rustich // als op myn Hy weerden hem soo Een Soone // te Ick binnen neghen Maenden, Maenden Hoort doch toe hoe ick Ick ginck besteden Dit wel versint // want siet ghy In de saeck blint // want ick t’Kints leven // maer even // twee Maenden De Heer die gaf die De Vryer // de Betaelden d’oncosten ghelijck, ghelijck Ach! Tryntje tis soo Ick meen gheleden schier // acht maenden Dat ick goe chier // met Kees Jaep Eens was te Hy soende // al doende, ô myn wat Ick dencker stadich Myn Billen // die Ick sterf eylaes! vraecht niet // vraecht niet Princen ick scheyde Ende sy ginghen Ick hebt ghehoort // en was Doch onghestoort // noch Myn dochte // sy sochte, en Vryer Ghelijck veel Meyskens Ick laete // die En wensch myn eyghen lief, myn lief een
