VRolicke geesten // die liever Op Kermis-feesten // als inden Ist u behaeghen // rijt vaerdigh Treet op de waghen // of in de De vrienden van 't spaer // die nemen u Weest wellekom gasten // Wel benje Siet wat hier binnen // Den kramer Om wat te winnen // Veylt hy sijn Den waeffel-kraemer // Ick heb seydt De beste kaemer // Gaet niet Dus wordje van d'een // En d'ander Om dat-je sout // Jouw gelt V gelt verhandelt // u tijdt Nae buyten wandelt // En daer wat Dan met u Liefje // daer weer in 't Steelt Snobbel-diefje // dan soen op Of graest u bemint // soeckt boompjes Of doet daer ghy // vermaeck in Leyt dan dat meysje // daer ghy mee Nae duyn een reysje // of soose Haer swacke leden // niet toe Soo veer te treden // blijft dan in't Of geeftse haer stem // nae Roomen 't heeft In Emaus, of Siet daer de brackjes // van Drincken niet zwackjes // 't Haerlemsche By drie vier glaesjes // van bier, of Noyt fraeyer baesjes // wout altijdt Een vijf of ses vaen // kan dan daer niet 't Is kermis // 't moeter nu De gheest moet woelen // van buyten Stracx nae de Doelen // elck set hem Daer weer ghedroncken // tot datse Smorgens beschoncken // de poort weer Daer menich in 't wout // in 't velt Of op een banck // sijn nacht-rust D'een zijnde wacker // geeuwt als een D'ander sijn macker // siet wonder Hoe duycker koom ick // dus op een Wat drommel droom ick // 't is door den Waer quam ick dus krom // ick wed dat ick Het jaer niet weer // te kermis
