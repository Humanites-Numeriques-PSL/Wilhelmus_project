NAer dien dat ick Subiect, en dienstbaer En mijn vryheyt Geheel te buyten Meer Dan my rebel // Heb ic, naer Alsulck een Wert namaels wel // Edel, dier // en // Is zy, die t'hert Al dreyght vier, en // Ick stae tot haer Deed'- ick sulcx Soo waer ick fel // Wie twijffelt Sulck tijt Wert namaels wel // Een alsulcken Blijf ick eygen Wert het eynde Soo sit ick edel Als Coninck Gans Israhel // T'sy hoe dat De minste Wert naermaels wel // Liever smert // en T'aenveerden, angstich En haer t'zijn Een knecht, drie seuen Soo werd' ick T'gewonnen spel // Dat dunct my Getrouwen Wert namaels wel // Keert sy mijn Te zijden, als yet My arm Toelaet luttel T'sal werden Dat mijn gequel // Want die Verdraeght, en Wert namaels wel // Es oock meerder Vervaerlick elcx Dat een dorper Geniet der minnen Venus Sonder bevel // Och! t'is te Dat sulck Wert namaels wel // Niet versagen // Mijn Princelick Liever dragen // Des weerelts Want diet my Is so ydoonen Voor cleyn Ghy namaels Soo wel beloonen //
