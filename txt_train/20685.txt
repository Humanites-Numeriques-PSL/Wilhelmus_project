DOrathee ,, ach! uw' twee Susters, God Sy besoecken // ('t wil niet Of Ghy, g'lijck als sy, wilt Sy Maer gehecht // en Zijn met All' uw' sinnen // all' uw' Aen den Heer van alle G' hebt geleert // G' hebt Uwe Susters En Sy leden // en Sy Voor de leeringh Christi Groote nood // ja de Hebben Sy Hooge gaven // 's Hemels Kregen Sy na dese Doen zijt Ghy // in de We'er daerom In die dagen // hebt Ghy Lustich op uw' le'en Ey! Ghy hebt // en Ghy Vrolijckheyd in 't In het blaecken // in het Uwer le'en, Ghy U Ghy ginght heen // Ghy ginght Na 't schavot U verbeyden // 's Hemels En Ghy quaemt daer in gansch Uyt dat land // heeft een God met schoone Ne'er gesonden // op dier Die Theophilus sagh Dese man // nam die En Hy sich Ja sijn leven // heeft Voor den Heer dees 's Hemels lot // schonck hem Voor dees Om sijn deughden // is van Hy, en Ghy, gestadich Och of Ghy // oock op Dorothee, woud Ey! bidt krachtich // God Dat Hy my sijn Hof wilt
