DEes Goddelijcke Nimph is soo Dat sy noyt Liefde weerlief Want d’Harder stout // van soete Die sy al schempende gaende Kranste met Laure kruyt // dees Knielende met Of Tyter klaecht // om hulp haer D’afkeerige Maecht // seyt altijt Hoe dickwils Tyter heeft verlaten ’t Om dese schoone Quelent van min // om dees Vluchten haer nae ten Boswaert Doch as sy hem komen sach // Klonck door het Bosjen Of Tyter klaecht // om hulp haer Dafkeerighe Maecht seyt altijdt Lest vont ick haer alleyn by een Wies haer Albaste Handen Kuerschuylde ras // wat vreucht myn Als sy int Waeter lobberent So haest sy myn wiert gewaer // liet Myn doen Of Tyter klaecht // om hulp haer Dafkeerighe Maecht, seyt altijt Hadieu versteende Vrou // helaes ick Duer u stuershe-den van u!
