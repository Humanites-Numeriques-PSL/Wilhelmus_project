O Mensche // Die nu Gods heyligh woort // Ten wensche // Brenght vruchten in accoort // Verduldigh // lijdt ghehuldigh // Verblijdt in 'sHeeren // kruyce vol Als Godt sijn kindt, bemindt, sorchvuldigh Onschuldigh quijt, hy 't door Van 't Helsch zwaer // misbaer maer den Die moet hier naer // lijden Int vyer, tis klaer // dus wee die Niet en Onlanghe Der beproevinghe vyer // Die ganghe Een lacchen goedertier // Met vreuchden // bloet // in jeuchden // Eeuwigh na desen // salt al Bidt en waect // en maect // In deuchden // goet, looft Godt Int foreest, en vreest, eest, geen Gods woort ghekeest // ter herten Alsoo wel heeft // laet inden Vruchtbarich Den stercken // Des gheloofs neemt nu an // Aenmercken // Christus uwen Hooftman // Sijnen weghen // bloot, door teghen // En wilt niet wijcken // noch oock Loths Wijf // wiens lijf // heeft cregen // Versleghen // doot, door't Wert sy een, soutsteen, gheen teecken 'tWelck sy ghemeen // wel moghen Trecken te been // om int Niet te Int Aerden // Soo den Prince behaeght // Den waerden // Nae vrede onvertsaeght // In smeecken // zijn, wilt weecken Bedd' en slaepstede // in den Als Paulus // doet dus, int reecken, Na teecken // fijn, met Dagh en nacht, heb acht, wacht, nae u Die met heyrkracht // komt Op dat ghy lacht // by hem Eeuwigh in
