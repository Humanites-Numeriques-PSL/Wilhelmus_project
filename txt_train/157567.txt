O Heere Godt van machte // grootVoor u sijn mijn gedachten // blootDaer en is niet soo ergen // man,Die hem voor u verbergen // kanWant ghy binnen en buyten // sietMen kan voor u opsluyten // niet.Tis voor u als den lichten // dachGheeft in my dat stichten // machEenen nieuwen bequamen // geest,Met al die uwen name // vreestDat ick mach hier beneden // pleynWandelen in der vreden // reynSonder de vrede seer dier // baerEn sal men mogen sien hier // naerGod in zijn eeuwich wesen // fijnDe knecht sal niet gepresen // sijnDie met onvrede quellen // gaetEn zijn medegesellen // slaetMet stock ofte met roeden // stijfEn meen ick op dat goede // lijfmaer met der tongen haesten // praetSpreken van zynen naesten // quaetSoo met der tonge griefde // ratMaer niet en leert de Liefde // datAl trock ghy ooc een blodich // mesTegen u vriendt die nodich // esEn u goet principalick // rietSoo doet ghy noch soo qualick // nietAls dat ghy met versmaden // maecktHoe men in kromme paden // raectDie zynen mondt tot laster // wendtDen vre hy veel tonvaster // kentWant na Syrachs gewagen // goetHy doet van al zijn dagen // boet,Die hem tot qualijck spreken // gheeftEn selven vol gebreken // leeftHeere bewaert my in de // patNaer die reyne beminde // stadtMijn tong' en my beneven // dienDat ick hier na mach tleven // sienEn goede dagen mede // voort,Och! Heere mijne bede // hoortHeer mijn gebedt verhooren // wiltDat daer door uwen tooren // stiltOver alle mijn sonde // boosDoor u genade gronde // loosVergeeft my dit altsamen // vroetWant ick my voor u schamen // moetLaet my Heere voor u vry // gaenVan sonden als den Publy // kaenDie voor u inden Tempel // batGeeft my na dat exempel // watDies ghy na u verklaren // vastQuijt maeckte van den zwaren // last.Of ick u soo behaeghde // reen,Heere ghelijck als Maegde // leen,Doen zy verkoos dat beste // deelScholt ghy haer quijt de reste // heelDer sonden tot versoeten // diesSy met tranen u voeten // wies.Veel biddens zy met tranen // deOock dat vrouken van Cana // neSy badt met groot betrouwen // snel,Voor haer dochters benouwen // felOotmoedich en van herten // leechVan u zy haer begheerte // kreegh,Haer dochter deet ghy Heer bystantSy was gequelt van den vyandtDeed haer veel persequeren // anMaer ghy saecht dat verneren // vansMoeders geloove machtich // sterckSoo toonde ghy u krachtich // werck,Seer veel krachtighe daden // soetHeere ghy deur genaden // doetWie tot u met geloove // quamMelaetsche, kreupel, doode // lam,Den blinden sach, den stommen // spracGhy hebt van haer genomen // tpackOch! Heere God bermhertich // vaerDen man die achtendertich // jaer,Opt Bedd hadde geleghen // kranckHebt ghy kloeckheyt en degen // ganckWt liefden hem goet ront // ghejont,En maeckte hem terstont // Gesont.Princelicke God devijn // sijt mijnEen eeuwich medecijn // met wijnEn Oly my inwendich // bintWaer dat ghyt Heer elledich // vindtIn my een gheest verslaghen // scheptDaer in ghy een goet behagen // hept.
