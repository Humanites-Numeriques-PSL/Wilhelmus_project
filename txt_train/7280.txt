Wy Spoelders jonc van dagen // Moghen als arme Van ongheluck wel klaghen // Men hoordt singhen noch Van onse Konst verheven // Ghelijckmen wel van ’t Weven // Men acht ons slecht als Daer wy nochtans, als De Kroon in Hollandt Niemandt doet ons verlanghen // Het mach ons wel Wy moeten altijdt vanghen // En gheenen lof ’t Is Guyten hier, en Leckers // Ten zijn Spoelders of treckers // Wat isser aen Gheeftse den gons // doch sonder Wat soudet zijn van ’t Maer dit doet ons tot leeren // Het weven jongh van Want wevers dat zijn Heeren // Soo haest zy laten Het Spoel-wiel als een snoo Men kander niet met houden Zy willen zijn het Verdrincken ’t gheldt, gaen Ghelijck een aerdich Het mutsken hanght om vallen // Het hooft als wilt van Met een wanbays van smallen // Ghebreyde Kousen Den mantel op oock snijen en Al is men eenen Turf maer Men wil niet houden zijn Men willet al te Den Borgher vroet // selfs wijcken Iae schande dunckt haer Den maendag is niet schaed’ // Meer daghen in de Moetmen noch drincken spaed’n en Tot datmen van Seer ’t achterkant aen ’t weven // Als dan ons wel te beven // Oyt vreese te Den Deunstock stijf // al op ons Hoe seere dat wy Want zy gaen weven nacht en //
