BRoeders houdt vast aen Gods Al wordt ghy hier der menschen Haer fabel gherel // haer liedt, haer Menschen saen // al Maer indien men Volherdigh Godt aenkleven // Men krijght dat eeuwigh leven // Wat noot is De Christen Menschen zijn, of Ter werelt slechts reysden Ghetroost int cruys // om wesen By den Heer // Wat Wy in dit Lijden, soo wy gheraeken // Soo staen gheheel ons saken // Wat noot ist T'schijnt wel een treurigh dingen Als 'sHeeren castijdinge Doch 't aertsch verdriet // gaet haest te Maer den loon // wonder Is seer groot hier Als naer soo weynigh lijden // Ons deel 't eeuwigh verblijden // Wat noot ist Mishaghen wy de Menschen // Dat sy ons niet en wenschen // Dan lijden, druck // en Als die weert // zijn het Iae doodt oft Oft seght m'ons veel blamatie // Staen wy wel in Gods gratie // Wat noot ist En hadde 'sHeeren Woordt David waer in ellend Want t'eenen troost // hem Godt Ter noodt gaf // desen Daer hy op Dus als ons komt benouwen // Soo wy Godts Woordt betrouwen // Wat noot ist De Heere groot, sonder Ginck door veel lijdens in sijn Wie nu ter tijdt // des lijdens Wel ghesint // oock Soo hy Die werdt ghekleedt met witten // Alle dinck hy besitten // Wat noot ist Paulus hadde victory, T'gheloof als eenen weerden Behouden reyn // had hy int Ongheschendt // tot den Soo hy De Croon was sijn verwachten // Konnen wy hem gheslachten // Wat noot ist Prince wat noodt ist, oft het Niet al en gaet nae sijnen Als nae Godts wensch // den binnen T'alder stondt // is En soo kloeck Dat men tot 'sHeeren glory // In sijnen Naem victory // Wat noot ist
