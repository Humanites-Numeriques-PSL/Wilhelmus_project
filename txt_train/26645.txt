GHy Christen schaer // Neemt uwen roep claer // wel Om verblijen hier naer // Gheen scheyden en werdt Als rechte Volghende Christus Laet ons beloften Dat wy vaten ter Oorbaer tot elcken Wesen van den Dat deur goet ghereucht // Den blinden met vreucht // tot En op dat ghy meucht // vol Wassen in Christo Wast nu ter tijt // met Als rancken wijt // int Op dat ghy zijt // Van Gods toorn Die hem sal Soo Paulus Op alle boose Die nae Godt niet en T'sal noch worden Wie in hem gheen vrucht en Sal hy nae Schrifts bedien // Dus wilt wel in tien // End' de boosheyt vlien // Sal hy, die Godt De boosheyt quaet // Niet met den praet // maer Met goeden raet // nae Christum den Herder Die ons heeft Met olye, en met End' veel liefde Als onsen Hy droech al ons Deur sijn woorden Ghenas nae Schrifts vertel // seer Israel // oock Van der slanghen fel // Was doodelijck T'fenijn seer bloot // was Dat der slanghen hoot // Maer deur Christus doot // Is dat tot niet Satan die heeft Sijn gheweldighe Daerom is hy vol Soeckende dach ende Laet ons houden goe Het uytvercoren Gaet hy met behent // Als een Leeu bekent // die Op dat hy verblent // Van Godt hooghe Gaet onghestoort // doch Door die enghe poort // soo't En maeckt gheen discoort // int Als eerlijcke En wilt niet Ghelijck de Die den Heere En verstootten sijn Dat was meest dat sy Want sy waren van En hebben den Heylant // Aen een cruys ghespant // met En dien hy sant // int Die hebben sy oock Satan bevecht, Gods Die nu t'onrecht, Soo ons Iob seght // die Godt vreesde en sijn Sijn wijf die hem wel Die heeft hem oock En sprack in sijn Blijft ghy noch vast by Iob die antwoorde Haer: ghy spreeckt als Wijven Want hebben wy dat goet, seer Met ouervloet, men Oock teghenspoet, Ontfanghen voor ons Prince vermaert, Met goet regaert, Want onsen aert, De Ziele In dit leuen Wel vast op Godt Op dat de liefde Nemmermeer en En wordt nemmermeer Al wordt ghy hier Met veel boos, verknijs, Soeckt met auijs, seer Int Paradijs, u Van dat leuende
