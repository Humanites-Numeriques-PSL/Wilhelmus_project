O Heere Godt van machten // Voor u zijn mijn ghedachten // Daer en is niet soo erghen // Die hem voor u verberghen // Want ghy binnen en buyten // Men can voor u opsluyten // T'is voor u als den lichten // Gheeft in my dat wel stichten // Eenen nieuwen bequamen // Met al die uwen Name // Dat ick mach hier beneden // Wandelen in der vreden // Sonder den vrede seer dier // En sal men moghen sien hier // Godt in sijn eeuwich wesen // De knecht sal niet ghepresen // Die met onvrede quellen // En sijn medeghesellen // Met stocken oft met roede // En meen ick op dat goede // Maer met der tonghen en haesten // Spreken van sijnen Naesten // Soo niet der tonghe griefde // Maer niet en leert de liefde // Al trockt ghy oock een bloedich // Teghen u vriendt die noodich // En u goet principalijck // Soo doet ghy noch qualijck // Als dat ghy met versmaden // Hoe men in cromme paden // Die sijnen mondt tot laster // Den vred' hy veel t'onvaster // Want naer Syrachs ghewaghen // Hy doet van al sijn daghen // Die hem tot qualijck spreken // En selue vol ghebreken // Heere bewaerdt my in de // Naer die reyne beminde // Mijn tonghe en mondt beneuen // Dat ick hier naer mach t'leuen // En goede daghen mede // Och Heere mijner bede // Mijn ghebedt, Heer, verhooren // Dat daer deur uwen toorn // Ouer alle mijn zonde // Door u ghenade, gronde // Vergheeft my die al t'samen // Want ick my voor u schamen // Laet my, Heere voor u vri // Van zonden, als den Publi // Die voor u in den Tempel // Gheeft my naer dat exempel // Dies ghy naer u verclaren // Quijt maecktet van den swaren // Of ick u soo behaechde // Heer ghelijck als Magde // Doe sy verkoos dat beste // Scholdt ghy haer quijt de reste // Der zonden // tot versoeten // Sy, met tranen u voeten // Veel biddens, sy met tranen Oock dat Vrouken van Sy badt met goet betrouwen // Voor haers dochters benouwen // Ootmoedich en van herten // Van u sy haer begheerten // Haer dochter deedt ghy, Heer, Sy was ghequelt van den Deed' haer veel persequeren // Maer ghy sacht dat verneren // S'moeders ghelooue machtich // Soo toondet ghy u crachtich // Seer veel crachtighe daden // Heere, ghy deur ghenaden // Wie tot u met ghelooue // Melaetsche, Creupel, Dooue // Den Blinden sach, den Stommen // Ghy hebt van haer ghenomen // Och Heere Godt barmhertich // Den Man die achtendertich Op't Bedt hadde gheleghen // Hebt ghy cloeckheyt en deghen // Wt liefden hem goet ront // En maecktet hem terstont // Princelijcke Godt diuijn // zijt Een eeuwich Medecijn // met En oly my inwendich // Waer dat ghy't, Heer, ellendich // In my een gheest verslaghen // Daerin ghy goet behaghen //
