Ia des Meys tijt, verblijt // nu int Sonder respijt, subijt, die jonghe Met groot jolijt // en vlijt // is elcken In daertsche krijt // want zijt al doen met Om also met gheneucht // te treden na de Want Bloemkens reyn // int aertsche Groot ende cleyn // staen over Lieflick certeyn // breet ende Met desen coelen Aurora goet // die moet // als dan oock De kruyden soet, die doet // sy groeyen Den water vloet // met spoet // vliet lancks de Die Godt behoet // seer Vroet // int loopen Der Amoreusen spel // Hoortmen in die Als Luyten // Fluyten // soeten Ontsluyten // wten // thert Daer buyten // ruyten // voglen Met desen coelen Hoe triumphant // playsant // moetet daer Waer met verstant // die hant // een wort Die lancx den cant // van tlant // gaen om der liefde brant // en vant // door soet Daer twee gheliefkens staen, plucken Die in eeren soo paren // Onsen Godt wel vergaren // Soo krijcht een jonckvrou haren // Stracx inden coelen Princen eerbaer // te gaer // wilt Liefs schoonheyt claer // van haer // niet en Oft groot misbaer // voorwaer // sou u Daer ghy eenpaer // minnaer // af wort Tis goed wel overdacht // worden Soo ghy u lief ghewinnen // Suldy dan selfs bekennen // Waer door u thert van binnen // Gheeft met den coelen
