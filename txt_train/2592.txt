O Droeve tijdt // die ick Met vruchteloos O aertsche Hel // doot mijn Door t'snoeyen van mijn Ick ben alleen // verd van de En tegen danck Helaes! van die // ick 't liefste Van alle jonghe Ach ongheluck // ach leyde Sult nimmer ghy Dit hart ghewelt // dat nu dus Dees teer benaude O schraele doot // comt en door Dees vrouwelijcke Ach! strengh gemoet // dat so Na u Kints bloet gaet Mijn Moeder wreet // doet my dit En ghy versufte Een Keysers kint // om dat het Besluytmen straf int U fel ghegrim // u wreetheyt U dreyghen, noch u Geen schric noch noot // noch duyvel En can myn Liefde Maer als ick sal // int traenen Door ongheneught Mijn schim bevreest // of bleecke Sal nae myn doot En quellen swaer // myn Ouders Mijn Echte Man Die sal ick dan // waer in ick Noch even dienstich Mijn Prinsch vertoeft // weest niet Bedaert, u wilt Mijn hart blyft u // al ist lyf In een Kercker De trou en eed // dien ick u Die zullen geen sints Hoopt, en vaert wel // de dinghen Op aerden licht
