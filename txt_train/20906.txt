WEest verblijdt // t' allen 's Hemels Aleluya // Ick U opdraegh all' mijn Want Ghy hebt, die 't all' Maeght! in U Alleluya // God in U had groot Desen Heer // heeft sijn Uyt het graf Alleluya // En is we'er op d' aerd' Onverwacht // uyt het Is uw' Soon Alleluya // G'lijck voorseyt Hy had voor Weest verheught // weest Waerdste Godes Alleluya // Opgestaen is ons' Maget bidt // dat wy 't Van den Hemel Alleluya // En dat wy ons God
