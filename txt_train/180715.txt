AL die hebben tot jaghen // Staet op en gaet int En hebt niet meer behaghen // Den slaep, die menich Daer is te vaen, o Iagher dat u T’eeuwich welvaren // T’eeuwich leven is ons Elck als Helt // vlijt en Hier toe openbaren // DRIe dinghen zijn bevonden // Bequaem tot deser Ten zijn netten noch honden // T’is t’gheloove vol Hope, en de Liefde boven al Om wel te vanghen // Niet hier // een dier // onder wilt Maer een sacht // Lam vol Wel weerdt om verlanghen AEN dit suyver ghepresen // Al ons welvaren Elck Iagher sonder wesen // Volght hem en niet en In zonden meer, maer schrickt daer Den Heer weest onder // Iae Iagher u hem ghevanghen Die hem heeft // eeuwich hy Dus cleeft // hem bysonder // HOEFven wy int ellendich // Yet beters t’onser Och neen wy, maer die blendich // Van hem afwijcken Haer lusten jaghend’, en niet achten T’lieflijck vermonden // Maer speelen hier met der zielen In den schoot // vanghen den Want dat is der zonden // NA t’beste moetmen speuren // Meer als nae een wildt Dewijl het ons ghebeuren // Ontvlieden t’eeuwich Der ghenaden tijt niet door licht Willens verliesen // De doot fel oft t’eeuwich leven Van tween hier // om Men t’beste verkiesen // GELijcken niet den rouwen // Esau moeten Maer Godt altijt betrouwen // Als Iacob, dat als Ons deel der ghebenedijdinghe Maer soo ick proeve // Om te vanghen dat Een noodich Vind’ ick my seer swack, waer Gods hulp ick behoefve //
