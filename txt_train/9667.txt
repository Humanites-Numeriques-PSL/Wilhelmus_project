DEn aenghenamen // Die is alsnoch Och doet te samen // Menschen in alle Om vanden slaep der Als nu ten Opstaen, tis tijdt Laet zijn De ooghen, die van binnen // Herte zijn, laet u sinnen // Met des Gheests Ooghen salve Salven, Men hoeft klaer ghesichte // Anders grijptmen lichte // Iae de schaduw' voor t'rechte Blinde leydslieden // Aen haer vruchten en V snel te vlieden // Bedrieghelijcke Opmercken, is Om niet te Want men vindt Veel Satans Alwaer men valsch' afgoden // Oft der Menschen gheboden // Sulcke loghen Stellen haer, op het Christi Sy souden verloren // Leyden d'uytverkoren // Waert moghelijck door haer S'vleeschs ydel wenschen // Moetmen al van hem Niet doen nae Menschen // Noch voor menschen Noch oock behaghen Noch houden So Bosors sone Het zy door Oft beloften, maer Moetmen Christum byzonder // Voor desen Heer Moet yeder, nae dit Rekenschap Die de goede kroonen // Maer sonder verschoonen // Straffen die quaet hebben Al die ghelooven // Verwachten te Schat, die men rooven // Can, noch sal in die Verderven, noch Door roest, noch Godt laet hem niet Tis niet te Al wat de Menschen zaeyen // Sullen zy namaels maeyen // Elck wil te waken De Heere sal den Seer haest Den tijdt der ghenaden // Die veel nu versmaden, Voor eeuwigheyt gheheel
