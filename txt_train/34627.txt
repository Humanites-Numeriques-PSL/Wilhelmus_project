ACh ! wat een smerte // braeckt het minnen Wanner een herte // sijnen brandt Sonder te klaghen // daer 't niet kan Het vinnigh pranghen // van het groot Het stoute minnen // seght my stadigh Dat ick mijn sinnen // soud' verklaren Maer bleeuwe schaemte // en vrees' van de Doet my vertraghen // mijn Lief naer te En noch doet hy my // d'alder-meeste Om dat hy by my // mijn Suster komt Ach ! hy is stadigh // fel en seer Ontrent mijn deure // 't welck my meer doet Dus ben ick even // Tantalus die Naer 't vocht dat neven // aen sijn lippen En noyt kan proeven // dan een staegh Om dat sijn lippen // noyt aen 't water Wat heeft hy aerdigh // doch in haer Dat sy is waerdigh // boven my Noyt wirdt haer wesen // boven 't mijn Noch haer soet spreken // teghen 't mijn Ick ben niet spijtigh // soo hy van haer Maer siet ick vlijtigh // kom' hem Wilt hy mijn banghe // ziel' maer eens In sijn ghenade // eer het wordt te Ick sal versaken // mijn Suster ter Midts ick niet haken // magh haer blijde Maer wil eer sterven // en oock metter Mijn borst roodt verven // met mijn heete
