O Hemel! komt op de-se Komt hier Komt Machten, U by Haer Met de Siet eens aen // want hier Op dees baen // Sint Francis // U Ee-ne man van deughden Hy van liefde brandt // en des Hemels Sal met groot Nemen in dees vroomen De Armoe heeft bemint Hy En d' Als eenich mensch oyt geld, of Of All' sijn goed // Hy En sijn bloed // ey! dit siet // 't is Hy volght naeckt sijn Jesvm Hy treckt uyt sijn kleed // Hy sijn goet Hy sijn Vader Hy neemt aen een and'ren Soldaten neemt Franciscvs Mannen Met welcke Hy des werelts Heeft tot God Wonder is 't // ('t dient Hoe den twist // ne'erlgeleyt // Hy By sijn God, die eeuwich Op de menschen gram // was 't onnoosel Om haer vuyle Godes haet, Francis Een sware strijd heeft Hy Van Hy vroomich toen de sneeuw Hy sich Siet! Hy woelt // en Hy 't Geen Hy voelt, 's lichaems lust // en All' de lust gaet aen een Siet met wat gevecht // Godes trouwen Na des redens Onderbrocht sijn dartel Hy heeft de Moeder Goods Bovenmaten Want Hy wist dat haer lieve Christus, onsen All' die haet // en Ja verlaet // die kleyn acht // dees Die is onser all' In all' sijne nood // tot dees Maeght Hy Hy verkreegh all 't Dat Hy socht door haer Ontrent den Feest van 't heyligh Op den Bergh Quam tot Hem ne'er uyt 's hemels (Siet eens van hoe Eene strael // die Hem Ach! ick dwael // Hem genaeckt // en Eene Seraph, die God Wonden vijf Hy gaf // welcke 't bloed liep Handen, Voeten, Waren toen in groote Gaet henen Seraph, vlieght Seght dat Sint (Soo als getuycht ons eyge G'lijck aen Christo Zy, en voet // d' handen Zijn bebloedt // ach! hoe wee // de'e Godes uytverkoren D' Aerd' is ongewoon // t' hebben Godes Of een in die Desen moet in d' Hemel Francisce! Gaet dan, spoedt U Spoedt na Godes Bidt dien Ghy g'lijck zijt, om Voor my, slijck en Ach! ick swerm // op dees En ick kerm // g'heel vervaert // door d' Werft Ghy my het eeuwigh Och of ick eens sagh // nae mijn stervens 's Hemels schoone En verkreegh des Hemels
