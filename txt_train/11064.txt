EEn  danckbaar liet // der Heren Heer Die op ons neder siet // om weder te Van t'groot verdriet, dat velen bracht te Door swaren siecte hiet // zeer qualick En merckt hoe Dat God den De Menschen teer // Als hy sandt Ghenade Wie zal zijn eer // in deesen nieuwe Iaar Wy die in noot // van hulpe scheenen En telcker uur de doot //zeer broesselijck Door vreese groot // veel volckr ons Om sulcken swaren stoot // t'ontgaen met haer Maer Doet buyten Haer oock met schant // Want Godes En can Hoe seer vayliant // tot geener tijt van t Iaar De mensch, gemien // wel spoedich tracht De plaetsen daermede sie // dat God bereyt zijn Maar t'punt van dien // waar door de straffen Wy uyt ons niet en wien // dat meest is te Wou elck met Het quaet Wt zijn ghemoet // De Heere Liet metter Die straf ontsoet // int nieuwe Iaar wel achter O Mensch verheeft // zijn lof die eeuwich Door dien ghi met en bleeft, in dotelick Zijn hulp aencleeft // ootmoedich u Voor hem daert al voorbeest, dat leven heeft De wijl hy En u Om troost en baet // te Die sich Op zijn En metter daet // int nieuwe Iaar zijn woort Hierom besnijt // u hart en ziel met En wilt tot alder tijt // des Hemels Prins U schult belijdt // en maeckt u alles Dat teghen t'ghoede strijdt // soo suldy u Om met De Daar elck na beyt // te Als Christus Wert in T'rijc is bereyt // voor u mijns vaders wt
