Kedor Laomor, die oock Wel joegh uyt spijten // die Maer dat hy vinck het was Esau wesende Iacobs Van eener Moeder // Rebecca Iacob vercoos als een wijs Eenen hoeder // der schapen te Maer Esau // seer ru // op dat Was een Iager, soo die schrift Meest al sijn leven // daer in Maer dat hy vinck het was Iosua wesende licht ter Heeft met ghewelde // oock vele Die Coninghen eenendertich men Die als rebelde // waren Binnen Canaan // verwan // seer Iosua al haer Diet anders wagen // sonder Gods Al watse vanghen is Gedion naer des Heeren Is wel ten rechten // ter jacht Vergadert met sijn drie hondert Ghestelt om vechten // en joegen Nae haer begeer // dat heer // van Die int landt waren uyt Om te verderven // des Heeren Maer wat se vingen was Ionathan zijnde sijns levens Was een Iager // niet van Hy track op met sijnen Wel een verslager // der Hy bracht // met cracht // en deet in doodts D’ander en hebben hem niet Elck henen vluchte // door dit Maer datse vingen was David die claeght dat hem sijn Stricken en banden // leyden Om hem te vanghen tot sijner Maer door sijn handen // den Heere Doen hy soo plat // den Goliath // Wie hoorde oock van vromer Die ander hoopen // moesten gaen Al watse vinghen was Nebucad Nezar, nae Schrifts Heeft oock ghesonden // een Heyr seer Die hebben Landen en lieden Als Leeuwen monden // wreet en Sy quamen // t’samen // tot Hebben Bethulia T’was haer bedencken // haer al te Maer datse vinghen was Veel wilde Iaghers nae Schrifts Hebben veel Iaren // dat volck Sy troosten haer met vredighe Oft leughens waren // het werdt Men stack // tot ghemack // pueleins onder Dat heeft den Heere quaelijck Soecken de Zielen // soo te Maer dat sy vinghen was Aen wilde Iaghers het noyt en Elck vierich straelde // liep Waer door soo Ieremias Dat volck seer dwaelde // maer Vol van bedroch // zijn doch // als Herders Hebben die schapen qualijck Die dus omginghen // met sulcke Al watse vinghen // t’was Paulus joegh maer noch niet Want tot Philippen // ghetuyght hy’t Sijn vaste hoep is hem niet T’stont soo gheschepen // hy joeghder En hadde lust // met rust // te sijn by Maer liever by den Heere Die anders leeren // tot groot Al wat sy vanghen t’is
