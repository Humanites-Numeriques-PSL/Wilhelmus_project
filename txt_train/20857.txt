O Hemels groot Xaveri! Met deugd verciert in d' hoogste Ick wil uw' deughd // en uwe Verhaelen hier met 's harts 'k Wil schrijven // 'k wil stijven // uw' eer // uw' Staeghs meer en Want ick mijn min // en al mijn Op U gestelt heb van 't Van uw' Xaveri! kan worden noyt genoegh Met eene I // gingh vlijtigh De werelt rondom even Ghy eerden // Ghy leerden // te saem // de En Jesus Met groote vlijt // tot 's vyands Gheen tegenspreken Ghy en Ghy had soo suyv'ren Xavieri! dat vuyl gedacht U was een Want als by nacht // een vuyl Den Duyvel U had Van minne // van sinne // Ghy bloedt // hoe Was dese Het speet U dat // uw 's lichaems Oock in de slaep sijn wet Uw' yver was soo Xaveri! dat Ghy niet hebt ontsien de U drouf gesteen // u drouf Vermurwden harten hart als Ghy smeeckten // Gy preeckten // staegs aen: // Goods Deed' Ghy 't volck Het speet U seer, dat God uw' Vergeten worden, en sijn Ghy waert Mariaes Xaveri! Sy achten U te zijn haer In all' uw' rouw // was dese Een troost aen U, ja u In 't lijden, in 't strijden stondt Sy // U Ghy raeckten Des helschs geweld // heeft S' ne'er Ghy wiert door Haer soo vroomen Ghy sijt geweest Xaveri! een Prins in 't Indiaensche Dat land geleert // dat land Hebt Ghy, en met Goods Wet Wat loont // wat kroone // wat lot // geeft Van Aen U sijn vrind // die Hem soo Dat Ghy aen Hem die Landen God heeft U soo Xaveri! en ingestort soo groote Dat Ghy riept, Heer! // gheeft my niet Van waer komt my toch dese Est Satis, est Satis ,, ay my! // ick Goods Ick ben niet waerd // dat my Ick bidd' U, Heer! uw' vreughde Is 't saecke dat Ghy Xaveri! geweest soo op dees aerd' Wat sal dat zijn // een soete T' aenschouwen eeuwigh Goods Vernomen // bekomen // die kans // Goods Hebt Ghy nu Ghy slaet om-hoogh // op God U Wien ick altijd te dienen Maer aengesien ick Xaveri! een sondigh mensch, g'lijck ick In ziel gewondt // met sware Soo roept uw' hulp mijn droeve Gestadich // genadigh // U toont // my Die boven Dat is Xaveri! ,, bidt dat ick Wel-levende ontga het
