Weest verblijdt // t'allen 's Hemels Alleluya, Ick u opdraegh all' mijn Want ghy hebt // die 't all' Maeght! in u ged Alleluya // Godt in u had groot Desen Heer // heeft sijn Uyt het graf Alleluya // En is we'er op d'aerd' Onverwacht // uyt het Is uw' Soon Alleluya // G'lijck voorseyt hy had voor Weest verheught // weest Waerdtste Godes Alleluya // Opgestaen is ons Maget bidt // dat wy 't Van den Hemel Alleluya // En dat wy ons Godt
