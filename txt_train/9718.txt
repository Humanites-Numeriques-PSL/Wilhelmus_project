HEt is een schoon dierbaer Datmen make // onses Gods Groot met lofsanck ende Soet van smake // is hy Iae vriendelijck ende Laet ons zijn faem // verbreyden Looft hem te rechten // die zijn knechten // Met groot Looft den Schepper Die alle dinghen heeft ghemaeckt Alle eere // tot hem keere // nu ende t'aller Synen stoel is boven Hoogh verheven // wijdt Veel duysent Enghelen Die beneven // voor hem Sijn gramschap saen, doet bergen De diepten saen // verdrooghen Voor zijn aenschouwen // mans en vrouwen // Looft hem waert Meerder dan alle lof is Godt is over al, verr' en In hem leven // ende sweven // allegader Laet ons Godt een nieu liedt Groote dinghen // doet hy Voor hem beeft swerelts Oock springhen // die steenrotsen In stucken, voor zijn toornigh Maer inden noodt // is hy seer Die betrouwen // op hem bouwen // Hy helptse Wt lijden ende groot Hy gaet met haer door water ende Hy can mercken // al ons wercken, hy proeft hert' en Hy weet al 'sMenschen Al haer trachten // en Klaer als dagh zijn hem de Groot van machten // is zijn Hy hoort en siet wel alle dinghen, D'oore geplant // hebbend', als wijs Soud' al vooren // oock niet hooren // Soude, door D'ooge gemaeckt is, oock niet konnen Hy weet alle dinghen eer sy Na Schrifts getuygen // voor hem buygen moet alle Godt is een Coninck Ende desen // woont in een Daer niemant toekomt Sijn wesen // van Menschen Is verborghen, hy heeft Al het ghewicht // der Aerden Seer groot bevanghen // en gehangen // Opt water Met synen woorde krachtigh ende Ende hy onderhoudet oock Sijn vermogen // ongheloghen // blijckt wel Al zijn wercken groot Diemen gronden // niet en Van zijn heerlijckheyt T'allen stonden // eenen Doet den anderen daer af Maer noyt en sagh // yemant God vanden Maer zijn verkoren // eengheboren // Heeft ons seer Dat verkondight, lof prijs zy hem te Die op den stoel sit, weerdigh alle Eeuwigh ghepresen // moet hy wesen // en dat Lam Doe wy Gods vyanden Iae sondaren // boos en Ginck hy ons zijn liefde Sonder sparen // gaf hy Voor ons allen, in doots Sijnen seer wel // beminden Sone Die ons in noode // met, zijn roode // Verloste Siet, wat een liefd' ons God de Vader Dat hy ons als kinderen zijn Maect deelachtigh // hem eendrachtigh // eeuwigh lof zijn Sijn ghenade Is verschenen // over Tot profijte vanden Die recht henen // in dit In Christo wandelt den wegh Maer het en sal // nae Schrifts Den obstinaten // sondaer baten // Hem en Gheen ghenade, maer onghenade, Toorn, angst, druck, en swaer Wie te vooren // synen tooren // hier niet en Tot quijtschelt van't sondigh Is ghegheven // die Iesus den name Tot ons leven // ende Des vyants macht heeft hy Blijft niet verdooft // wie ooren heeft die Wilt ontfanghen // met verlangen // Reyn Schickt u te gaen door d'enghe Met liefd' en broederlijck Brengt den Name // Gods bequame, vrucht der lippen Vreest Godt al ghy Menschen Gheeft hem eere // want het Nu den tijdt zijns oordeels Nae zijn leere // altijdt Aenbidt hem, en niet en Die heeft ghemaeckt // Hemel en 'tAertsche Zee en Fonteynen // desen reynen // Zy Eeuwich lof, eere, prijs ende Ghy Conincklijck Priesterdom, hier Vry met vreuchden // al zijn deuchden // al wort ghy In Gods lof wilt niet Christen rancken // Wilt in alle dinghen Spijs en drancken // hy Gheen Musch en wort voor hem vergeten Sorgt noch en weent // niet, om dat vleesch Maer wel gade // Gods ghenade // Daer in Met vaster herten, blijft vroegh ende By den Heere, volght sijnen Om niet verdolen // hem bevolen // blijft Chrsitelijck
