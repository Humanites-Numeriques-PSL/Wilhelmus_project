De liefde blent // gaet my Met groot torment // Nu swaerlijck waer ic my went // Zijnde van u lief jent // Soo moet ick In verdriet // Door liefde hiet men my gaen te niet // En gansch dus suyver greyn // Acht u so t’aensien certeyn // Mijn liefde wilt mijn gheweyn // Doch helpen Want u licht claer // wesen Doet u dienaer // in liefden V sanck eenpaer, uytspraeck Brengt my beswaer // en doet my V bestier // brengt my hier, veel Dat my schier // rust gaet Dies my doch jont // Lief u So sal terstont // mijn hert Wesen ghesont // in vreuchden Goddin u keert // en u Hem die u eert // laet troost Liefde useert // quae raedt Nae u begheert // suldy vreucht Des nijders macht // nu Want haer cracht // alleen tracht’V te dies nu verlaet // Door mijnen dees valschen praet // T’gheschiet door en my bystaet, V druck sal Princes soet // hart lijf en In suer in soet // sal u standich Soo ghy wel doet // Lief met Dat ghy u spoet // my te Reyne vrucht // en geneucht, sal u Door myne deucht // altijt Hartseer en pijn // nijders Met valschen schijn // sal van u Ghy lief myn // troost gaet
