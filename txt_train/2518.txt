Myn sinlijckheyd // heb ick geleyd // op een die ick moet Die my onthoudt // 'twelck myn so rout // dat ick vaak wensch om Niet dat ick haar // mijn Lief eerbaar // yet quaets souw willen Ja woud' ick al // (dat ick niet sal) 'k en souw voorwaer niet Want d'oogjens bly // die sy tot my // eertijts so lieflijck Die heeft sy noch // waer door ick, och! haer hertelijck En haer wel-vaert // is lief en waert // in mijn getrouwe Souw 't hart van mijn // wel konnen zijn // sonder haer te Neen't, seker neent // het sucht en weent // als het mijn Lief siet En in haer vreucht // schep ick geneucht // en over groot Want sy besint // die haer bemint // soo't onberoemt mach Geen last of noot // sal voor de doot // my van mijn Lief doen Hoe soud' ick // siet, beminne niet // die so men mach Is wijs van Raad // arrem van praat // en rijck van goede Hier door 't oogh viel // op d'ed'le Ziel // van mijn Lief In wien ick, ach! docht dat ick sach // het alder-schoonste Kleyn is mijn roem // of ick schoon noem // haer wel-geschickte Want d'Ouders die // gaen so ick sie // haer aen Rijcker En die haer eerst // scheen wel begeerst // gaat vluchtigh sy Maer die aen-neemt // voor eygen, vreemt // laet sich door 't geld Het duyvels Geld // heeft oock geweld // sie ick in 's menschen Want die quaat doet // kan metter spoet // de Liefde licht Geen Liefd maer schijn // want siet an mijn // of het my Kan Daermen Jonst koopt // hoe dat afloopt // sal ons de Tijd wel
