Van achtentachtich, Het nieuwe dat is warachtich, Heel Diet can geloouen // Het sal Dat blinde douen // Hooren noch En tisser geseyt // Dat menige Sonder bescheyt // Sal na een vryer Dus weest vry vrolijc, Ghi knaepkens Oft bent ghy een molic // Tis euen Men sal v vryen // Om tgroote En oock somtijen, Om hooge Dus hout v wat fris, Tgeluc Waer datse doch is, Diet lach betalen Wy consenteeren // De meysgens Dat zijt met eeren, Gaet vragen En mogen vryen // Wie dat haer Wil hijt niet lijen // Dat sy hem So blijft hy in schant, Die onachtbaer En crijcht int lanc // Geen lief die by hem Princesse schoone // Tis nu v schenct hem die crone, Daert hert om Het moet doch wesen // Datmen Dus wilt niet vresen // Wie eerst Die liefste en eer // verachtmen niet Maer maectse weer // An beyde sien
