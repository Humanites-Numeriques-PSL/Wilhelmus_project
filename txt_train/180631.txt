REyn van manieren // is den Echten Die hem hantieren // in den rechten T’is slechten raet // die hem verbieden, Beter is trouwen // dan te lijden Want die hoereren // Godt condemneren // Des wy lauderen // T’huwelijcx T’is waer dat vrienden // in malcanders Dickwils wel dienden // als bystanders Maer anders groot // met een affecty Staen man en wijf ter noot // malcander Wie een goe Vrouwe // Die Godt ghetrouwe // Bevrijdt van rouwe // Te langher Een wijf die deuchdich, is van wesen s’Mans herte vreuchdich // sy genesen Ghepresen moet sy zijn om haer Een edel gave Gods is sulck een Tot troost van haren // Die wel bewaren // Haer tongh en sparen // Weder Soo lief bequame // op der eerden Als een eersame // Wijf vol weerden Met sweerden vliet // een Roover in der En brenght den roof haer die boven Hy voor vriendinne // Veel zijn door minne // Gheworden En Een vroom Wijf schoone // is als brande De lamp ydoone // op’t heylich Zijnde bestande haer // eerbaer Een gulden Pijlaer op een silveren Is sy gheleken // Om niet te breken // Oft te ontsteken // Door boosen Die een Wijf zedich // voor sijn proven Huylich, cuysch, zedich // machse loven Al boven gout // en boven edel Haer cieraet is wat neerstich is en Al moets’ hier lijden // Haer vreucht in tijden // Met een soo blijden // In s’Hemels t’Wijf sal door baren // ende kinder Ten Hemel varen // datse ginder Soo in der cracht // ghleoove, liefde Heylicheyt, eerbaerheyt, haer blijft Voor des versijcke // Werdt sy in Gods rijcke // d’Enghels ghelijcke // Wert Prince d’oprechte // soo sy dede Sal men in Echte // niet besteden Te vreden seer // wel eeuwich sonder Sullen sy in s’Lams bruyloft feeste Soo vrolijck maken // Elck mach wel haken // En neerstich waken // Om daer te
