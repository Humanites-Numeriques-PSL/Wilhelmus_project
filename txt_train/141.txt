Ick weet een Peerle Gheprent // in Schoonheyt Noyt Man en heeft Bekent // haerder Diamant ofte Sangwijn verciert in En vintmen tot ghenen Soo fijn // In gheen Ghelijck die Sonne Eenpaer, Int tshemels So is mijn Lieff Aldaer blinckende Waer ick een Cooninck Playsandt // Rijck van Ick versette voor haer mijn En sant // En oock mijn Dat ick mijn Dach en nacht, op haer set als Zy is my te hooghe van Gheacht // Ende oock te groot van Dees peerle daer my dat Zeer wreet // om moet Tis mijn lief dat ick Hoe zy heet // die fluer der Mocht ick met mijn shersen Onsoet // haer noch te Zo sout ick tot een Mijn bloet // wt laten Sturffe ick dan van rouwe Den dood // om haer duer Zo ware mijn wondekens Ter noot // Gheheel Ick gheue elck minnaer Beuel dat hy liefde Maer dat hy der liefden Zeer fel, Niet en Die hem spieghelt Aen my // die spieghelt hem Elck minnaer verre ofte Wie tzy, slaet hier wel op Prince mindy een edel Int pleyn // Als ick bouen Zo en muechdy der liefden Certeyn, met recht
