HEer uwen name // is groot Wijdt uytghebreyt // is uwe V Majesteyt // vol Heer u ghenade // duert vroegh en Sonder verganck // eeuwigh Even soo lanck // zy u lof Heere uwe weghen // zijn al te Altijdt tot goet // zijt ghy Lof u zijn moet // in Heer uwe wercken // die doen ons Dat ghy altijt // niet om Almachtigh zijt // O Heer u wesen // moet zijn V goetheyt heeft // ons al O Heer ghy gheeft // daert al by O Heer u woorden // ons met Wijsen hoe dat // wy al Te gaen den padt // na u schoon V hulp, o Heere // hier t'allen Behoeft ons wel // want wy zijn Dus maeckt u snel // ins ons Heer, uwen wille // vry van Leert ons al doen // van zonden Als rancken groen // op elck O Heer, u Rijcke // daer toe Ons weerdigh maect // dat niemant Verloren naeckt // maer neerstigh
