OP op mijn geest // en wacker Om mijn ghedacht te Al wat men eert // ter Werelt V bet mijn Lief te Want alles swicht // het voor het Dat ick moet eeuwich Voor't eenich goet // dat my Mijn al te kiessche Het grof geslacht // des vollicx Van andre waerdt te O domme tijt // die my Ghy hebt noch breyn, noch De gheen die ghy // hooch in Verheft, en zijn gheen Of zijn't, ick kan // de myne Voor een Mannin niet Die ghylie viert // zijn wel Met schoon en Maer by den dach // en (dunctme) De claerste nacht niet De myne tooch // (want jong zy Heltinne moeders De wyse deucht // die't al wel In't merrech van haer Wie boven al // op d'aerdsche Doorluchtich is Wies deur gevaer // en commer, Veel' in versmachten, Geen menschen handt // mijn dierste Wt grousame' Maer loutre wacht // van hoger Des Hemels, heeft Den Hemel droech // wel laet en Met reden, sorgh Voor't schoonste waert // daer hy In had, soo veel Van yder daet // die hy 'Tmisvoeghen is Zy singt of praet // zy lacht zy Een strickjen staet // Een strickjen daer // verwarren In, al die kennis O wonder hooch! // O werelds O glans van onse De clare Son // niet op gaen Soo ghy u aenschijn Alleen want hy // verrijst, voor Mijn lief, mijn licht, mijn voor u de May // nu maxels Verziert, van kruyt en Maer als ghy scheyt // uyt Sal't aen u staen, te Wat, dat ghy wilt // of Pallas Of Iunoos scepter Op dat u wijck // de Venus uyt haer
