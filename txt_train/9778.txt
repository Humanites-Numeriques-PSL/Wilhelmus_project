NEemt waer, o Heer, ws Kercken // Laet Sathan haer niet siften // Die op u Woordt en Schriften // Door uwen Gheest wilt wercken // Tot stercken // Hoewel ick met de Slang oudt, en wil paeys maken // Sy toont, om my t'ontschaken // Een vluchtigh moy aertsch Doch ydel // En laet my niet om vallen // Noch breeden padt verkiesen // Ick laes u sou verliesen // Tis beter op den smallen // Hier t'allen // Acht Babel niet, o ghy // Haer sy u Koninginne // En my u vyandinne // Maer nemmermeer sal my // Op dy / te Want siet om niet bedroghen // Mijn kracht zijn 't Euangely // Ghemeen ghy niet met En zijt, by u gheen En moghen // O Heer, al ben ick nu Ghequelt oft moet nu seer zijn // Ick hop', ick werd', o Heer // mijn Eens int volkomen suyver Van u // 'tQuaet eerst zult ghy den quaden // Die qualijck doen oft spreken // V gramschap werdt ontsteken // Maer wilt my met ghenaden // Weldaden // Och laet doch by t'waerachtich // Vwer Euangelisten // Volherdich al van twisten // O Heer, door uws Gheests krachtich // Eendrachtich //
