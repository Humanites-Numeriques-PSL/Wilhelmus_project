SChoon duyvinne // die reyn van My meer bemint dan uws selfs Want ghy inne // 't vyer mijner Verslonden zijt, van hier Door mijnen gheest // uyt 'tAerdtsch Tot by my int Hemelsche In mijn slaepkamer Rust u hier dan // tot dat De hitte van // den middagh Sonder scheyden // wilt u Met my, en vast vereenight Gheen verleyden // hoe sy V deeren mach, ghy sult En vreest niet want // elcken Sal ick met krachte neder En legghen tot ons Dan wert altoost // ghewisselt Droefheyt in troost, haer vreucht in Openbaren // met mijn Sal ick eer lanck, tot u Den sondaren // tot een Die mijn aenschijn sullen Vluchten beducht // sonder Werden benout van herten Och siet, hoe lief en weerdt Is sy, die wy // hielden voor Sy is nu bly // wy droef O Princiere // blijft Lijdtsamigh tot behoudt der Met den viere // uwe Vervolghers rijck sal ick Den nieuwen Wijn // int rijcke Sal u herte vreuchdigh En uwen mondt vol lachens By my u steed' // eeuwigh zijn Troost u hier meed, int aertsche
