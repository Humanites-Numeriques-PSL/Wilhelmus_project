ROemt u des Ghy die de waerheyt // Godt in Looft, ghelijck in Van u Bewijst danckbaerheyt // Die u laet En kennen Synen wille seer Doch uyt rechter Wilt mercken, vrienden // Wat een ghenade Dat ons de Heere Die met ons wercken noyt en verdienden // So wy wat Van Gods volkomen // Die schoon Wt ghenaden Wat wil Oft hem beroemen // T'welck na Woordt, is gras ende Indien wy dencken Meer te wesen als Ny worden bedroghen // Soo wy om ons Op onse wijsheydt Verdiensten, wercken ofte vermogen // Laet ons Godt Oock yeghelijcken Sachtmoedt En lasteren Op dat 't Des gheests recht blijcken // Met laven, En weldoen den Door een liefde die Christum nae volghen, Wy waren t'alder // Oock verblendt int Vol van boosheydt Draghend' int herte op malkander // Maer noch Wy Gods ghenade Als wy Ons ghewercken Dus sonder Laet ons ghestade Vry van Neersticheydt doen In als te nemen Ons beroepinghe Tot voldoeninghe // Soo is dan Christus Voor die hem volghen Wilt hy seer milt, tot een versoeninge //
