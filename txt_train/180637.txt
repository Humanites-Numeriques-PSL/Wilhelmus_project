DE Mensche is te wonder // Soo langhe als hy sonder // Met vleesch en bloet te rade // Want hy tot alle quade // Daer lust ende gheneghen // Soo dat hy in veel weghen // Alsoo wort hy te samen // Den soeten aenghenamen // Van Godt een uytghelesen // In welcken salich wesen // Hy die met recht bekeeren // Openhertich des Heeren // Het woort sijn ziele maken // Salich, maer eerst versaken // Hy sijn eyghen qua leven // Aennemen en aencleven // T’woort met een reyn sachtmoedich // Daer sijn vrucht overvloedich // Wie in des Heeren reden // Sterck als van t’lichaems leden // De kennisse der waerheyt // Ghecrijght, en wort in claerheyt // Van t’duyster jock der zonden // Hy onder lach ghebonden // Maer die gaen buyten desen // Sullen soecken en lesen // Nochtans de waerheyt vinden // Want soo met alle winden // Daer waeyt dees onghestadich // Al hebben sy weldadich // Maer om vlieden t’verleyden // Soo dientmen t’onderscheyden // Die Bruydt uyt alle Secten // Die doors’Heeren perfecten // Noch wandelt onghevloden // In alle Gods gheboden // Reyn liefde tot den naesten // Die tot onderwijs haesten // En oock tot beteringhe // Op t’woort al sonderlinghe // Dit velt-teecken ydoone // Die onbevleckte schoone // Dese staet al op vaster // Dan die met haren laster // Den sone van haer moeder // En spreeckt van sijnen broeder // Maer die vernieude jonghe // Honich vloeyt haer ter tonghe // Sy heeft in desen heunich // T’is t’woordt Gods daers’ op steunich // En oock om op te wassen // Hier op behoeft te passen // Dat hy niet en verstroye // Come ter rechter kope // Welck huys Christus loyalich // En sonderlinghe salich // Als hy t’oncruyt verloren // Te zijn terwe vercoren // Daer gheeft ons Godt ghenade // Die ons van allen quade //
