TObias tot sterven Riep synen soon, en sprack daer Mijn kindt // Hoort met accoorden // alle mijn Als God mijn ziel heeft Mijn lichaem in een graf laet En wilt // oock V Moeder draghen // jonst al u Houdt haer in en weerden end' in Denckt opt perijckel en Hoe sy // droegh, Met groote smerte // onder haer End' als sy sal ghestorven Begraeftse nevens my, by Altijt // met En neerstigh poogen // hebt God voor Draeght in u herte Godt Soo dat ghy hem altijdt Zijn meught // van Tot 'slevens enden // t'hemwaerts u Siet dat ghy niet t'eenighen V en bewillight in gheen T'ghebodt // van En wilt versteken // noch gheensins Van uwen goede helpt den Aensiet hen-lieden met By dien // Sal u ghestade // Godt met Helpt neerstigh die behoeftigh Hebt ghy vele // wilt vele En doet // oock Met herten reynigh // al hebt ghy Gheeft trouwelijck al sonder Soo sult ghy rechten loon Ter noodt // die En zonden lossen // de De Aelmossen vol Is in der noot een groote En troost // Seer groot beseven // voor God Mijne sone wilt u oock Van hoerery, laet die Hebt een // Wijf Wilt daer beneven // gheen meer De hooveerdy en gheeft gheen In u herte, noch woorden Van in // Wracht sy 'tbederven // daer wy om End' alle die om u Terstont wilt haren loon En houdt // niet Van gheen persoonen // verdiende En doet oock geen menschen Dat ghy nood' hadt van ander Ter noodt // u Deelt t'allen tijden // die hongher Wilt om den naeckten te Van uwe kleederen Maer meest // Lijck zijn begraven // deelt uwe Met den Sondaeren vol Soo en wilt oock drincken noch Maer gaet // om Den Wijsen vraghen // tot allen Danct altijdt God, en bidt Dat hy u wil regieren Dat fijn // al Voornemen blijcke // zijn woordt En sorght niet, mijn Sone Al leyden wy een arem Voorwaer // hier Wy hebben sullen // van als te Indien dat wy tot allen God vreesen en de zonde En goet // doen Nae recht wy souden // dit wel
