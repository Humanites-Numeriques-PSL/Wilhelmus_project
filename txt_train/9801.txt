O Heer wanneer ick met De groote kracht, der liefden Die ghy aent vervallen Van Adam hebt ghestelt te En hoe ghy noch daer onder // Groote ghenade wonder // Ick uwen Naem loven bysonder // Wat wonderlijck verkleenen // Hebt ghy oprecht, ghekroonste // Bewesen dat ghy eenen // Quaemt zijn van uyt u schoonste En naemt op u der zonden // Van my arem ghewonden // In lief' aen u ben ick verbonden // Hoe laet ghy slechte weten // Dat wyse duysenden Is, al die schoon secreten // Ws wils, hoe lust u soo te Voor kleen verworpen scharen // Dat ghy haer openbaren // Het welc ghy voor den seer ervaren // Hoe spraeckt ghy al int duyster // De Werelt in En niet met eenen luyster // Moest yemant van u Rijcke // De kennis? om soo hooghen // Behoortmen wel te pooghen // Men vint, al sou men daerom doogen // Soo wie begint, aen u En boete snel, doet In u kennis en leering' Sal hy worden allencx Maer sonder dat, hy even // Blijft in het sondigh leven // Waer in men niet en kan aenkleven // Een segghen int aenhooren // Als of den Vader teghen // Hy heeren heeft te vooren // Dat boose gaen haer weghen // Doch den zaeyer van't quade // Sijn selven maer te rade // Geen bouman goet na eygen schade // Den Huysvader voorwistet // Maer voorschickte noyt quaet oft Dijn zaedt willens ghy quistet // Al vallet weynigh in goe Waerom? om dat tot desen // Niet doen 't gheloove wesen // Ghelijck ghy wilt, om haer ghenesen // Ghy sijt een Medecijn, en Roept ghy en biedt ghesonde // Levende Water, Wyyn // en Maer der sonden ghewonde // Weygheren laes te kommen // Haestende na 't verdommen // In onschult sy moeten verstommen // Siet, mist men u rechtveerdich, Opt voorbeschicken Gods te // Gheen onschult men dan weerdigh // Van onbereyt zijn, oft niet te // Als men handen en voeten // Dus nu men tijdt ter boeten Al willens wy niet dwalen moeten // Een plaetse van begin // Isser, ja vant begin der // Daer werden sy al in // Die met deuchden werden Maer d'ander werdt ghebleken // Int eeuwigh fel ontsteken // Och men schromt daer van te spreken // Gods voorschicking', en willen // In eenderley ten mach niet Maer waer in soudt verschillen // Hy wil altijdt, sonder Dat alle Menschen souden // Inder waerheyt behouden // Och of sy al daer toe doen wouden //
