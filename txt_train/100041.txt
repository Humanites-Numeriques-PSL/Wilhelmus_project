HOe lang is 't heden // schoone Doch wel geleden // dat gy hebt Op roode tipjes // duizentmaal Van zoete lipjes // onze 't Was toen u woorden // en geveinsde Mijn hart bekoorden // en tot kussen 't Was toen de vlamme // van u geile Mijn tegen quam, en, in mijn armen Wat hebt gy strikken // listig Om te verstikken // mijn Hoe kost gy vleiden // midden in die Die strak na 't scheien // by een ander Neen Iuliane // denk vry dat mijn Haar eerste tranen // offert mijn Geen Liefde, sweer ik // dat van Zal 't hart van Fredrik // in haar vlamen Ik acht geen Eeden // die een Minnaar Zo vaak vertreden // met een losse Zo lang de dagen // zijn van Zal haar u klagen // tegen 't harte
