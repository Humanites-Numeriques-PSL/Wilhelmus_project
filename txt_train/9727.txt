IN kromme weghen, duyster en Vol doode vruchten, en wercken Daer namen wy ons ydel Maer uyt ghenade werden wy Eenen soeten reucke, stem, ende licht Wt 'sHeeren Hof ende Dies wy gheringhe // merckten Dat wy de baen // misten int Maer wy vonden daer staen // van desen // De deur ontdaen // end' hem moet wesen // Die ons daer heeft so vriendelijck Wy vonden daer in des Hofs Van alderhande welrieckende Maer boven al // hooghe Een roos int dal // die door haer soet Syon des Heeren uytvercoren Geeft troost en vreucht, welvaert en En vast beneven // der beecken Wierden gheplant // seer Groen spruytkens playsant // die als leden, Opwiessen inden bandt // van vreden // In deucht, en jeucht, seer schoon en Menigh was vroylijck hier ter Soo lang als in dit Cruyde-hofs Den Zuyden windt waeyde seer Maer werden door den Noorden windt Want daer een bitter wortel is Door eenen gheest // vol van En vol moedtwille // ginghen daer Vele verstoort // buyten de Want desen windt van Noordt // doet verkonden // Alle die aent woort // niet en houden // Haer laten omwaeyen met fel Princelijcke Bruydegom, tot Sullen wy ons keeren, oft henen Dan by u in u Hof ons wooningh' Ghemerckt dat wy soo klaerlijcken Die by u niet volherden in Waer sy int eynde O Heer ons waken // helpt met In allen strijdt // zijt ons Dat niemant ons in zijd-wegen dwaligh // Maer wat profijt // is t'onser Geeft ons, want een is noodigh
