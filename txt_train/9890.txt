gheest Al wat niet gheestigh // en smaeckt Dat moet tempeestigh // als Opt zant verdwijnen, neven den Doncker bruwijnen // en moghen De waerheydt perfeckt // Houden Oft schoon Sy verwint int 'sHeeren Broeders eendrachtich // elck in sijn Laet ons aendachtigh, na Petrus Onder den Heyden // ghelijckt Goe wandel leyden // dat elck hem Tot beterschap // die met Door Ons hebben sal 'tRecht vromer leven qua tijden // voorhanden Laet ons vermeyden // in kleen oft Dat met besinnen // selfs niemant Buyten oft binnen // van ons Dat soete lucht // Van goet Door goede Van ons zy 'tZy vremd' oft vrienden // wie dat ons Soo wel wy dienden // te zijn Want, siet, den name // den welckens Aenroepen tsame // desen wort Voor borge betrout // En wie hier op Hoe sal doch Desen te kort doen Wie en sal pooghen // om laten Eenen soo hooghen // borgh int Of soo in schaden // datmen Daer door versmaden // synen Naem En 'tvolck vermaert // Van syner Iae recht als Bedrieghelijck in zijn Die dus laet steken // zijn borgh in Godtloose treken // doet hy, tis Oock staet gheschreven // hoe dan in T'slecht overbleven // Israels Gheen leughenen doch // Iae tonghe Doende Ghevonden zijn en Dus moet den Tempel // Gods staen Christi exempel // is klaer en Te gheenen stonden // was oyt Bedrogh ghevonden // in zijn mondt Wie valscheyt bedrijft // Oft leugen Niet by hem Maer uyt zijn huys hy Niemant bedrieghen // ghy niet en Sprack die van liegen // noyt hadde Siet, wilt aenschouwen // een Oprecht vol trouwen // daer gheener T'zy meer oft min // Bedrogh en is Des Heeren Die blijckt hier breedt en Och gaet niet qualijc // als vrient Met vrient verralijck // ist oock niet Dat Broeders nouwe // en souden Moghen betrouwe // den Broeders Twelck haestich Soo't straffen ophiel // 'Twaer tot Van 'sHeeren volck en 'tWaer een oncieren // der leer Soo qua manieren // te lyden Daer soud uyt groeyen // een quaet Elck soud' verfoeyen // sulck volck eer Onstichtich het waer // Ia duysterheyt Voor licht, oock Voor reucke vuylen Als wy wel schaffen // op waerheyts Sulck quaet te straffen // is meerder Na 'svromer wenschen // en 'sHeeren Voor Godt en menschen // t'antwoorden Dan datmen becroont // Oft yet Want
