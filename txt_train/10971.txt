Als liefde blent // myn ghemoedt hadt Heeft my ghewent // rontsom nu hier en End’ met rotment // my ghesocht te Maer reden jent // heeft my gherecht Tot dat sy my bracht by u licht seer Maer soo haest uwen ooghen // My int hert dede dooghen // Wast niet in tvermoghen // Te weerstaen zijn cracht Cupidoos licht // meynt hem aen my te Van tvier ghesticht // in myn amoureus Maer tis tghesticht // van u Lief Twelck my verplicht // aen u nu houdt met Dies u die eer alleen gegeven Wech liefd’ met u ontspresen Ghy condt my niet doen vresen // Maer myn liefkens wesen Was in die const Een ghestadighen strijdt // sedert ick u Bevingh subijt // myn gemoet seer Liefde somtijt // doet my wat vreuchts Maer wanhoops krijt // ghy zyt te wreedt van deen in myn vreucht, en dander lijden Soo dat ick int verdriet // Int eynde gantsch te niet // sal Een rijck datmen siet In tween, is swack van Teghens myn danck, seer wreet ghy u gaet moet ick met dwanck, u minnen dach en Met lyden strack // gaet ghy myn Lief Als ghy verlanck // soeckt in tgheen dat ick Dies Goddin Lief, myn liefde nu V fierheyt gantsch besyden // Aensiet hoe my het lyden // En dat myn verblyden // Als ghy my Lief
