IN mijnen noot // roep ick tot u, o Heere,Laet my niet bloot // want ick ben zwak en tereWant ick doch niet // sonder u en vermagh,In dit verdriet // troost my, Heer nagt en dagh.Troost my, o Heer // en wilt my niet verlatenIn dit verseer // komt my Heere te baten,Op u alleen // betrouw ik Heere mijn,Wie mach ons scheen // van uwe liefde fijn?U liefde soet// bewijst Heere geduirigh,Wat men ons doet // dat wy niet wesen truirigh,Door uwen Geest // Heer, Daniel by staet,In Babel meest // al by de Leeuwen quaet.Die Leeuwen fel // zijn oock in de WoestijneEn soecken snel // ons te brengen in pijne,En slangen fier // spouwen vyerigh fenijn,O Godt zijt hier // ons' rechte Medicijn.Geen kruyt so soet // dat men sou mogen lesenNoch plaester goet // en kan ons niet genesen,Maer Heere gy // die Israel by-stondt,Door Christum vry // maeckt ons zielen gesont.Ons zielen kranck // zijn altijt in dangiere,Door Peste stranck // en sieckt' meniger tiere,So dat vergaet // menigh met jammer groot,Die haer geen raet, soecken tegen den doodt.Sy minnen meer // dat tijdelijcke leven,En 's werelts eer // van vleys en bloet gedreven,Sy zijn eylaes // seer jammerlijck verblendt,Als Maegden dwaes // door haer traegheyt geschent,Sy zijn by nacht // oock slaperigh bedegen,Als die geen acht // op't licht hebben gedregen;Als haer den Heer // sal opwecken subijt,Sullen sy seer // kloppen, maer ter ontijt.Dan sullen sy, beschaemt, daer buyten blijven,Dat roepen vry // en sal haer niet beklijven,'t Werdt dan te spa // om na Oly te sien,Want geen gena // sal haer den Heer verleen.Vrienden gemeen // wilt bidden ende wakenOmgordt u le'en // laet u licht vyrigh blaken,U Zedigheyt // laet wesen openbaer;Weest soo bereydt // als wijse Maegden klaer.Als reyne Bruydt // wilt u also vercieren,Toont overluydt // u licht in all' manieren;'t Is nu de uir // van den slaep op te staen,Door Christum puir // tot den Vader te gaen.Princelijcke Godt // in eeuwigheyt verheven;Dit voor een slot // mijn vriendekens gegeven;Die van't getal, zijn weer geboren nieuw,'K schenck u dit al // voor een vriend'lijck Adieu.
