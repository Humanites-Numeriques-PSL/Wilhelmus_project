LAet ons offeren Den Heere met Dat hy ons schanck, spijs ende Natuere kranck, Tot haer En tot der Zielen Sijn woort een spijse En van alle goet Lof, prijs zy sijnen Sijn fame // laet ons hooghe Ghehoorsaemheyt En brenghen Voort alle vruchten goet // Der liefden Want siet, men moet // Op sulcken Den Vader Anders spreeckt hy Hoe neemt ghy mijn In uwen mondt // En ghy en Niet inden grondt // Lijden // En ghy werpt mijn woorden Achter u, ja oft Dus int gheloove Laet ons den Heere Verblijden // wilt al zijn En zijnen lof Sijn goetheyt Nemmermeer en Al waer zijn zaedt // Int herte Dese van't quaet // Sal hy Behoudt in u Dit zaedt, o Broeders, Alst is gheplant // In een goet Cant abundant // Veel vruchten Want den wasdom gheeft Gods In herten die zijn Soo vele als dat Gheen cause en heeft om Die vragen // vlijtigh na zijn Haer sal hy wel Waer men op zijn woort Daer gheeft hy dat het Maer hy roep vast // en noodt te Al die belast // Zijn, en Prince Gods woordt Is een beschutsel Teghen aenstoot // Van grooten Daer Amos bloot // af gaet Den hongher, maer niet na Die veel menschen sal Om ontvlien de tweede T'versoenigh woordt sal haer Sy dwalen // die haer houden Dit woordt, groot van Staend' als een open Dus komt met goedt Ghy al die hoort // En seght komt De deur van't woort // sal haestigh
