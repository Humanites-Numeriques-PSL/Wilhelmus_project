ELck medegenoot // int geloove In veel droef aenstoot // met verduldicheyt Stelt u vreese Al doen haer exploot // Werelt, Hel ende Hoe menigen kloot // dat elc oyt op u Victory sult ghy Door Christum Wilt u Want hy uyt Is t'allen Met zijn volck Als Om haer Wie oyt strijt aennam // in den name van Oyt hy te boven klam // exempel Hy verwan die David Goliam // den Reuse fel en Die in den name quam // der Philistijnsche En d'Israelijten Met trompetten Iericho De mueren vielen Veel wonder Hebben Al die vast Den Heere Petrus die eerst maer // voor een maeght hadde Dat hy tegen haer // niet dorst belijden Die sietmen nu voort Voor een groote schaer // door Gods Geest sterck als een pylaer // en heeft den Heere Voor elck end' een Sijn antwoorde Diets verbodt Gaf dus een Oordeelt, ist Dat selve Door u Sal wijcken Dat 'sHeeren Geest // sterc maeckt en Stephanus beweest // als groot vervolgh Stormigh op quam Iae Paulus die eest // vervolgde, heeft noch Ghedaen int aertsch foreest // menige wilde Weder vochten Iae met goedt In al zijn Op Christum En sprack, door Ick 't al In sijnen Werd' ick Van mijns Princen hant // werdt my int eeuwich De Croon geschoncken, want // hy heeft tot Ons synen Gheest Dus Broeders vailliant // en laet doch u Wt sin en verstant // rooven dat God Heeft, en daer in Door zijnen gheest Maer sonder V beste Om vast Hebt goeden Ghy krijght den Van 't eeuwich
