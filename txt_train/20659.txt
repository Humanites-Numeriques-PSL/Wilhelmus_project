ALder-soetste Jesu Mijnen Heer! mijn Heyl, mijn Groot geluck // wensch ik U in den Die Ghy lijdt // om mijn groote sonden op dees U jonge bloed // trouw vol-doet// voor de Van mijn zond' // waer mee ik U heb O! mijn Heer // ick begeer // Te-gen U // op te staen, dit beloof ick nu. U 'k Wensch U een geluckigh JESU Lief! ick wensch dat Uwe eer // in dit Jaer // meer en Met de mond // wijt verbreyden mogen stond op 'k Wensch dat Ghy gaf // van nu af // dat niet Maer met vlijt // ick U dienden t'allen 'k Hef mijn oogh // na om-hoogh // en ick Van dees uur // U te dienen in soet en Ick sal nu beginnen JESU Lief! 'k sal my van zond' Ick wil nu // staende-voet dienen Metter daet // ick nu mijne zonden gantsch Ach! ick beschrey // het gevley // en 't Van mijn vleysch // 'k volgh voortaen des redens Ick besteen // ick beween // all' het 't Welck Ghy haet // die gheen quaet ongestraft en Ick aenbidd' U, kleyne Ick weet niet, wat dat uw' liefd' Ghy komt teer // in een stal, hier ter Op dees aerd' // die nochtans des Hemels Throon zijt Op 't aerdtsche dal // in een stal // voor ons Komt Ghy Heer// arm en naeckt, en sonder Ghy ontsiet // geen verdriet // Ghy 't Jonge bloed // Ghy dit all' om mijn' zonden Wat sal ik U geven JESU Lief! voor dese groote 'k Sal mijn le'en // van den hooft, tot En mijn min // U opdragen met mijn hert en 'k Schenck met ootmoed // U mijn goed // en mijn Om dat Ghy // dit geschoncken hebt aen Heer! ick stijf // ziel en lijf // tot Van uw' lof // die my schencken sult 's Hemels
