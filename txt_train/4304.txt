SVchten, treuren, is nu mijn Als menich edel ieught Een ionge maeght hout my S'gelijcx en stont getytelt By al die oyt // veel wonder // Gelijck de snee op t'velt Ligh ick nu onder- // Gans sonder // V ermaledijt sy dach // en // Die my brocht dit verstrangen // Want eer ick wel deursach // den // Was ick een arm gevangen // T'verlangen // van // een troostbaer Die ick geensins ontfangen // Heeft my t'herte Ten duysent Rechtveerdich is mijn Dat ick om haer sulcx lijden Ick en in alles Wat smert, en leet sy my Om smaken t'soet, Geniet ick Want naer veel drucx en Sal vreught staen voor By Tooght sy my Die niet dan deught haer voren // En pleeght wreede Aen my, gelijck gesworen // Geboren // seght // my elck te Een scu, een arm verloren // Soo my falgeert in Troostbaer Eylaes Venus my doch T'benouwen deser banden // Ick minne een dat soetste Dat my mijns lijfs ontpanden // Wilt vanden // val // my Ghy hebt het doch in handen // Slaet doot, of wilt Mijn Peynst wat verdriet, en smerten // My int begin, was En met hoedanich herten- // Ick nu moet nemen mijn Is my ontseyt // troost te Soo wert de doot lijdsaem Beter ist, eens Dan duystmael
