IN Esaie // prophecyeInt tweede staet gheschrevenHoe dattet zy // ten laetsten tyAls dan sal hooch zijn verheven,Den berch ghewis// op welcker isMijn groot ghedruys // des Heeren huys,Als datter veel hoopen // hene loopen,Met vlijt // verblijtElck ander dus toe  coopen.Al afghelaen // comt laet ons gaen,Tot desen berch ghepresen,Die vast sal staen // nimmermeer vergaen,Maer eeuwich t'hoochste sal wesenIn welck die Godt // van Iacobs rodt,Nu hout present // sijn schoone tent,Dat hy ons leer te deghen // sijne weghen,Sijn paen // te gaen,En in gheen vreemde steghen.Niemant aenhoort // noch gheenen oortDan Syon uytghelesen,Heeft Godes woort // met goet accoortIerusalem ghepresen,Voert Godes wet // die onbesmet,Oordeelen sal // die Heydens al,In oprechtigher claerheyt // na der waerheyt,Sijn recht // bevecht,Die volcken sonder swaerheyt.In sulcken ganck // sal elck te danckSijn swaert tot ploechysers breken,Die spietsen lanck // tot sickelen blanck,En dencken niet meer te wrekenVan dese gheen // ja oock met eenDie daer begheert // eenighe sweertTeghen yemant te luycken // en ghebruycken,Gheen striedt // gheschiedtVan haer noch sulcke stuycken.Also dees reet // van den PropheetLuydt gatet in onse tijden,Want veel is leet // haer overtreet,Waer over haer nu verblijdenHier niet alleen // Christus ghemeen,Maer oock seer claer // d'enghelen schaer,Want so staet hier beneven // claer geschrevenVan een // onreenDie hem tot boet wil gheven.Comt doch eens naer // aenschout die schaerDie nu na t'beste haecken,Die voormaels waer // s'werelts dienaerBetracht nu andere saecken,Eertijts was sy // eyghen nu vry,Gheneycht tot quaet // nu tot weldaedt,Gierich oncuys sy saten // boven  maten,Met veel // sulck speel,T'welck sy nu al verlaten.Oock siet hier meer // hoewel noch teerDie mee beginnen te comen,Al haer gheweer // legghen ter neer,Aensien die menschen als boomen,Wijn sonder alst // melck onghevalst,Begheeren sy // als kinderen ny,Op dat al jonge rancken // door die dranckenAemacht // tot cracht,Sy comen en God dancken.Peys en vree // is in dees stee,Met God voor sijn ghenade,Dus wendt u tree // ghy alle nu mee,Die vreest der zielen schade,Nu isset vry // comt alle mee by,Het is nu claer // t'recht gulden jaerna der Basuynen clangen // wilt verlangen,Dees dach // vermachlElck een sijn erf te langhen.Sal nu dees teyt // der salicheytDees aenghename daghen,Met goet bescheyt // worden verbreyt,En dat na Gods behaghen,Sulcx niet allien // hoort te gheschien,Met groot begheer // der reyner leer,Maer wy moeten niet beven // ooc ons levenDaer naer // gantsch claerMet neersticheyt begheven.Op dat wy geen // der swacker leen,Tot erghernisse strecken,Want vele ween // staen over deen,Die zijn des lichaems vlecken,Haer straf met spoet // haers naesten bloet,Och t'is seer swaer // comt al op haer,Als vruchteloose boomen // en onvromen,Met vliet // toesietOm sulckes te ontcomen.En wilt doch niet // den blinden yet,Tot eenen aenstoot setten,V leven biet // sonder verdriet,Dat elck daer op mach letten,Duldich in druyck // noot angst misbruyck,In slaghen daer // ghevanckenis swaer,In oproer arbeyt vaecken // oock in waecken,Het zy // of ghyOm u gherucht oock raecken.Nimmermeer laet // aen u yet quaet,Met waerheyt zijn te wreecken,Corinthen staet // ick houde maet,Op dat ick andere te preecken,Niet self en ben // verwerpelijck slem,Dus laet oock wy // waecken bety,Dat wy sonder beswaren // recht verclarenOns baen // wilt gaen,Want wy zijn Gods dienaren.Gheen oorsaeck gheeft // der onbeleeftIn dese tijt der ghenaden,Dat ghy niet sneeft // en hy niet heeftWaer mede hy u mach schaden,Op dat ons pont // niet gae te gront,Maer onse licht // com int ghesicht,Op dat ghelijck wy lesen // oock by desen,Gofs naem, eersaem,In ons hooch wert ghepresen.Alsulcken leer // eyscht God die Heer,Die beyt met woorden en wercken,Soeckt sijne eer // cloeck is te weer,Sijn teere spruytkens te stercken,Dese met deucht // verweckt die jeucht,En maect verblijt // alle die belijt,Christum en sijnen vader // drucx ontlader,Doet goet // met spoet,Ghy Christen alle gader.Voor sulcke vreucht // voor sulcke deucht,Voor sulcke groote ghenade,Die ons verheucht // al dat ghy meucht,Lovet God vroech en spade,Wie sal ghenoech // in eenigher voech,Lof prijs en eer // doen onsen Heer,Van dese schoone saken // vant vermakenDoort sien // van dien,Die totter kennisse raken.Tot lof en danck // der lippen gheclanck,Laet ons gheheelijck begheven,Vry ende vranck // met vrolijck ghesanck,Altijt so langhe wy leven,Oock hert en sin // van dit ghewin,Want hy is goet // die't wassen doet,Bidden dat hy ons soare // en beware,Na dit ghecrijt // met hem subijt,Ons hem ghelijck verclare.
