'T godlijck wesen goet // heeft des menschen Met vleesch en bloet // becleet in dit Die de mensch onvroet // twist in zijn leden Tot zijn onspoet // van vleysch tegen den So dat in onrust minst en Dier leeft bedeest // door aartsche lusten Maar God den sulcken van quaalen Die zijn woorden vreest // in desen nieuwe Die d'oude mensch treckt uyt, met lust die daar wt Wort dan Christi bruyt // en trect den nieuwen So t'Sarpent beduyt, verschut hy zijnen Soect het eenich cruyt // dat hem verlost van Die sal van desen Ghenaad' ontfaan // als een ghetrou Dat hy het vleesch den Gheest maeckt En leest onbelaan // in desen nieuwe Maer eyghen wil verblint // die yet geschapens Daar selven bevint // in benaude commer Want de schrift ontbint // wie vleyschelijc is Dat die hier verwint // int ent tot loon de Die bracht den eersten mensch in Doen lust hem bood // D'appel als een Hebt hier op acht ghy menschen D eyghen wil verstoot // In desen nieuwen Christus ons Eglentier,, In liefd' bloeyende Heeft u Drinskens hier // verlost uyt desen Schuylt onder den banier // van die u cocht so So wordy schier // van eyghen wil Siet dat ghy oock quaa lusten U hart besnijt // en soeckt dees Elck Spieghelt hem tot zijn eyghen Soo leeft hy verblijt // in desen nieuwen
