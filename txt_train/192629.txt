AVe Maria, Godts liefste Lief ydone,Der Maegden kroone // Maria weest gegroet,Gratia plena, klaer als de Sonne schoone,In 's Hemels troone // vol van ghenade soet,Der deughden volle vloet // in suyverheyt ootmoet,Hebt ghy gebaert // Gods Soon ons hoogste goet.Dominus tecum, de Heer sy met u reyne,Suyver Fonteyne // vrucht aller Werelt wijt:Benedicta tu in mulieribus, alleyne,Al waert ghy kleyne, ghy waert gebenedijt.Geen Vrouw tot eeniger tijt // so salig als gy sijt,Nochte soo verheven, van Godt tot 's vyants spijt.'t Benedictus, gedanckt, gelooft gepreesen,Moet altijt wesen // dat hooghweerdigh pant:Fructus Ventris tui, uw' vrucht reyn uytgelesen,Vyt Iesse geresen, Jesus Christus playsant,Die als een Leeuw valjant // ons brack der hellen bant,Ende die gevangen bracht // in sijns Vaders lant.Sancta Maria, Princess' hoogh verkooren,Edel geboren // van Koning Davids Zaet,Ora pro nobis, bidt voor ons die te voorenWaren verloren, verdoemt door sonden quaet,Tot dat ons quam te baet, Jesus ons toeverlaet,Van u geboren // o schoone dageraet!
