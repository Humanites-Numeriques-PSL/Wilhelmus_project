DIe vred' Gods die te boven // gaetDiens name hoogh te loven // staet,Wensch ick u al te samen Waer dat ghywoont, met namen,  Int groene lant // aenelcken cant, Wandelt na u betamen,De Heer biedt u sijn hant.Al schijnt ghy nu verlaten // schier,De Heer com u te baten // hier, Och wilt datwel bewaren, Dat hy u ginck verclaren,Sijn heylich woort // dat ghy ghehoortHebt, van synen dienaren,Brenght doch goed' vruchten voort.Mijn liefste uytvercoren // al,Wel hem die dees stem hooren // sal, Comthier mijn uytvercoren, Mijn Rijck staet u tevooren, Ontfanght het loon // in 'sHemelsthroon, Ghy sult vreuchden orbooren,Ontfanghen 'slevens Croon.Siet, niemant sal verblyden // saen,Dan die nu deur het lyden // gaen, En dieals rechte leden, Christi voetstappen treden,Den smallen ganck // haer leven lanckMet Christelijcken zeden,Ghelijck een vruchtbaer ranck.Treurt nu niet om den tegen // spoetDen welcken ghy verdraghen // moet,Ist so Gods wel-behaghen, Soo wilt hemwillich draghen, Lijdtsamich zijt // in uwentijt, Het duyrt hier weynich daghenGhy sult noch zijn verblijt.En wilt doch des vergheten // nietDat u de Heer noch weten // liet,Sijn heylich woordt gepresen Wilt sprekenende lesen Te samen hout // elckander bout,Elck zy ghetrou in desen,Dat hem hier is betrout.Recht houdt doch u becoepen // vry,Ghy Ouden, dit ontknoopen // wy, Wilteen goet voorbeelt gheven, Den Iongen metu leven, 'Tmoet zijn gewaeght, weest nietvertsaeght Ghy sult worden verheven,Soo ghy u dapper draeght.Siet toe ghy jonghe Ieuchde // bloot,En hebt doch gheen gheneuchde // grootIn s'weerelts lust voor ooghen Dat ghy nietwordt bedroogen, Van 'svyandts strijt // dieu benijt, Sijn doen is al ghelooghenWacht u daer voor met vlijt.Den Weduwen tot reender // eer,Gheeft Paulus schoon tot eender // leerDat sy haer hope maken, Op Godt vooralle saken, Want hy beleeft // d'uytcomen gheeftWel hem die dit can smaken,Gods Gheest niet teghen streeft.Iaeght doch nu na den vreden // soet,En sout in al u reden // doet, Elck wil sijntongh' bewaren, Om niemant te beswaren,Oock leughens quaet // de Heere haet,Hier in als Gods dienaren,Bruyckt doch de rechte maet.T'samen wilt doch nu draghen // 'tlast,Elckanders na't ghewaghen // vast,'Tis Christus Wet waerachtich, Weest desaltijdt ghedachtich, Al dat ghy saen // gheernhadt ghedaen, Doe nae dat ghy zijt machtich,So sult ghy wel bestaen.Vrienden ghy staet int herte // mijn,Het soude my groot smerte // zijn,Dat ick van u moest hooren, Anders als hierte vooren, Doe ghy ghesont // bekendet rontDat ghy als uytvercooren,Woudt houden Gods verbont.Ghesonden is dit cleen // ghedichtTot u, dat ghy wordt reen // ghesticht,En dat ghy oock meucht vaten, Dat ghy nietzijt verlaten, Maer dach en nacht // in ons ghedacht,De Heer com' u te baten,Bewaer u deur sijn cracht.
