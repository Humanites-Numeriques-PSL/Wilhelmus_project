EEn roede groen // uyt Jesse Stam 's Winters saysoen // wy 't vruchtbaer hout Wiens vrucht is spijs // om de ziel te Heylig en wijs // met noch meer ander Die door sijn kracht // levendig In ons gebracht // als rancxkens op te Dit vruchtbaer hout // staet aen een schoon 't Brengt twaelf fout // vruchten van waerden En op een Jaer // alle maenden sijn Voor al die daer // van te rapen Zijn geresolveert // voor alle vremde Ongesommeert // wie kan sijn deught Wie hier de smaeck // van sijn vrucht heeft 't Geeft hem oorsaeck // om te meer zijn 't Is in den mondt // boven spijse der Het maeckt gesont // in den Lust-hof te Men wort daer door // voorsichtig en En om wel voor // 't quade te zijn Prins uwen lust // zy in Gods Wet Vierigh gerust // in Gode seer Gebruyckt dit hout // tot een staf in u Het maekt u stout // te gaen door de Van 't Jammer-dal // daer men veel verdriet Het leyt u al // van d'een tot d'ander
