Wilt nv al // Groot en Den Mey helpen Want Auroras Zijn nu an Van ons heel vroech Comt int dal, Met Aenschout Phoebus Vulcanus moet Die was ons In den Winter Wech Mars dienaren // De noch Ons vreucht alsomen Neptunus baren // Boreas Maect nv qua stromen // Want haer cracht // is Siet Saturnus Die haer macht // niet en Doet Ceres Deur smeys bedouwen // Meucht ghy Men siet daer // Venus Pallas deur Geeft reden te Voor menich Flora claer // De doet bloemkens Bewijst Mercurius Retoricas Dianas dieren // Schoon met Sietmen versellen, Haer hoofden chieren // Met Luna opstellen, Pan om hoen // Gaet in Lammerkens Om te voen // Gaet hem Ter velden Seer groen van bladen // Al deur smeys Gaet ten troon // hoort De Nimphen Vreucht wert daer Deur Cupidos Van Apollo Eenen schoon // Lauwer Wert hem daer Dus sit hy By Venus Als een Godt Musicas keelen // Hoortmen daer En accorderen // Harpen en Veelen, Lieflicke Den Mey ter eeren // Noch tgheluyt // Van t'Gevogelt Alle cruyt // Oock Lieflick van Noch staet ontloocken // Den Mey Metter daet Meucht ghi princen Deur des Musels Sijn alle dese By haer Fonteyn Nu ontslaet // Sonder Jupiter zijn // Soo dat Bachus By Venus zijn Deur Copido haer De deur zijn stralen // Plutos doet Deur cracht der minnen, Lieffelick talen // Gaet hy Om te verwinnen, Dat niet deucht // Wert Hymenius En deur vreucht met Gaet Momus Die doch moet laecken // Des Meys
