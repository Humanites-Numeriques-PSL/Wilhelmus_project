V wel bequame // Door uwen Name // Helpt rechten wonder // In u macht, stercken // Men 'svyandts wercken // En al kan t'onder // In uwe groote // Eer ick de bloote // Helpt my toerusten // Want tis van noode // Op dat ick doode // Altijdts de lusten // T'gheloove t'eenen // My altijdts leenen // Want meer dan stalich // My dient in elcken // En oock in welcken // De hop' om saligh // Dien Hellem vryde // Oock aen mijn zijde // 'sGheests zweert, de waerheyt // Tis eenen goeden // Die reyn ghemoeden // In suyver klaerheyt // In liefde vyerigh // Die goedertierigh  // Wilt my rechtsinnigh // Mach sy my blyven // Ick sal beklyven // En overwinningh //
