O Christus lief minnare Die my schenckt u dierbare Na u verlangt my t'allen Soo den Hert zijnd' in vare Nae de Fonteyne klare // In stortich pijnelijck Ick leve in zwaer Hoe langhe, Lief, sal dit Wanneer sal ick uyt rouwen // Wanneer sal ick aenschouwen // Aenschijn, schoon boven O Lief in 'sWerelts wijcke // Van my in gheen versijcke // Maer helpt my, dat mijn teere Niet en buyghen ghelijcke // Maer van u soete Rijcke // My hier in dit dal Want ick vinde my Van den Prince der De Werelt int verstranghen // Alle dat nijdich Slanghen // En laet, mij niet met Ionstigh voor my gheneghen // Leydt my in uwe weghen // Door uwen Gheest, o Lief Want siet, ick ben bedeghen // Een teecken daer elck teghen // Voor rasernije acht men mijn En ick nochtans Wt uwe sijde sonder V Lief, u Bruydt, u Kercke // Ick doe door u o stercke // De Helsche poorten O Prince der Princieren // Ghy die herten en nieren // Aenhoort mijn bitter Vwen Gheest goedertieren // Dat hy door zijn regieren // Mijn ledekens na u Want 'tzijn nu quade Hierom Lief moet ick u Sal ick van al dit lijden // Met u niet haest verblyden // En 'sLevens Croone
