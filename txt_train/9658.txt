DE Mensche is te wonder // Soo langhe als hy sonder // Met vleesch en bloet te rade // Want hy tot alle quade // Daer lust ende gheneghen // Soo dat hy in veel weghen // Alsoo wort hy te samen // Den soeten aenghenamen // Van Godt een uytghelesen // In welcken saligh wesen // Hy, die met recht bekeeren // Openhertigh des Heeren // Het woort zijn Ziele maken // Saligh, maer eerst versaken // Hy zijn eyghen qua leven // Aennemen en aenkleven // T woort, met een reyn sachtmoedich // Daer zijn vrucht overvloedigh // Wie in des Heeren reden // Sterck, als van t'lichaems leden // Die kennisse der waerheyt // Ghekrijcht, en wort in klaerheyt // Vant duyster jock der sonden // Hy onder lagh ghebonden // Maer die gaen buyten desen // Sullen soecken en lesen // Nochtans die waerheyt vinden // Want soo met alle winden // Daer waeyt dees onghestadich // Al hebben zy weldadigh // Maer om vlieden 'tverleyden // Soo dientmen t'onderscheyden // Die Bruydt uyt alle Secten // Die door sHeeren perfecten // Noch wandelt onghevloden // In alle Gods gheboden // Reyn liefde tot den naesten // Die tot onderwijs haesten // En oock tot beteringhe // Oyt woordt al sonderlinghe // Dit veldt-teecken, ydoone // Die onbevleckte schoone // Dese staet op op vaster // Dan die met haren laster // Den sone van haer moeder // En spreeckt van zijnen broeder // Maer die vernieude jonghe // Honigh vloeyt haer ter tonghe // Sy heeft in desen heunigh // Tis twoort Gods daers' op steunigh // En oock om op te wassen // Hier op behoeft te passen // Dat hy niet en verstroye // Come ter rechte kope // Welck huys Christus loyaligh // En sonderlinghe saligh // Als hy t'onkruyt verloren // Te zijn tarwe verkoren // Daer gheeft ons Godt ghenade // Die ons van allen quade //
