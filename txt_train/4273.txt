MInnelick Bloemken van geure // Die boven cracht In uwen Ghy haestich t'suere // Veranderen in Ghy zijt een Soo Daer niet dan deught // en eere in Aerdiger En was in hondert // Daer meer troost by Die mijn Gans wtgesondert // So haestich bringht te Nu ghy u Behulpich Hoe can verdriet // my deeren Rustich (Vrij aller plagen // Ben ick, by uwen Te zijn In u behagen // Schat-ick den hooghsten Faelgeert u Op desen // Geen toeverlaet // op aerden wesen // Int herte My als een plante // En hebt, van S'gelijcx ick Als een constante // Prente in mijnen Sulck een Houd-ick in Als t'wesen in // der Goden Adieu, hier Moet het gescheyden // Princesse Wilt (met Jonstich verbeyden // Dienaer seer Die niet en Vreucht, of Dan u present // te wesen t'aller
