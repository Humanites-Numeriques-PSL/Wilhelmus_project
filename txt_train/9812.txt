PAscha, dat hier // voor by te 'tSondigh bestier // moeten wy 'tDangier // van Als fenynighe Slanghe De Christen schaer // heeft hier Geen blijf-stadt, maer // voorby moet Al naer // de Daer haer // hert is, en haren Gheacht zijn sy // als Of wy voorby // door een stadt Daer wy // schier Ons aentrecken watter Of men hier hoort veel vreucht Wy moeten voort // recht henen Nae 't woordt // van Die ons roept nae Tis groot abuys // hier Om vlieden 't cruys // en blyven Sijn huys // opt Buyten 't eeuwighe 'S Heeren dagh zal // als strick Die op dit dal // woonen en Met al // den Op de dinghen die hier zijn Want vyerigh moet // vergaen dees Iae al haer goedt // en borghers Dus doet // hy Die uyt midden van haer wijckt Want die hier geen wooning en Dan op de been // om voort te Alleen // door Die blyven in 'sHeeren 'tGhemoedt geschort // op, Het herte wort, met Godt Ghegort // in Der liefde die vyerigh 'sLams goetheyt meest // de sulcke Ter Paesschen-feest // daer sy Den gheest // met Die de ziele van binnen Door d'ongheree // en Woestijn oft Zee // recht henen Gheen wee // noch En leyde ons af van den Sy sullen met // hem t'huys Die nae sijn Wet // verwinningh Oock het // hier Wel haest wesen volstreden Dan sal men hoen // en cranssen Des levens groen // en den Roof koen // Sal onsterffelijckheydt
