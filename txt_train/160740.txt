QUijtschel van sonden geeft de Door zijn louter Die hier af laten en niet Lust hebben tot den Als een groete // laet hy Inde Schrift orconden // Komt met moete // neemt u Goetwillich voor die zonden // Ick en begheere niet den (Seyt de Heer) der Maer dat zy haer bekeeren Behoort, en laet Tsondigh sneven // dat zy t Sy heyden of Jode // 'K wil vergeven // dat zy Wie tot my komt hoe gode // Op dien daer van het quaet Wil ick niet als den Seer nauwe straffen zijn Noch hem oock niet Wilt doch komen // met den Soeckt u ziele bate // Hebt verschromen // voort Eer dattet hier te late // Heb ic u niet gebeden // Dat ghy u zout Tot my, en my met vreden // So wil ick u recht Mijne wegen // zijt U vant boos voorstellen // Dat mijn zegen // u Hier namaels uyt der Hellen // Waeromme wilt ghy sterven // Vraegt God door die Keert weder om verwerven // Mijn goetheyt U onthouwe // en mijn Weder aen u warghde // Boet en rouwe // heeft die Bewesen, g'naemt Maeghde // Ick op der aerden niet en Voor die rechtveerdich Maer voor die daer ic meest Lagen in 's doodts Met veel wonden // Heb ick onverholen // Tzelven stonden // wel Met Mattheus den Tolle // Ist dat met u noch heden // Wilt u doch naer my Zijn u zonden voorleden // So sal ic u Tot versmullen // wilt Mijn Huys seer ootmoedich // Sonden sullen // wit als Worden al waer zy bloedich // tVrouken van Canane seer Dat bat met seer veel Die Heere gaf nae haer Hy sacht geloof int Christum wrochte // en Dat niet en misluckte // Voor haer Dochter // wiert al Want den Duyvel verdructe // Den Pharaseen en Hadden vooren In den Tempel om bidden Die heeft hem gaen D'ander wende // hem niet En dede gestadich // Hy bekende // zijn En sprak Heer mijns genadich // Die Publicaen quyt schelt Van sijn Goddeloos Rechtveerdich in sijn huys hy Voor den ander Prince cierich // U deucht doet versachten // Die manierich // tot u Comt, ghy niet verachten //
