Komt hier Nymphjes al te Die tot vreughden altijdt zijn Vlecht een Krans // voeght u by desen Aen de Ry // by de Harders, pluckt nu Bloempjes Ay soeckt doch Kruydt // van viertuyt, tot Van dees Maeght // die meneert dat haer Noemt haer lof // nu seer grof // spaert gheen Maer verciert // nu haer hooft met een Krans Tot cieraet al van haer Die sy nu door brenght met groote Sy ist waert // door haer deught, dit sy baertmet Want sy haer door 't soet vermaeck Dus wilt haer eer // breyden veer, met Want 't gheluyt // speelt in 't dal doch op u Schept doch hier // goede chier // met Haer ontmoet // met Laurier haer dees eer aen Ick sal u in 't eerst voor Spoeydt u Nymphjes soetjens achter Komt nu ras // hier ontrent // met Want ghy bent // met u kruyden Hymus wel Orpheus queelt // ende speelt met u Op Musijck, aen dit vloeyend' Beeckje Dat daer stroomt // door 't Gheboomt // niet en Voor 't Gediert // dat in 't Bosch met gedruys so Phoebus helder glanssend' Brenght ons met zijn straeltjes in 't Want de Zon // hier op d'Aerdt, aen dees V bewaert // op dat vreughdigh, ghy dan wort Ghy wort gekroont // en geloont // nu ter Dus weest ghy in den Heere nu t'saem Die ons loon // nu seer schoon // stelt ten Voor ons oogh, dat het straelt, Tot den Hemel
