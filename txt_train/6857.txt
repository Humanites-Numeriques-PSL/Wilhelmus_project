Glorieuse violette // Koninginn’ van Suyver vat vry sonder smette // Inden geur Soeticheyt sonder Sulck een vreucht ghy sijt // die vermeucht Smert en leet te Met vreucht, jolijt // verheucht, Ons weemoedich Precieus Paleys van trooste // blinckend’ als den Sterre lichtend’ uyt den ooste, als de Sonne claer Uwen lof elck metter Hondertsout verclaert // als een goe Iun der Hemels Benaut, vervaert // verstaut, Is nu elck Alderheylichste Princesse // Maria Gods Moeder Ons aller Advocatersse // Siet ons in dit droevich Tot u roepen wy Voor u dienaers doet // der genaeden Vwen Soon Dat hy ons ‘tgoet // sijns rijcke Will’ hier naemaels
