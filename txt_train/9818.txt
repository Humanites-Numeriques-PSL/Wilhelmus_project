REyn van manieren // is den Die hem hantieren // in den rechten // Tis slechten raet, die hem verbieden, Beter is trouwen, dan te lyden Want die horeren // Godt condemneren // Des wy lauderen // T'Huwelijcx Tis waer dat vrienden // in malcanders Dickwils wel dienden // al bystanders Maer anders groot, met een affecty Staen man en wijf ter noot // malkander Wie een goe Vrouwe // Die Godt ghetrouwe // Bevrijt van rouwe // Te langher Een Wijf die deughdigh // is van wesen 'sMans herte vreuchdigh // sy genesen Ghepresen moet // sy zijn om haer Een edel gave Godts is sulck een Tot troost van haren // Die wel bewaren // Haer tongh, en sparen // Weder Soo lief bequame // op der eerden Als een eersame // Wijf vol weerden Met sweerden vliet // een roover in der En brengt den roof // haer die boven Hy voor vriendinne // Veel zijn door minne // Gheworden sinne // En Een vroom wijf schoone // is als brandt De Lamp ydoone // opt heyligh Zijnde bestande haer // eerbaer Een gulden Pilaer op een silver Is sy gheleken // Om niet te breken // Oft te ontsteken // Door boosen Die een Wijf zedigh // voor zijn proven Huysich, cuysch, zedigh // maghse loven Al boven gout // en boven edel Haer cieraet is, wat neerstigh is en Al moets' hier lijden // Haer vreucht in tijden // Met een soo blijden // In 'sHemels 'tWijf sal door baren // ende kinder Ten Hemel varen // datse ginder Soo inder kracht // gheloove, liefde Heylicheydt, eerbaerheyt haer blijft Voor dees versijcke // Werdt sy Gods rijcke // D'Enghels ghelijcke // Werdt Prince d'oprechte // so sy deden Sal men in Echte // niet besteden Te vreden seer // wel, eeuwigh sonder Sullen sy in 'sLams Bruylofts feeste Soo vrolijck maken // Elck magh wel haken // En neerstigh waken // Om daer te
