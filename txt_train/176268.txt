HOe diep zijt ghy ghesoncken // met u ghewenschte O soete sachte voncken // en borrelt in mijn Ghy komt my staegh vermanen // van myner sielen Som parst ghy blye tranen // ten droeven ooghen Gheen onweer u en blusten // uyt mijn verheert En of ghy som wat rusten // 'twas ga'ren van u V crachten die my glissen // door't merrech in't Soud' ick niet willen missen // om 'sdagheraets Al doedy somtijts quynen // als't lichaems ooch Mijn siel siet lijckwel schynen // de schoonheyt die zy En schept soo groot behagen // in't Goddelijck Dat zy nae smart te draghen // voor sulcken wonder Een wonder dat verflaeuwen // alle ander schoonheyt En dalen haer wijnbraeuwen // en sincken haren De leden en de zeden // en't vierighe Op Pallas leeft ghesneden // van Pallas eyghen Omhelst mijn gheest vol minnen;, en zy, van hare Omhelsen al mijn sinnen // met even stracken Had my dit heyl begheven // soo was mijn weelde En al het doffe leven // ontseech my Des die van 't overwelven // des Werelts Bouwheer Gheef dat ick mis my selven // eer ick u liefde
