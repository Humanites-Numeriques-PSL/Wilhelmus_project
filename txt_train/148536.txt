ICk arme Kindt // waer sal ick keerenDan my verneeren // onder Godts handt,Dat my den windt // der valscher leerenNiet en magh deeren // Heer doet bystant.O Heer Godt Vader // vol van genade,Van alle quaede // my doch bewaert,Gy Fonteyn-ader // vloeyt vroech en spadeOp dijnen zade // van dijner aerdt.O Godt, doet my // verjubileeren,Ick sal landeeren // u Name groot,Al soud' men vry // my persequerenEn tribuleeren // tot in der doot.Sal ick niet haest // met u verblijden,In witter zijden // o Vader jent?'T leven verbaest // is niet dan strijdenTot allen tijden // druck en ellendt.Tot u, o Heere // wil ick my geven,Mijn steen verheven // mijn toeverlaet,Want in u leere // vind' ick het leven,Hoe soud' ick beven // als gy bystaet.Ick bidd' u seere // om uwe gracy,Tot elcker spacy // verhoort u Bruydt,Vertroost u teere // door jubilacy,Als 's Werelts nacy // sal roepen luydt.Als haer de Menschen // seer verblijdenIagen en rijden // nae's Werelts loop,Dan is mijn wenschen // en mijn belijden,Wilt my bevrijden // voor sulcken hoop.Nae u alleene // zijn mijne gangen,Ick ben behangen // met sonden snoot,Nae anders geene // staet mijn verlangen,Wilt my ontfangen // Heer in der noot.Sal dees turbacy // noch lange duiren?Dit doet my truiren // troost my, o Heer,Want door blamacy // sy veel vervuiren,Door het versuiren // in uwe leer.Soo wilt bewaeren // sparen en wachtenMy door u krachten // mijn Middelaer,Wilt my vergaren // leyden en achten,By de geslachten // onder 't Outaer.O God mijn Heerder, doet my doch groeyenDoor liefde bloeyen // als ranck vruchtbaer,Ick weet geen weerder // wilt my besproeyen,Laet op my vloeyen // u Geest eerbaer.O gy Fonteyne // vol goets gepresen,Daer uyt geresen // is't leven jent,Gy kont alleyne // ons Ziel genesen,Wilt by ons wesen // tot in den ent.Nu bid' ik Prince // mijn Godt almachtigWeest nu doch achtigh // u Schaepkens al,O Heer, bemintse // uyt liefden krachtigh,Weest ons gedachtigh // in't jammer-dal.
