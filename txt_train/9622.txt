ALle vleesch is hoy, na Schrifts Kort is den tijdt, van smenschen Als bloem in't veldt, ontploken Moet hier subijt, sijn glory Wat is met den Mensche Een uyr en is hem niet Hoe langhe hy // sal leven Sijn perck hebt ghy // o Godt Dat oudt verbondt // is stervens End' nae den doodt // wy al Noch eenen dagh des oordeels Daer werden bloot // alle Als Christus komt met sijn Sullen huylen alle Der Aerden quaet // veel drucx Al die den raet // Christi Voor den Rechtstoel Christi Op den persoon // werdt gheen Die 't herte kent, sonder Sal gheven loon // elck nae sijn Kracht en sal niet baten den Noch geen kloeckheyt den wijsen Recht oordeel klaer // sal Strijcken aldaer // dat hooft der Seer vreeslijck Godt hem wreken Over 't getal // syner Die de Wereldt beminnen Want haren val // wert groot ter Haerlieder vyer sal eeuwigh Haerlieder worm die blijft Rouwe, gheween // werdt daer Angst ende een // knerssen der Dan sullen die rechtveerdich Vry van ghepijn // ofte Blincken ghelijck der Sonnen Met vreuchden fijn // niet om Int Rijcke Gods, nae haer Onsterffelijcke Croonen T'Lammeken soet // sal haer tot Der tranen vloedt // van't aenschijn Een Princelijcke Heeft Godt bereydt // sijn Dus is des loons groot Vast toegeseyt // ende Tusschen die Godloos Godt teghen En die hem dienen Wat neersticheyt // men hier toe Gheenen arbeydt // en werdt
