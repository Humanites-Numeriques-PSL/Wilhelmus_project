Het is een schoon dierbaer Datmen maecke // onses Gods Groot, met Lofzangh ende Soet van smaecke // is hy Iae vriendelijck ende Laet ons zijn faem // verbreyden Looft hem ten rechten // die zijn knechten, Met groot Looft den Schepper Die alle dinghen heeft ghemaeckt Alle eere // tot hem Nu end’ t’aller Synen Stoel is boven Hoogh verheven // wijdt Veel duysent Enghelen Die beneven // voor hem Sijn gramschap doet berghen Die diepten saen // verdroghen Voor sijn aenschouwen // mans en vrouwen, Looft hem, waer ‘t Meerder dan alle lof is God is overal, verr’ en In hem leven // ende Allegader Laet ons Godt een nieuw Liedt Groote dinghen // doet hy Voor hem beeft ‘s werelds Oock springhen // de steenrotzen In stucken, voor zijn toornigh Maer in den noot // is hy seer Die betrouwen // op hem bouwen Hy helptse Wt lyden ende groot Hy gaet met haer door water ende Hy kan mercken // al ons Hy proeft hert en Hy weet al ‘s menschen Al haer trachten // en Klaer als dagh zijn hem de Groot van machten // is zijn Hy hoort en siet wel alle dinghen, D’oore gheplant // hebbend’ als wijs Soud al vooren // oock niet hooren, Soude, door D’ooghe gemaeckt is, oock niet konnen Hy weet alle dinghen eer sy Nae Schrifts getuygen // voor hem Moeten alle Godt is een Coninck Ende desen // woont in een Daer niemand toe komt Sijn Wesen // van menschen Is verborghen: hy heeft Al het ghewicht // der aerden Seer groot bevanghen // en ghehangen, Op ‘t water Met zynen woorde krachtigh ende Ende hy onderhoudet oock all’ Sijn vermoghen // Blijckt wel Al zijn wercken groot Diemen gronden // niet en Van zijn heerlijckheyt t’Allen stonden // eenen Doet den anderen daer af Maer noyt en sach // yemant God vanden Maer zijn verkoren // een-gheboren Heeft ons seer Dat verkondicht // Lofprijs sy hem te Die op den stoel sit, weerdich alle Eeuwich ghepresen // moet hy En dat Lam Doe wy Gods Vyanden Iae sondaren // boos en Ginck hy ons zijn Liefde Sonder sparen // gaf hy Voor ons allen in doots Synen seer wel // beminden Sone Die ons in noode // met zijn roode Verloste Siet, wat een liefd’ ons God den Vader Dat hy ons als kinderen zijn Maeckt deelachtigh // hem Eeuwich Lof zijn Sijn ghenade Is verscheenen // over Tot profijt van den Die recht heenen // in dit In Christo wandelt den wech Maer heten sal // na Schrifts Den obstinaten // sonder baten, Hem en Gheen ghenade // maer onghenade, Toorn, anghst, druck en zwaer Wie te vooren // synen Hier niet en Tot quijt-scheld van ‘t sondigh Is ghegheven // die Iesus den name Tot ons leven // ende Des Vyants macht heeft hy Blijft niet verdooft // wie ooren heeft die Wilt ontfanghen, met verlanghen, ‘t Reyn Schickt u te gaen door d’enghe Met liefd’ en broederlijck Brenght den name // Gods Vrucht der lippen Vreest Godt al ghy Menschen Gheeft hem eere // want het Nu den tijd zijns oordeels Nae zijn Leere // altijdt Aenbidt hem, en niet en Die heeft gemaeckt // Hemel en ‘t aertsche Zee en Fonteynen // desen reynen, Sy toe Eeuwigh lof, eere, prijs, ende Ghy Conincklijck Priesterdom, hier Vry met breughden // al zijn Al wordt ghy In Gods lof wilt niet Christen rancken // Wilt in alle dinghen Spijs en drancken // hy Geen mensch en wort voor hem vergeten, Sorgt noch en weent // niet, om dat vleesch Maer wel gade // Gods ghenade, Daer in Met vaster herten blijft vroegh ende By den Heere, volght zijnen Om niet verdolen // hem Blijft, Christelijck
