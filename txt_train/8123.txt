O Ionckheydt groene, Bloeyende In dit saysoene, Schickt u tot Wilt u begheven, tot Alle u leven // daer toe Met sout doch menght Wat ghy Spreeckt ofte singht // alsoo ons Paulus Laet u doch raden, o Iongheydt Teder van bladen soet van Wilt u doch keeren, van 's werelts Vreest doch den Heere // terwijl ghy Want siet die doodt, spaert kleyn noch Hoe jongh, hoe oudt, 't wordt al tot hem En wilt niet treden // om wat nieus te Als Dyna deden // ten ouden Als sy aenschoude, des Landes Quam sy in rouwe, en swaer Haer schoonheydt jent // heeft hy V hier van went // ghy jonghe dochters Maer wilt aenschouwen, Het vroom Vande ghetrouwe // dochter Iephta Die gingh versmaden // al 'swereldts En liet verbraden, haer lichaem Op den Outaer // van haren Spieghelt u klaer // en houdt dit voor een Dit differenti, wilt doch En maeckt doch menti // al van des Hoe d'een bedorven // ende D'ander ghestorven // heel Dus wilt doch fijn // de ouders Ghehoorsaem zijn // sedich en Op dat ghy princelijcke Dochters Mocht mede van de wijse Maeghden Die metten Heere, werdt Want tot d' oneere // sal worden Dit droevigh Liedt // ick ken u Gaet in 't verdriet // day voor u is Daer sal gheschieden // soo groot Nae Schrifts bedieden // in 't Helsche Wringhen der handen // druck ende Knerssen der tanden // t'eeuwighen Hierom o jeught // schickt u tot Op dat ghy meught // hier van bevrijdet En dan meught hooren, het soet Mijn uytvercooren // mijn lieve Wilt tot my treden, ontfanght de En gaet in vreden // in 's Hemels Blincken hoort mijn // als der Sonnen Dat sal dan zijn // der uytvercoren Dus wilt vercieren, u Lampen Met goe manieren // waer dat ghy Laet u licht rijsen, wilt liefde Altijdt bewijsen, soo sal Godts V blijde ontfaen // en voor u En dienen saen // al in zijn Vaders Oorlof hier mede, o Ieught Godts eeuwighe vrede wensch ick u Dat by ons blijve // tot 't ende En ons verstijve // in een vroom En nae desen tijt, te zijn Wensch ick met vlijt // de Ieught tot eenen
