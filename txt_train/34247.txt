DEn Mey die komt ons by // seer Den Winter kout ons nu verlaten ’t Geboomte dat komt uyt // en Door Phoebus, die ons nu sijn warmte Dianâs nachten langh // Boreas stuur en Heeft ons verlaten, in vrolijckheydt Den Nachtegael singht // het Geytjen dat Het Swaluwtjen Godt Pan die sat in ’t veldt // en Sijn Schaepjens die daer weyden in het Sijn Pijpjen hy doen nam // en Op eenen bergh, daer ’t seer geneugh’lijck In d’oogh kreegh hy seer dra // sijn soete Die Coridon ontmoete seer Sy wierden verstoort // hy leyde haer En wierpse in ’t Coridon seyd’, komt rust // met Mijn soete Sylvia, wat neffens Een suchtjen haer ontvloodt // seer Nochtans lonckten haer ooghjens even Coridon sprack haer toe // mijn soete Beckjen, ‘k En bender niet verre, sucht ghy om Seght Herderinne // quelt u de Komt klaget my Och! neen, ’t en is dat niet // maer Mijn Schaepjens zijn verstrooyt hier in dit Een Lammetjen ick hier mis // ’t Is voor mijn herte een groot Coridon laet ons gaen // Sylvia blijft wat In ’t ruysschen der bosschen is geen Niet dan gewoel // van bladertjens Ey! maeckt my niet Ick schencke desen Mey // seer Aen een Princesse die mijn hert Die soete Veldt-goddin // der Die haer Minnaers vermaeckt tot allen Maer siet doch eens met lust // hoe dat Coridon Sijn Sylvia mint, sijn herteken Maeghdetjen reyne // lost my van so leef ick
