huys des Heeren // sullen Wy gaen met eeren // soo hy Om vreuchd’ verwinnen // hier boven Ons voeren binnen // de poorten Blijft op den ganck // u leven Al zijn wy kranck // die hop’ v’steect ons Een stadt ghebouwen // heerlijck en Haer Borghers houwen // die  Men mach by trouwen //  seer hoogh Haer licht aenschouwen // het lucht int Och vrienden jent // maeckt oock Dat ghy nu bent //  borgers elck als een Recht doch u wesen // als Christen Aen Ziel ghepresen // hert en Als d’eerste deden // met Ziel en Eenich in vreden // te samen Gods glory schoon // spant daer de In s’Hemels Troon // sal blincken haer In dese mueren, eert Godt den Lieft sonder trueren // elckandren Gaet opwaerts beden // voor Godt Drijmael t’gheeft reden // int nieuwe Hem offer bringht // danckt, loeft en Dit vyer ontsinckt // de liefde in ons Climt op de graden // volheyt der Dat ghy gheraden // t’gheloove Betuygen waerlijck // als volck Gods Wt Godt schijnbaerlijck // ghesproken Met cracht en daet // soo Christus Elck huys bestaet // dat men so bouwen Kent, wilt ghevoelen // daer zijn De Rechter-stoelen // nae rechter Oordeel te gheven //  in Davids Nae een elcx leven // sonder Crijcht elcken slecht // daer loon nae D’onnutte knecht // sal daer worden Perfeckten vrede // soo’t Godt Crijcht niemandt mede // dan die nu Liefde te vooren // int herte hier Reyn uyt vercooren // sonder  Godtloosen siet // en hebbens Want vrede diet // Christus den sijnen Ick wensch u t’samen // vrede van Liefde met namen // t’nieuwe Naer Schrifts verclaren // Wilt openbaren // volght Christi In deucht gaet voort // al nae sijn Niet anders hoort // en sal behaghen En in u toorens // breete ende Moet zijn te voorens, vred’ over Daer men behoedich // de wapens Bereydt en spoedich // seer Om de trompet // te blasen Zijt ghy gheset // t’volck Gods is Treedt op u voeten // t’gheluydt nu Ontwaken moeten // sy niet te Den vred’ wilt preken //  straft nu by Soo onbesweken // u Ziel Gods volck certeyn // de kudde Bewaert nu reyn // want ghy haer wachter En  wassen, groeyen // moet ghy Gods En voerich bloeyen // te wensch u Om t’lof te bringhen // voor Gods Is nu het singhen // en t’wenschen Vreedlijck vergaert
