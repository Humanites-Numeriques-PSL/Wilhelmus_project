PLach mijn jonck Dus te versuchten // Gelijck het nu Soo dat my Volght, met veel duchten // Om een maeght wel Geen soo Te voren // Geen schoonder kint // en is geboren // Een sulcken Doet my der minnen // Dat ick vertwijffelt Of sy my Sal doen gewinnen // Daer ick om dolen Ick Mijn smerten T'blijft even nae // geen troost en is voor Troostbaerder Sy ten beginne // Sy my nu dunct te Hoe can soo Sulcken vriendinne // My keeren haer Meerder Doolyck En heeft noyt mijn // jonck hert neder Roup ick om Wie sal verhooren // T'is al om niet Voor mijn Verstopt haer ooren // Rebel, en hert Siet my Arm Ander verleent // sy troost, en my Van Is sy deurschoten // Wiens schickt vercoelen Dies Met een gesloten // Fineert al mijn Misdeed' ick Dat sy my Och neen ick niet // dan een reyn hert Scheyd' ick dan Een deerlick schromen // Wt desen Van sulck op Sy haer beromen // Als een maeght straf Mijn reyn Niet falen // Dwelck met zijn bloet // t'rantsoen betalen //
