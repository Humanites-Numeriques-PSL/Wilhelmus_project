PAulus De gheloouige Christen En secht wilt Vwe by een coemste Als de sommighe (die sulcx Maer vermaent malcander Na schriften leer // en dies te Als ghy siet dat den dach des Heeren Tcomen by Doet de liefde groeyen al Dits niet Gheseyt, van vermaninghe Maer te besoecken Broeders en susters Christi Tot sHeeren prijs // als een Der reynder liefden // na Christenen Men siet Deen kole doet dander Inder Als deen Broeder dander hoort So wroecht den Gheest, dan elcx Waer by tghemoet weder vierich // By coemste soet // die is seer Want daer wt de liefde manierich // O Godts Merckt in contrarie van Een kole Als die lang alleene moet Tgloeyen vergaet, vrienden By een coemste dies, niet swichten // Maer alsoot dient // wel Neerstich by een, om elcken stichten // Alst so Dat by een coemste mach Reyn Sherten goet ghemoet wilt En alle ydel clap wt Als lieden,  herboren inden Thoont talder tijt // dat ghy hier Soeckende tonsienlijcke Sgheests saluen Ons, dat by een coemste Veel Tot een oprecht Godsalich Hier om vrienden Christi Neerstich begheuen, wilt v hier Vermaen, straf, troost, wt goet Gheschiedende, wilt doch nemen int Broeders Vseert (alst pas gheeft)  v Malcander Aldus nemende na Wat oorboor is ter Daer af wilt ramen, ter seluer Vroom inden gront // poocht talder Met tontfanghen pont, te doene Die Hebben gheweest, in dese Recht omme De slappe knien, (vierich int En helpt so by een coemste Dat ghy moecht smaken, sGeests claren // Als goeden schat // voor Godt sal Een seer bequaem, eersaem vergaren // Waer twee Vergaert zijn in des Heeren Daer is Int midden, als  hooft van Hierom leden Christi Coemt te same, so hier is Op dat door dien // tGodlijck By v woone hier en in Het is So lieffelijck voor sHeeren Dat Broeders Haer eendrachtich, by een Dauid die acht sulckdanich Als salue vloeyende op Aarons Ja soeter dan // dan dau waer De berch Hermon bedaut wert, dit Dus Dat by een coemste is te Broeders by Cracht en daet souden wy Ende ons licht so laten Dat Godt ghepresen worde daer Sonder onschult // Broeders De by een coemste na tbeuel Hier mede Broeders en Susters Christi Sgheests  leuen Laet blijcken voor groot ende En voor ledich thuys zijn Coemt doch by een tsy nacht ofte Watmen hier van // meer segghen Met tsaluen elck dat wyder dencken Die dit Heeft, voor die de waerheyt recht Bidt dat elck Alle wat twist mach doen By coemste niet latende Ghelijck Paulus onuerholen // Dus voor een slodt //  deeuwighen Broeders generalijck beuolen //
