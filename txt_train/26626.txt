GOds wijsheyt slechtich // voor s'wee-relts Ghedaelt oprechtich // van bouen Hoogh disputeren niet en Sulck argueren den gheest T'recht simpel In de waerheyt Wel begrijpen T'wordt door den Gheest Al wat niet geestich // en smaeckt Dat moet tempeestich // als Op't zant verdwijnen // neuen den Doncker bruwijnen  // en moghen De waerheyt Houden Oft schoon Sy verwint int S'Heeren ghemeente // met hert en In goe vereente // staet vast Op de reyn waerheyt // als een Van suyuer claerheyt // die hier noch Niet henen en Wat windt dat In haer Gods woordt, versterckts' Broeders eendrachtich // elck in sijn Laet ons aendachtich // nae Petrus Onder den heyden // ghelijck Goe wandel leyden // dat elck hem Tot Die met Door Ons hebben sal T'recht vromer leuen // Van Abel Staet al voorschreuen // tot ons Schoeriem, oft pleytken // van eenich T'bleetende geytken // daer oock al Niet schuldigh en Dan liefden Die t'gheener Den Naesten en doet In dees que tijden // voorhanden Laet ons vermijden // in cleen oft Dat met besinnen // selfs niemandt Buyten oft binnen // van ons Dat soete Van goet Door goede Van ons zy T'zy vremd' oft vrienden // wie dat ons Soo wel wy dienden // te zijn Want siet den name // den welcken Aenroepen t'same // desen wordt Voor borghe En wie hier op Hoe sal doch Desen te cort doen Wie en sal pooghen // om laten Eenen soo hooghen // borgh int Oft soo in schanden // dat men Daer door versmaden, sijnen Naem En t'volck Van sijner Iae recht als Bedrieghlijck in sijn Die dus laet steken // sijn borgh in Godtloose treken // doet hy, t'is Oock staet gheschreuen // hoe dat in T'slecht ouerbleuen // Israels Gheen leughenen Iae tonghe Doende Ghevonden zijn en Dus moet den Tempel // Gods staen Christi exempel, is claer en Te gheenen stonden // was noyt Bedroch ghevonden // in sijn mondt Wie valscheyt Oft leughen Niet by hem Maer uyt sijn huys hy Niemandt bedrieghen // ghy niet en Sprack die van lieghen // noyt hadde Siet, wilt aenschouwen // een Oprecht vol trouwen // daer gheenen T'zy meer oft Bedroch en is Des Heeren Die blijckt hier breedt en Och gaet niet qualijck // als vrient Met vriendt verralijck // ist oock niet Dat Broeders nouwe // en souden Moghen betrouwe // den Broeders T'welck haestich Soo't straffen T'waer tot Van s'Heeren volcken T'waer een oncieren // der leer Soo qua manieren // te lijden Daer soud' uytgroeyen // een quaet Elck soud' verfoeyen // sulck volck eer Ontstichtich het Iae duysterheyt Voor licht, oock Voor reucke vuylen Als wy wel schaffen, op waerheyts Sulck quaet te straffen // is meerder Nae s'vromer wenschen // en s'Heeren Voor Godt en menschen // t'antwoorden Dan datment Oft yet Want't hem Weerstrijdich Gods De Liefd' om sluyten // nu dit Wat daer is buyten // deucht niet Sy denckt gheen schade // te doen Sy slaet wel gade // eens anders Gheen vreucht sy en In eenich De waerheyt Verblijdt al haer Broeders met eenen // wilt noch Men leest in gheenen // Brief van te Ghestraft te wesen // wat Godt Maer wel wy lesen // van te En hoe dat, Vercoren Paulus, vry Daer ouer heeft Princen bevroedich // elck int Door een ootmoedich // hert, acht elck Hoogher een ander, // dan hy self Onder malcander // ghy sult In vredich Ghegordt zijn Met den bandt Rechter
