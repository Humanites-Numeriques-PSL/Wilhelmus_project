HOe magh, o Godt, het volck soo rasen, En Menschen dus uws Woordts vruchtbaerheydt // Te swack eylaes sy doch als dwasen // Die tegen dy en dyne waerheydt // V heylich kint en can geen swaerheyt // Meer door vervolg, dan aen sijn leden // So heden doen, die blint u claerheyt // Als Pilaet, oock Herodes deden // Gelijck als daer, in so versteenden // Ierusalem, Menschen gheneghen // Leet doen u kint en volck, ja meenden // Te roepen uyt, met een fel teghen // Gaet het noch toe, dan ghy door zegen Wt eeuwigh zaet u noch veel kinders // Dees laet ghy noch om wel bewegen // Heer, uwen Geest, die in veel hinders // Doe Pharao oock meend' uyt te roeyen // Israel wies door u vermoghen // 'sGelijcx u volc heeftmen sien groeyen // Daer doch al veel benynders V maert' eylaes seer loos bedrogen // Dat sy het sweerdt van u ghegheven // Om dooden toogh, en houdt ghetoghen // Op u huysvrou u seer verheven // Maer Syons Vorst sal in de wolcken // Sijn Bruydt tot troost seer haest verschijnen // Salmen geschrey, want alle volcken // Seer sullen zijn bevreest voor sijnen // Vergolden wert al dobbel 'tpijnen // V volc gedaen, met brant en eenen // Die quelligh sal sonder verdwijnen // Ia eeuwigh werdt daer druck en weenen // De Bruydt sal hoogh uyt d'Ardtsche banen // Want by zijn hooft moet zijn t'lichame // 'tVerwinnigh Lam sal haer van traenen Aen slevens born' in die bequaeme // Gherust sy sal eeuwigh den Name // Met blijtschap groot, van haren weerden // Maer vlucht, en sucht, in onveylsame // Den rootsten, ghy Tyrans op eerden //
