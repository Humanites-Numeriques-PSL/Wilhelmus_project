HErders eerbaer // alle die Met arbeyt swaer // nu nemen Die schaer // der Christen Onder wolven int veldt // Hoe nijdigh datse En vreest niet haer En vreest niet, want // op geen En staet gheplant // Gods huys Sijn handt // krachtigh van Oock nemmermeer vercort // en Den vloet zijnder Hy vriendelijck Hy sort zijn bloet // in Dus Herders vroet // hebt goeden Al boet // die oude Nu met discoordigh zaet // veel En zijt daerom niet Gods woort zy uwen Met Godts woordt raedt // slaet daerop Hoewel veel praet // nu Van smaet // achterklap Iae roverye van faem // ende En steunt doch maer te Op 'sHeeren woort Bequaem is 'twoort // des Heeren Want het verstoort // alle Wat voort // verheven De kennisse Gods comt // En laet u niet Met redenen Redenen kloeck // buyten het Werpt inden hoeck // hoe schoon Den vloeck // menich Leert, af noch toe te doen // so Tabernakel en Moet zijn nae Gods Godt alleen wijs // zy eeuwigh Al zijn advijs // is seer Hoe kijs // de Menschen Al haer wijsheydt voor Godt // is Gods dwaesheydt maeckt te Al 'sMenschen wijsheyt 's Menschen wijsheyt // van God Dus die bereydt // nu 'tkudde Arbeydt // int woort des Houdt vast daer over, tis // Soo sult ghy heylsaem En keeren // Erghernis sien, wy veel Maer wee voor wien // men moet Afsnien // en mijen als Haer woordt, eermen versint // In deuchden wort hy Die quaet gheklap Oorlof ydoon // Herders, de Der eeren schoon // wert uwen Den soon // werdt soet om Ay ghy ghetrouwe knecht // Mijn rijck is u te Ghy die wettelijck
