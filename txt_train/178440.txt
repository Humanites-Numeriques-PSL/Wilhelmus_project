Comt ter feesten // suyver Godts dienaren Die daer boven // in Godts Sijnen lof verbreyt // tot alder Helpt ons singhen // groote Van een suyver Die ons’ leven // heeft Brenghende hier voort // Godts eeuwigh Hoogh ghepresen // Maeght en moeder Naer u wenscyhen // alle V soeckt groot en cleyn // in d’aerdsche Laet doch dalen // uwe Der Wilt u ooghen // naer ons Keert van ons het quaet // o Heel deurwonden // deur de Maeckt ons al En u daeden // vol Toont ons al ghemeyn // als Al ons crachten // en Zijn tot uwen Elck een prijst // en V alderhooghste eer // reyn maghet In de sure // laetste Als men sterven By ons staet // metter En neemt in u behoet // Maria
