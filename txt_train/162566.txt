T'Valt mijn // een pijnAldus verkleynt te wesen,By der Ieucht fijnBen ick vergheten heel,Tfenijn // voor wijnWert seer van haer ghepresen,Het drincken dunckt hen eel,Daer schade schijnt voordeelSoo dom is jeuchts ghespeel,Welks bloem valt van haer steel:Siet hoe dat hier onachtsaemheytDen tijt verquist ende afleyt,Sulck seyt // ontseytGoet oordeel diet misgreyt.Comt Ieucht // en veuchtV onder mijnen leeren,Goe oorden en deuchtLeert // die sijt goet van aert,V vreucht // gheneuchtEn wilt van u niet weeren,Ontfanght mijn liefde waert:Hierom gheen moeyte spaert,Schout vliet // wat traecheyt baertSy niet, dat schaedt vergaert:Wilt onlust van u jaghen dan,V tijt dan wel ghebruyckt voortan,Den strijt // met vlijtAenveert om dien ghy vrijt.Siet toe // ick doeEen yghelijck ten goede,Die op mijn hebben // goe,Acht met hert sin, ghedachtDenckt hoe // ick spoeAls die niet werde moede,My te ghebruycken trachtDen tijt na niemant wacht,Die sluymt, by daech oft nachtVersuym, baet gheen na clacht,'Tverlies tot zijnder grooter schantVan een so costelijcken pant,Niemant // t'weer vantDus oeffent u verstant.Die Wet // ghesetOm weren wat mach schaden,Neemt aen sonder verlet,Sijt oorden onderdaen,Te bet // ghy metDen tijt so sonder smaden,Des wijsheyts pat sult gaenHaer inden gront verstaen,Dit is, die rechte baenGhewis, gheen ydel waen,Diet oyt te recht verstonden siet'Tverbont viel hem so zwaer doch niet,Ghy cont // goet rontVermeerderen uwe pont.Nu groeyt // en bloeytMijn gheestkens uyt ghelesen,Met s'Gheests natuer besproeytWeest neerstich int ghebou,Lust voeyt // cloeck roeytEn laet u niet verknesenIck wijsheyt uwe Vrou,Sal gheven u mijn trouDijn Naem // ick voor behouDen faem // diet oyt vergou,Met rijckdom, goet en eer belaenVer van haer salse sijn ghedaen,Die teer // in leerMaer int versuymen meer.Mijn zee // oock meeDe jonckheyt vordel dede,Wilt ghy niet sijn ghehoontMy voor u tuchtster croont,Rust vree // in steeVan tweedracht staen alrede,Soo my met reden toontDat oorden by u woont,Met const, wert ghy verschoont,Tijts jonst, u wel beloont,Hierom u tijt doch wel besteetTot deucht en eer zijt snel bereet,D'ontucht // ontvluchtZijt gierich tot leer sucht.
