In ‘t geest’lijck Bloemen-hof // staet eene Blom Waerdigh om hebben lof // hy staet in ‘t effen Te Saron seer playsant // lieflijck en Boven all’ schoonheyt: want // ’t is Godes Soon Hy staet in ‘t herte: daer // niet is eenigh Want dese hebben haer // ghekeert tot de Des levens, daer is pleyn // geen Was-bat meer te Om ‘t hert te wassen reyn // dan dees eenigh Druckt my spreeckt Christus soet // als een Zegel op ‘t Een herte vol ootmoet // dat rouwe draeght met Veel troost ghegeven wert // een troost vol soeter Komt al, ‘t zy nae oft vert // gaet tot dit Blom-Hof Spreect dan met blyden // spreect dan met ’s herten O Name Iesu! goedt // hoe soet is uwe Hoe groot is dit ghewin // hoe saligh zijn de Die dit te recht sien in // die werden niet O Name Iesu! wien // hoogh sit ter rechter Voor u moet alle knien // ootmoedigh buyghen, Ghy verwont triumphant // alle onse O Coningh seer valiant // naer u moet ons liefd O Iesu, Lam Gods eel // op Syons Berghe Daer ghy de Scharen veel // opsetten sult de Hoe heerlijck is u throon // hoe lieflijck is u Hoe groot is uwen loon // u Naem moet zijn O Iesu, u aenschijn // ist schoonste boven Hoe lieflijck salt daer zijn // daermen ‘t aenschouwen Komt al ghy Christen tal // komt laet ons vrolijck Comt maeckt nu bly geschal, comt wilt Gods lof Komt maer uyt sondig huys, kleeft niet meer aen het Verlaet ‘t sondigh abuys // met al des werelts Niet meer met haer en gaet // op haere kromme Haer aerts gierigheyd haet // zijt tot wel doen Dan vind ghy ’s herten wens // dan vint gy wat ghy Kont ghy verstaen o mens // soo wert u hert Ick denck dat ghy zwaer tilt // de werelt heel te Maer of ghy die behilt // wat sout u konnen ‘t Is doch al ydelheyd // wat de werelt hoogh Rijckdom haest onder leyd // hoe hoog dat schijnt haer Al wast Konincx geslacht // ’t moet al ter Aerden Dat dit niet meer bedacht // en werd, dat is groot Den onderganck is zwaer // van sulcken dwasen Van sulck een aerts dienaer // die ‘t hert van God Al tot dat aertsche slijck // al wilt ghy ‘t nu niet n Sal haest komen, gelijck // een val-strick neer Dan sult ghy worpen heen // u gouden Goden Die hier u troost wel scheen // maer dan in den Laet dalen u terstont // o mensch! wilt dit Laet af u quade vont // God wil u beter Wilt ghy dat beter sien // dat God u schencken Wilt ghy weten in wien // ghy dit sult vinden In Christo, haest u snel // siet zijn leven, Sijn woordt, waerheydts bevel // daer in sal hy u Toeft niet, beyt niet, o mensch // in dit Blom-hof te Hier vint ghy ’s levens wensch // haest u, wilt niet stil ‘t Moet hier doch al vergaen // maer Gods rijck sal Gaet voort door d’enge baen // soo sult ghy eeuwigh
